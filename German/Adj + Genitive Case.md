# Adj + Genitive Case
[[Adj Transformation]]
[[Genitive Case]]
某些形容词可以支配第二格名词作为补足语，表示"具有……性质的"、"充满……的"等含义。

常见带第二格的形容词：

| 形容词 | 含义 | 例句 |
|--------|------|------|
| **voll** | 充满的 | ein Glas voll Wassers（一杯满满的水） |
| **ledig** | 单身的、无……的 | einer Sache ledig sein（摆脱某事） |
| **bewusst** | 意识到的 | sich einer Sache bewusst sein（意识到某事） |
| **schuldig** | 负有责任的 | einer Tat schuldig sein（犯有某罪） |
| **würdig** | 值得的 | einer Ehre würdig sein（配得上某荣誉） |
| **fähig** | 有能力做……的 | zu einer Tat fähig sein（有能力做某事） |
| **bedürftig** | 需要……的 | der Hilfe bedürftig sein（需要帮助） |
| **sicher** | 确信的 | sich einer Sache sicher sein（对某事确定） |
| **müde** | 厌倦的 | des Lebens müde sein（厌倦生活） |
| **überdrüssig** | 厌倦的 | des Kampfes überdrüssig sein（厌倦斗争） |

形容词词尾变化（零冠词时）：
- 阳性/中性单数：-en 词尾 → voll guten Weins
- 阴性单数：-er 词尾 → voll guter Suppe
- 复数：-er 词尾 → voll eßbarer Sachen

口语中常用介词替代第二格：
- voll Wassers → voll mit Wasser
- bedürftig der Hilfe → bedürftig von Hilfe
