# 德语状态被动语态 (Zustandspassiv)

## 一、基本概念

状态被动语态（Zustandspassiv）表示**动作完成后的状态或结果**，强调的是状态本身，而非动作过程。相当于中文的"……着"或"已……"。

---

## 二、构成

### 基本结构

```
sein（变位）+ 第二分词
```

| 时态 | 结构 | 示例 |
|------|------|------|
| 现在时 | sein + Partizip II | Die Tür ist geöffnet. |
| 过去时 | waren + Partizip II | Die Tür war geöffnet. |
| 完成时 | sind + Partizip II + gewesen | Die Tür ist geöffnet gewesen. |
| 过去完成时 | waren + Partizip II + gewesen | Die Tür war geöffnet gewesen. |

---

## 三、各时态详解

### 1. 现在时（Präsens）

```german
Die Tür ist geöffnet.
门开着。

Das Geschäft ist geschlossen.
商店关门了。

Die Arbeit ist erledigt.
工作已完成。
```

**变位表**：

| 人称 | 形式 |
|------|------|
| ich | bin gefragt |
| du | bist gefragt |
| er/sie/es | ist gefragt |
| wir | sind gefragt |
| ihr | seid gefragt |
| sie/Sie | sind gefragt |

### 2. 过去时（Präteritum）

```german
Die Tür war geöffnet.
门当时开着。

Das Fenster war geschlossen.
窗户当时关着。
```

**变位表**：

| 人称 | 形式 |
|------|------|
| ich | war gefragt |
| du | warst gefragt |
| er/sie/es | war gefragt |
| wir | waren gefragt |
| ihr | wart gefragt |
| sie/Sie | waren gefragt |

### 3. 完成时（Perfekt）

**结构**：sein（变位）+ 第二分词 + gewesen

```german
Die Tür ist lange geöffnet gewesen.
门开了很长时间。

Der Laden ist den ganzen Tag geschlossen gewesen.
商店一整天都关着。
```

### 4. 过去完成时（Plusquamperfekt）

```german
Die Tür war schon geöffnet gewesen, als wir kamen.
当我们来的时候，门已经开着了。
```

## 四、Vorgangspassiv 与 Zustandspassiv 对比

### 核心区别

| 特征 | Vorgangspassiv（过程被动） | Zustandspassiv（状态被动） |
|------|---------------------------|---------------------------|
| 助动词 | werden | sein |
| 强调 | 动作过程 | 动作结果/状态 |
| 时间焦点 | 动作发生时 | 动作完成后 |
| 中文翻译 | 被…… | ……着/已…… |

### 对比示例

| 过程被动 | 状态被动 |
|----------|----------|
| Die Tür wird geöffnet. | Die Tür ist geöffnet. |
| 门正在被打开。 | 门开着。 |
| Der Kuchen wird gebacken. | Der Kuchen ist gebacken. |
| 蛋糕正在被烘烤。 | 蛋糕烤好了。 |
| Das Fenster wird geschlossen. | Das Fenster ist geschlossen. |
| 窗户正在被关上。 | 窗户关着。 |
| Der Brief wird geschrieben. | Der Brief ist geschrieben. |
| 信正在被写。 | 信已写好。 |
| Das Problem wird gelöst. | Das Problem ist gelöst. |
| 问题正在被解决。 | 问题已解决。 |

### 图解关系

```
动作发生 → 动作完成 → 状态持续
    ↓           ↓
Vorgangspassiv  Zustandspassiv
(wird geöffnet) (ist geöffnet)
```

## 五、状态被动的形成条件

### 1. 动作必须已完成

状态被动描述的是动作完成后持续的状态：

```german
✅ Das Geschäft ist geschlossen.（商店关门了——状态）
❌ Das Geschäft ist geschlossen worden.（这是过程被动的完成时）
```

### 2. 动词必须能产生持续状态

并非所有动词都能构成状态被动，只有能产生**持续结果状态**的动词才可以：

| 适合状态被动 | 不适合状态被动 |
|--------------|----------------|
| öffnen（打开）| sehen（看见）|
| schließen（关闭）| hören（听见）|
| reparieren（修理）| treffen（遇见）|
| waschen（洗）| besuchen（拜访）|
| streichen（粉刷）| |

**示例**：

```german
✅ Das Auto ist repariert.
车修好了。

✅ Die Wäsche ist gewaschen.
衣服洗好了。

❌ Das Auto ist gesehen.（不自然）
车被看见了。（结果状态不明确）
```

---

## 六、适合构成状态被动的动词类型

### 1. 动作结果可延续的动词

| 动词 | 状态被动 | 含义 |
|------|----------|------|
| öffnen | geöffnet | 开着的 |
| schließen | geschlossen | 关着的 |
| sperren | gesperrt | 被封锁的 |
| blockieren | blockiert | 被阻塞的 |

### 2. 表示变化/转变的动词

| 动词 | 状态被动 | 含义 |
|------|----------|------|
| verletzen | verletzt | 受伤的 |
| beschädigen | beschädigt | 受损的 |
| zerstören | zerstört | 被毁坏的 |
| verschmutzen | verschmutzt | 弄脏的 |

### 3. 表示完成的动词

| 动词 | 状态被动 | 含义 |
|------|----------|------|
| beenden | beendet | 已结束的 |
| erledigen | erledigt | 已完成的 |
| lösen | gelöst | 已解决的 |
| erfüllen | erfüllt | 已实现的 |

### 4. 表示装扮/覆盖的动词

| 动词 | 状态被动 | 含义 |
|------|----------|------|
| anziehen | angezogen | 穿着衣服的 |
| bedecken | bedeckt | 被覆盖的 |
| schmücken | geschmückt | 装饰好的 |
| bemalen | bemalt | 画有图案的 |

## 七、状态被动中的施事者

状态被动通常**不提及施事者**（动作执行者），因为重点在于状态本身。

```german
✅ Das Fenster ist geöffnet.
窗户开着。

❌ Das Fenster ist vom Hausmeister geöffnet.
（不常用，若要说明执行者应使用过程被动）
✅ Das Fenster wurde vom Hausmeister geöffnet.
窗户是被管理员打开的。
```

---

## 八、状态被动与形容词的区别

状态被动与形容词形式相同，但语义有所不同：

| 类型 | 特点 | 示例 |
|------|------|------|
| 状态被动 | 动作的结果状态 | Die Tür ist geöffnet.（被打开的状态）|
| 形容词 | 固有性质 | Die Tür ist offen.（门是开着的）|

**对比**：

| 状态被动 | 形容词 |
|----------|--------|
| Das Geschäft ist geschlossen. | Das Geschäft ist zu. |
| 商店关门了（被关的）。 | 商店是关着的。 |
| Der Schrank ist verschlossen. | Der Schrank ist verschließbar. |
| 柜子锁上了。 | 柜子是可以锁的。 |

---

## 九、情态动词与状态被动

情态动词可以与状态被动连用，表示推测或可能性：

```german
Das Geschäft muss geschlossen sein.
商店肯定关门了。

Die Tür kann geöffnet sein.
门可能是开着的。

Das Problem sollte gelöst sein.
问题应该已经解决了。

Die Arbeit dürfte erledigt sein.
工作可能已经完成了。
```

## 十、无人称状态被动

某些不及物动词也可构成无人称状态被动：

```german
Es ist viel geredet worden.（过程被动完成时）
→ Es ist viel geredet.（无人称状态被动，较少用）

常用的是：
Es ist viel diskutiert.
讨论很多。
```

---

## 十一、常见错误

### 1. 混淆过程被动与状态被动

❌ *Die Tür wird geöffnet.*（想说状态）
✅ Die Tür **ist** geöffnet.（门开着）

❌ *Die Tür ist geöffnet worden.*（想说状态）
✅ Die Tür **ist** geöffnet.（门已开着）

### 2. 错误使用助动词

❌ *Die Tür hat geöffnet.*
✅ Die Tür **ist** geöffnet.

### 3. 对不适合的动词使用状态被动

❌ *Das Buch ist gelesen.*（不自然）
✅ Das Buch **wurde gelesen**.（过程被动）

❌ *Er ist gesehen.*（不自然）
✅ Er **wurde gesehen**.（过程被动）

### 4. 状态被动中提及施事者

❌ *Die Tür ist von ihm geöffnet.*
✅ Die Tür **wurde von ihm geöffnet**.（过程被动）

## 十三、过程被动与状态被动转换

| 过程被动 | 状态被动 |
|----------|----------|
| Die Tür wird geöffnet. | → Die Tür ist geöffnet. |
| Das Fenster wurde geschlossen. | → Das Fenster ist geschlossen. |
| Das Auto wird repariert. | → Das Auto ist repariert. |
| Der Brief wurde geschrieben. | → Der Brief ist geschrieben. |

## 十四、记忆技巧

1. **核心口诀**：sein + 第二分词 = 状态被动
2. **助动词**：状态被动用 sein，过程被动用 werden
3. **时态简单**：状态被动常用现在时和过去时，完成时较少用
4. **无施事者**：状态被动不说明谁做的，只说什么状态
5. **动词选择**：只有能产生持续状态的动词才能用状态被动

## 十五、综合对比表

| 特征 | Vorgangspassiv | Zustandspassiv |
|------|----------------|----------------|
| 助动词 | werden | sein |
| 时态形式 | 6种时态 | 主要用现在时、过去时 |
| 焦点 | 动作过程 | 状态结果 |
| 施事者 | 可出现（von/durch）| 通常不出现 |
| 适用动词 | 所有及物动词 | 能产生持续状态的动词 |
| 中文翻译 | 被…… | ……着/已…… |
| 示例 | wird geöffnet | ist geöffnet |
