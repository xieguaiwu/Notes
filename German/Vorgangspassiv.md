# 德语过程被动语态 (Vorgangspassiv)

## 一、基本概念

过程被动语态（Vorgangspassiv）表示**动作正在进行或被完成**，强调动作本身，而非动作的执行者。相当于中文的"被……"。

---

## 二、构成

### 基本结构

```
werden（变位）+ 第二分词
```

| 时态 | 结构 | 示例 |
|------|------|------|
| 现在时 | werden + Partizip II | Das Buch wird gelesen. |
| 过去时 | wurden + Partizip II | Das Buch wurde gelesen. |
| 完成时 | sind + Partizip II + worden | Das Buch ist gelesen worden. |
| 过去完成时 | waren + Partizip II + worden | Das Buch war gelesen worden. |
| 将来时 | werden + Partizip II + werden | Das Buch wird gelesen werden. |

---

## 三、各时态详解

### 1. 现在时（Präsens）

```german
Das Haus wird gebaut.
房子正在被建造。

Die Arbeit wird erledigt.
工作正在被完成。
```

**变位表**：

| 人称 | 形式 |
|------|------|
| ich | werde gefragt |
| du | wirst gefragt |
| er/sie/es | wird gefragt |
| wir | werden gefragt |
| ihr | werdet gefragt |
| sie/Sie | werden gefragt |

### 2. 过去时（Präteritum）

```german
Das Haus wurde 1990 gebaut.
这栋房子建于1990年。

Der Brief wurde gestern abgeschickt.
信件昨天被寄出。
```

**变位表**：

| 人称 | 形式 |
|------|------|
| ich | wurde gefragt |
| du | wurdest gefragt |
| er/sie/es | wurde gefragt |
| wir | wurden gefragt |
| ihr | wurdet gefragt |
| sie/Sie | wurden gefragt |

### 3. 完成时（Perfekt）

**结构**：sein（变位）+ 第二分词 + worden

```german
Das Haus ist gebaut worden.
房子已经被建好了。

Der Auftrag ist erledigt worden.
任务已经完成了。
```

**注意**：被动语态的完成时用 **sein** 而非 haben。

**变位表**：

| 人称 | 形式 |
|------|------|
| ich | bin gefragt worden |
| du | bist gefragt worden |
| er/sie/es | ist gefragt worden |
| wir | sind gefragt worden |
| ihr | seid gefragt worden |
| sie/Sie | sind gefragt worden |

### 4. 过去完成时（Plusquamperfekt）

```german
Das Haus war schon gebaut worden, als wir kamen.
当我们来的时候，房子已经被建好了。
```

### 5. 将来时（Futur I）

```german
Das Haus wird gebaut werden.
房子将被建造。
```

### 6. 将来完成时（Futur II）

```german
Das Haus wird gebaut worden sein.
房子将已经建好。
```

## 四、主动句转被动句

### 基本转换规则

| 主动句成分 | 被动句成分 |
|------------|------------|
| 第四格宾语 | 第一格主语 |
| 第一格主语（施事者） | von + 第三格（可省略） |
| 动词 | werden + 第二分词 |

**示例**：

| 主动句 | 被动句 |
|--------|--------|
| Der Arzt operiert den Patienten. | Der Patient wird vom Arzt operiert. |
| 医生给病人做手术。 | 病人被医生做手术。 |
| Jemand stahl das Fahrrad. | Das Fahrrad wurde gestohlen. |
| 有人偷了自行车。 | 自行车被偷了。 |

### 施事者的表达

使用 **von + 第三格** 表示动作的执行者：

```german
Der Brief wurde vom Chef unterschrieben.
信件被老板签署了。

Das Bild wurde von einem berühmten Maler gemalt.
这幅画是由一位著名画家画的。
```

使用 **durch + 第四格** 表示手段或媒介：

```german
Die Stadt wurde durch ein Erdbeben zerstört.
这座城市因地震而被摧毁。

Die Nachricht wurde durch das Radio verbreitet.
消息通过广播传播。
```

## 五、不及物动词的被动语态

不及物动词（无第四格宾语的动词）也可构成被动语态，称为**无人称被动语态**。

**特点**：
- 无第一格主语
- 用代词 **es** 作形式主语（位于句首时）
- 强调动作本身

```german
Es wird hier nicht geraucht.
这里禁止吸烟。

Ihm wurde geholfen.
他得到了帮助。

Es wurde viel gelacht.
大家笑得很开心。

Nachts wird nicht gearbeitet.
晚上不工作。
```

**注意**：如果句首有其他成分，es 省略：

```german
Hier wird nicht geraucht.
这里禁止吸烟。
```

## 六、情态动词与被动语态

### 结构

```
情态动词（变位）+ 第二分词 + werden
```

| 时态 | 结构 | 示例 |
|------|------|------|
| 现在时 | muss + gelesen + werden | Das Buch muss gelesen werden. |
| 过去时 | musste + gelesen + werden | Das Buch musste gelesen werden. |
| 完成时 | hat + gelesen + werden + müssen | Das Buch hat gelesen werden müssen. |

### 常见用法

```german
Das Problem muss gelöst werden.
这个问题必须被解决。

Die Arbeit kann heute erledigt werden.
这项工作今天可以完成。

Der Brief sollte gestern abgeschickt werden.
这封信本应该昨天寄出。

Das Fenster darf nicht geöffnet werden.
这扇窗户不许打开。
```

---

## 七、带 zu 的被动不定式

### 结构

```
zu + 第二分词 + werden
```

常与 *sein* 连用，表示"能够被……"：

```german
Das Problem ist zu lösen.
= Das Problem kann gelöst werden.
这个问题可以被解决。

Die Arbeit ist schnell zu erledigen.
= Die Arbeit kann schnell erledigt werden.
这项工作可以很快完成。

Das Buch ist nicht zu verstehen.
= Das Buch kann nicht verstanden werden.
这本书无法被理解。
```

## 八、被动语态的替代形式

德语中常用其他结构代替被动语态，使表达更自然。

### 1. sich lassen + 不定式

```german
Das Problem lässt sich leicht lösen.
= Das Problem kann leicht gelöst werden.
这个问题容易解决。

Das Fenster lässt sich nicht öffnen.
= Das Fenster kann nicht geöffnet werden.
这扇窗户打不开。
```

### 2. sein + zu + 不定式

```german
Die Aufgabe ist schwer zu erfüllen.
= Die Aufgabe kann schwer erfüllt werden.
这项任务难以完成。

Es ist viel zu tun.
= Es kann viel getan werden.
有很多事要做。
```

### 3. man + 主动语态

```german
Man spricht hier Deutsch.
= Hier wird Deutsch gesprochen.
这里说德语。

Man verkauft hier keine Zigaretten.
= Hier werden keine Zigaretten verkauft.
这里不卖香烟。
```

### 4. funktionieren 等动词

某些动词本身含有被动意义：

```german
Die Tür öffnet sich automatisch.
门自动打开。

Das Buch verkauft sich gut.
这本书卖得很好。
```

## 九、Vorgangspassiv 与 Zustandspassiv 对比

| 特征 | Vorgangspassiv（过程被动） | Zustandspassiv（状态被动） |
|------|---------------------------|---------------------------|
| 助动词 | werden | sein |
| 强调 | 动作过程 | 动作结果/状态 |
| 中文翻译 | 被…… | ……着/已…… |
| 示例 | Die Tür wird geöffnet. | Die Tür ist geöffnet. |
| 含义 | 门正在被打开 | 门开着 |

**更多对比**：

| 过程被动 | 状态被动 |
|----------|----------|
| Das Fenster wird geschlossen. | Das Fenster ist geschlossen. |
| 窗户正在被关上。 | 窗户关着。 |
| Der Kuchen wird gebacken. | Der Kuchen ist gebacken. |
| 蛋糕正在被烘烤。 | 蛋糕烤好了。 |

## 十、常见错误

### 1. 混淆 werden 和 sein

❌ *Das Haus wird gebaut.*（状态）
✅ Das Haus **ist** gebaut.（状态：房子已建好）
✅ Das Haus **wird** gebaut.（过程：房子正在被建）

### 2. 完成时用错助动词

❌ *Das Haus hat gebaut worden.*
✅ Das Haus **ist** gebaut worden.

### 3. 情态动词被动语态词序错误

❌ *Das Buch wird gelesen müssen.*
✅ Das Buch **muss gelesen werden**.

### 4. 无人称被动省略 es 时机错误

❌ *Es wird hier nicht geraucht.*（句首正确）
❌ *Hier wird es nicht geraucht.*（句首已有成分，es 应省略）
✅ Hier **wird** nicht geraucht.

## 十一、实用例句

```german
1. Die Schweine werden an den Metzger verkauft.
   猪被卖给屠夫。

2. Das Gesetz wurde vom Parlament verabschiedet.
   法律由议会通过。

3. Ihm wurde mit einer Gehaltserhöhung gedroht.
   他被威胁要降薪。

4. Es wird darum gebeten, nicht zu rauchen.
   请勿吸烟。

5. Der Antrag muss bis Freitag eingereicht werden.
   申请必须周五前提交。

6. Das Problem ist nicht zu ignorieren.
   这个问题不容忽视。

7. Die Tür lässt sich nicht öffnen.
   门打不开。

8. Man sagt, er sei reich.
   据说他很富有。（被动含义的主动表达）
```

## 十二、记忆技巧

1. **核心口诀**：werden + 第二分词 = 过程被动
2. **时态记忆**：现在用 *wird*，过去用 *wurde*，完成用 *ist...worden*
3. **情态动词**：情态动词 + 分词 + werden（情态词在两头）
4. **无人称被动**：不及物动词也能被动，es 开头来占位
5. **替代形式**：*sich lassen* 和 *sein...zu* 让表达更地道
