# Kausative Verb

## 定义

使役动词（Kausative Verben）是表示"使"、"让"、"导致"等含义的动词，用于表达让某人或某物做某事，或使某事发生的动作。

在德语中，使役动词通常涉及主语促使宾语执行某个动作或进入某种状态。

## 基本结构

### 1. 直接使役结构
```
主语 + 使役动词 + 宾语 + 不定式
```

### 2. 间接使役结构
```
主语 + 使役动词 + 宾语 + 连词 + 从句
```

## 主要使役动词列表

### A. 常见使役动词

| 德语 | 英语 | 中文含义 | 使用方式 |
|------|------|----------|----------|
| lassen | to let/cause | 让、使 | lassen + 宾语 + 不定式 |
| machen | to make | 使、让 | machen + 宾语 + 形容词/名词 |
| sehen | to see | 看着 | sehen + 宾语 + 不定式 |
| hören | to hear | 听到 | hören + 宾语 + 不定式 |
| helfen | to help | 帮助 | helfen + 宾语 + 不定式 |
| fühlen | to feel | 感觉到 | fühlen + 宾语 + 不定式 |
| lehren | to teach | 教 | lehren + 宾语 + 不定式 |
| heißen | to be called/mean | 意味着、叫 | heißen + 宾语 + 不定式 |

### B. 语义使役动词

| 德语 | 英语 | 中文含义 |
|------|------|----------|
| veranlassen | to cause/induce | 促使、导致 |
| bewirken | to effect/cause | 引起、促成 |
| verursachen | to cause | 造成、引起 |
| hervorrufen | to provoke/cause | 引起、唤起 |
| bewegen | to move/persuade | 促使、感动 |

## 详细用法

### 1. lassen（最重要的使役动词）

#### 用法一：让某人做某事
- **结构：** `lassen + Akkusativ + Infinitiv`
- **例句：**
  - Ich lasse das Auto reparieren.（我让人修车。）
  - Der Lehrer lässt die Schüler Hausaufgaben machen.（老师让学生做家庭作业。）
  - Sie lässt ihren Haare schneiden.（她让人剪头发。）

#### 用法二：允许、许可
- **例句：**
  - Lass mich gehen!（让我走！）
  - Er lässt mich nicht ausgehen.（他不让我出去。）

#### 用法三：使某物处于某种状态
- **例句：**
  - Lass die Tür offen!（让门开着！）
  - Ich lasse das Licht an.（我让灯亮着。）

#### 用法四：lassen 的反身用法
- **例句：**
  - Ich lasse mich nicht täuschen.（我不会让自己被欺骗。）
  - Er lässt sich nicht helfen.（他不让人帮助他。）

### 2. machen

#### 用法：使某人/某物成为某种状态
- **结构：** `machen + Akkusativ + 形容词/名词`
- **例句：**
  - Die Nachricht machte mich traurig.（这消息让我难过。）
  - Er macht seinen Sohn zum Arzt.（他让他儿子成为医生。）
  - Das Essen macht mich satt.（这食物让我饱了。）

### 3. 感知觉使役动词

#### sehen（看见）
- **结构：** `sehen + Akkusativ + Infinitiv`
- **例句：**
  - Ich sehe ihn kommen.（我看见他来了。）
  - Wir sahen sie tanzen.（我们看见她们在跳舞。）

#### hören（听到）
- **结构：** `hören + Akkusativ + Infinitiv`
- **例句：**
  - Ich höre ihn singen.（我听到他在唱歌。）
  - Sie hörte das Kind weinen.（她听到孩子在哭。）

#### fühlen（感觉到）
- **结构：** `fühlen + Akkusativ + Infinitiv`
- **例句：**
  - Ich fühle mein Herz schlagen.（我感觉到我的心脏在跳动。）
  - Er fühlte das Auto vibrieren.（他感觉到车在震动。）

**注意：** 这类动词的不定式省略 "zu"

### 4. helfen（帮助）

- **结构：** `helfen + Dativ + Infinitiv`
- **例句：**
  - Ich helfe dir packen.（我帮你打包。）
  - Sie hilft ihrer Mutter kochen.（她帮她母亲做饭。）
  - Kannst du mir aufräumen?（你能帮我收拾吗？）

### 5. lehren（教）

- **结构：** `lehren + Akkusativ + Infinitiv`
- **例句：**
  - Er lehrt mich schwimmen.（他教我游泳。）
  - Die Mutter lehrt ihr Kind lesen.（母亲教孩子阅读。）

### 6. heißen（意味着、叫）

- **结构：** `heißen + Akkusativ + Infinitiv`
- **例句：**
  - Er heißt mich warten.（他叫我等。）
  - Das Gesetz heißt uns, Steuern zu zahlen.（法律要求我们纳税。）

## 其他使役表达

### 1. 使用 "etwas machen, dass..." 结构
- **例句：**
  - Ich mache, dass du gehst.（我让你走。）
  - Er macht, dass das Licht an ist.（他让灯亮着。）

### 2. 使用 "dazu bringen"（促使）
- **例句：**
  - Ich bringe ihn dazu, zuzugestimmen.（我促使他同意。）
  - Sie brachte ihn dazu, die Wahrheit zu sagen.（她促使他说出真相。）

### 3. 使用 "bewirken"（导致、引起）
- **例句：**
  - Das Medikament bewirkt eine Besserung.（这药导致病情好转。）
  - Sein Verhalten bewirkt Probleme.（他的行为引起问题。）

## 句子结构与语序

### 1. 主句结构
```
主语 + 使役动词 + 宾语 + 不定式 + 其他成分
```
- **例句：** Ich lasse meinen Bruder das Auto waschen.（我让我兄弟洗车。）

### 2. 疑问句结构
```
使役动词 + 主语 + 宾语 + 不定式 + ...?
```
- **例句：** Lässt du deinen Bruder das Auto waschen?（你让你兄弟洗车吗？）

### 3. 否定句结构
```
主语 + 使役动词 + nicht + 宾语 + 不定式
```
- **例句：** Ich lasse meinen Bruder nicht das Auto waschen.（我不让我兄弟洗车。）

## 时态变化

### 1. 现在时
- Ich lasse ihn kommen.（我让他来。）

### 2. 过去时（Präteritum）
- Ich ließ ihn kommen.（我让他来了。）

### 3. 现在完成时
- Ich habe ihn kommen lassen.（我已经让他来了。）

**注意：** 在完成时中，使役动词（如 lassen）放在句末，而非不定式。

### 4. 过去完成时
- Ich hatte ihn kommen lassen.（我当时已经让他来了。）

### 5. 第一将来时
- Ich werde ihn kommen lassen.（我将让他来。）

### 6. 第二将来时
- Ich werde ihn haben kommen lassen.（我将已经让他来了。）

## 被动语态

使役动词也可以使用被动语态，表示被动接受使役：

### 1. 被动语态结构
- **例句：**
  - Er wird von mir gehen lassen.（他被我让走。）
  - Das Auto wird repariert werden.（车将被修。）

### 2. 使用 "lassen" 的被动
- **例句：**
  - Das Problem muss gelöst werden.（这个问题必须被解决。）
  - Es wird viel über das Thema diskutiert.（这个话题被广泛讨论。）

## 与从句的结合

### 1. 使用 "dass" 从句
- **例句：**
  - Ich lasse ihn, dass er kommt.（我让他来。）
  - Er veranlasst, dass wir alle zusammenkommen.（他促使我们所有人聚在一起。）

### 2. 使用 "ob" 从句（疑问）
- **例句：**
  - Ich lasse ihn prüfen, ob es funktioniert.（我让他检查是否可行。）

## 使用注意事项

### 1. 不定式省略 "zu"
大多数使役动词后的不定式不需要 "zu"：
- ✅ Ich sehe ihn kommen.
- ❌ Ich sehe ihn zu kommen.

### 2. 宾语的格
- 大多数使役动词使用第四格（Akkusativ）作为宾语
- `helfen` 使用第三格（Dativ）作为宾语

### 3. 语义区别
- **lassen**：侧重于"允许"或"让别人做"
- **machen**：侧重于"使...成为某种状态"
- **bewirken/verursachen**：侧重于"导致"、"引起"

### 4. 反身动词的使役
- **例句：**
  - Ich lasse mich nicht beeindrucken.（我不会让自己被影响。）
  - Er lässt sich nicht überzeugen.（他不会被说服。）

### 5. 情感动词与使役
某些表达情绪的动词也可以起到使役作用：
- Die Nachricht freut mich.（这消息让我高兴。）
- Das ärgert ihn sehr.（这让他很生气。）

## 与英语使役动词的对比

| 德语 | 英语 | 说明 |
|------|------|------|
| lassen + Inf. | let/make + Inf. | 英语用 make/have 表示使役 |
| machen + Adj. | make + Adj. | 结构相似 |
| sehen/hören + Inf. | see/hear + Inf. | 结构相似 |
| helfen + Inf. | help + Inf. | 英语有时加 to |

## 练习句型

### 基础句型
1. Lass mich machen!（让我做！）
2. Er lässt uns warten.（他让我们等。）
3. Die Mutter macht das Kind glücklich.（母亲让孩子快乐。）
4. Ich sehe sie gehen.（我看见她走。）
5. Wir hören ihn singen.（我们听到他唱歌。）

### 高级句型
1. Der Manager veranlasste, dass das Projekt vorangetrieben wird.（经理促使项目推进。）
2. Seine Entscheidung bewirkte große Veränderungen.（他的决定引起巨大变化。）
3. Ich fühle das Schiff schaukeln.（我感觉到船在摇晃。）
4. Der Lehrer lehrt die Schüler gut schreiben.（老师教学生好好写字。）
5. Das Gesetz heißt uns, die Regeln zu beachten.（法律要求我们遵守规则。）

## 总结要点

1. **lassen** 是最常用的使役动词，有多种用法
2. 感知觉动词后的不定式不需要 "zu"
3. 大多数使役动词的宾语用第四格
4. `helfen` 的宾语用第三格
5. 完成时中，使役动词放在句末
6. 使役动词可以与从句结合使用
7. 区分"让某人做"和"使某人成为"的不同表达
8. 注意时态变化和被动语态的使用
