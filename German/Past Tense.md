# Past Tense

## 一、德语过去时态概述

德语中主要有四种表示"过去"的时态：

| 时态 | 德语名称 | 主要用途 |
|------|---------|---------|
| 现在完成时 | Perfekt | 口语中最常用，表示过去发生的动作或状态 |
| 过去时 | Präteritum | 书面语和正式场合使用，常用于叙述 |
| 过去完成时 | Plusquamperfekt | 表示过去的过去 |
| 过去将来时 | Futur II | 表示从过去某时看将要完成的动作 |

---

## 二、现在完成时

### 1. 定义与用途

现在完成时是德语口语中最常用的过去时态，用于表示：
- 已经完成的动作：Ich habe gegessen.（我吃过饭了。）
- 对现在有影响的过去经历：Ich bin nach Deutschland gefahren.（我去了德国。）
- 在一定时间内完成的动作：Er hat das Buch gelesen.（他读了这本书。）

### 2. 构成规则

**公式：haben/sein + 第二分词**

#### 2.1 助动词 haben（大部分动词）

**例句：**
- Ich habe gearbeitet.（我工作过。）
- Du hast geschrieben.（你写过。）
- Er hat das Buch gelesen.（他读了这本书。）

#### 2.2 助动词 sein（部分动词）

**必须用 sein 的动词类型：**

1. **表示位置移动的动词**
   - gehen（走）→ ich bin gegangen
   - kommen（来）→ ich bin gekommen
   - fahren（开/乘车）→ ich bin gefahren
   - laufen（跑）→ ich bin gelaufen
   - fliegen（飞）→ ich bin geflogen
   - reisen（旅行）→ ich bin gereist

2. **表示状态改变的动词**
   - werden（变成）→ ich bin geworden
   - aufwachen（醒来）→ ich bin aufgewacht
   - einschlafen（入睡）→ ich bin eingeschlafen
   - sterben（死）→ ich bin gestorben
   - wachsen（生长）→ ich bin gewachsen

3. **动词 sein 和 bleiben**
   - sein（是）→ ich bin gewesen
   - bleiben（停留）→ ich bin geblieben

**例句：**
- Ich bin nach Hause gegangen.（我回家了。）
- Er ist nach Deutschland gefahren.（他去了德国。）
- Sie ist eingeschlafen.（她睡着了。）

### 3. 第二分词构成规则

#### 3.1 规则动词（弱变化）

**规则：ge- + 动词词干 + -t**

| 动词           | 过去时       | 第二分词         |
| ------------ | --------- | ------------ |
| machen（做）    | machte    | gemach**t**  |
| arbeiten（工作） | arbeitete | gearbei**t** |
| lernen（学习）   | lernte    | geler**t**   |
| wohnen（居住）   | wohnte    | gewohn**t**  |
| sagen（说）     | sagte     | gesag**t**   |

**注意：**
- 以 -ieren 结尾的动词不加 ge-
  - studieren → studiert
  - dekorieren → dekoriert

#### 3.2 不规则动词（强变化）

**规则：ge- + 词干变化 + -en**

| 动词 | 过去时 | 第二分词 |
|------|--------|---------|
| gehen（走） | ging | gang**en** |
| kommen（来） | kam | komm**en** |
| sehen（看） | sah | geseh**en** |
| nehmen（拿） | nahm | genomme**en** |
| trinken（喝） | trank | getrunke**en** |
| essen（吃） | aß | gess**en** |
| schreiben（写） | schrieb | geschrieb**en** |
| sprechen（说） | sprach | gesproch**en** |
| fahren（开） | fuhr | gefahr**en** |
| stehen（站） | stand | gestand**en** |

#### 3.3 混合变化动词

**规则：词干元音变化 + -t**

| 动词 | 过去时 | 第二分词 |
|------|--------|---------|
| bringen（带来） | brachte | gebrach**t** |
| denken（想） | dachte | gedach**t** |
| kennen（认识） | kannte | gekann**t** |
| nennen（称） | nannte | genann**t** |
| rennen（跑） | rannte | gerann**t** |
| wissen（知道） | wusste | gewuss**t** |

#### 3.4 可分前缀动词

**规则：前缀与 ge- 分离**

| 动词 | 第二分词 |
|------|---------|
| aufstehen（起床） | auf**ge**stand**en** |
| ankommen（到达） | an**ge**komm**en** |
| einladen（邀请） | ein**ge**lad**en** |
| zurückkommen（回来） | zurück**ge**komm**en** |
| weggehen（走开） | weg**ge**gang**en** |

**例句：**
- Ich bin aufgestanden.（我起床了。）
- Er ist angekommen.（他到了。）
- Wir haben eingeladen.（我们邀请了。）

#### 3.5 不可分前缀动词

**不可分前缀：be-, er-, ver-, ent-, zer-, ge-, miss-**

**规则：不加 ge-**

| 动词             | 第二分词            |
| -------------- | --------------- |
| besuchen（拜访）   | besuch**t**     |
| erklären（解释）   | erklär**t**     |
| verkaufen（卖）   | verkauf**t**    |
| entdecken（发现）  | entdeck**t**    |
| zerstören（破坏）  | zerstör**t**    |
| gefallen（喜欢）   | gefall**en**    |
| misslingen（失败） | misslunge**en** |

## 三、过去时

### 1. 定义与用途

过去时主要用于：
- 正式书面语
- 叙述故事、报道
- 报告和学术写作
- 情感动词 haben, sein, werden, können, müssen 等

### 2. 规则动词变化

**一三人称单数相同**
**人称词尾：**

| 人称 | 词尾 |
|------|------|
| ich | -te |
| du | -test |
| er/sie/es | -te |
| wir | -ten |
| ihr | -tet |
| sie/Sie | -ten |

**例词：machen（做）**

| 人称 | 过去时 |
|------|--------|
| ich | mach**te** |
| du | mach**test** |
| er/sie/es | mach**te** |
| wir | mach**ten** |
| ihr | mach**tet** |
| sie/Sie | mach**ten** |

**注意：**
- 以 -d, -t, -chn, -ffn, -gn, -tm, -dm 结尾的动词，加 -e-
  - arbeiten → arbeitete, arbeitetest, arbeitete, arbeiteten, arbeitetet, arbeiteten
  - reden → redete
  - antworten → antwortete

### 3. 不规则动词变化

不规则动词过去时需要记忆，主要模式：

| 动词 | 过去时 |
|------|--------|
| sein（是） | war |
| haben（有） | hatte |
| werden（变成） | wurde |
| gehen（走） | ging |
| kommen（来） | kam |
| sehen（看） | sah |
| geben（给） | gab |
| nehmen（拿） | nahm |
| trinken（喝） | trank |
| essen（吃） | aß |
| schreiben（写） | schrieb |
| sprechen（说） | sprach |
| lesen（读） | las |
| fahren（开） | fuhr |
| stehen（站） | stand |
| liegen（躺） | lag |
| sitzen（坐） | saß |
| finden（发现） | fand |
| werden（变成） | wurde |

**人称变化：**

| 人称 | 词尾 |
|------|------|
| ich | - |
| du | -st |
| er/sie/es | - |
| wir | -en |
| ihr | -t |
| sie/Sie | -en |

**例词：gehen（走）**

| 人称 | 过去时 |
|------|--------|
| ich | ging |
| du | ging**st** |
| er/sie/es | ging |
| wir | ging**en** |
| ihr | ging**t** |
| sie/Sie | ging**en** |

**例词：haben（有）**

| 人称        | 过去时         |

| 人称 | 过去时 |
|------|--------|

| du | wurd**est** |
| er/sie/es | wurde |
| wir | wurd**en** |
| ihr | wurd**et** |
| sie/Sie | wurd**en** |

### 4. 混合变化动词

这些动词词干元音变化，但按规则动词加词尾。

| 动词 | 词干变化 | du | er/sie/es |

## 四、过去完成时

### 1. 定义与用途

过去完成时表示"过去的过去"，即在过去某时之前已经完成的动作。

**公式：war/hatte + 第二分词**

**例句：**
- Er hatte das Buch schon gelesen, bevor er es kaufte.（他在买这本书之前已经读过了。）
- Wir waren schon angekommen, als du anriefst.（你打电话的时候我们已经到了。）

### 2. 助动词过去时

| 人称 | sein（过去时） | haben（过去时） |
|------|---------------|----------------|
| ich | war | hatte |
| du | warst | hattest |
| er/sie/es | war | hatte |
| wir | waren | hatten |
| ihr | wart | hattet |
| sie/Sie | waren | hatten |

### 3. 用法示例

| 时态 | 例句 |
|------|------|
| 现在 | Ich habe gegessen.（我吃过饭了。） |
| 过去 | Ich hatte gegessen, als er kam.（他来的时候我已经吃过饭了。） |
| 过去完成 | Ich hatte das Buch gelesen, bevor ich ins Kino ging.（我去电影院之前已经读了这本书。） |

**更复杂的例句：**
- Als ich ankam, hatte er schon gegangen.（当我到达时，他已经走了。）
- Sie war schon eingeschlafen, als der Film anfing.（电影开始时她已经睡着了。）
- Wir hatten den Koffer schon gepackt, bevor der Taxifahrer ankam.（出租车司机到达之前我们已经打包好行李了。）

## 五、过去将来时

### 1. 定义与用途

过去将来时表示从过去某时看将要发生的动作或状态。

**公式：werden（过去时）+ 不定式**

### 2. 构成

| 人称 | werden（过去时） |
|------|------------------|
| ich | wurde |
| du | wurdest |
| er/sie/es | wurde |
| wir | wurden |
| ihr | wurdet |
| sie/Sie | wurden |

**例句：**
- Er sagte, er würde kommen.（他说他会来。）
- Ich dachte, das würde klappen.（我以为这会成功。）

### 3. 第一将来时与第二将来时

**第一将来时：würde + 不定式**
- Er sagte, er würde das Buch lesen.（他说他会读这本书。）

**第二将来时：würde haben/gehabt + 第二分词**
- Er sagte, er würde das Buch gelesen haben.（他说他读完了这本书。）

## 六、情态动词的过去时

### 1. 情态动词原形

| 动词 | 含义 |
|------|------|
| können | 能够 |
| müssen | 必须 |
| wollen | 想要 |
| sollen | 应该 |
| dürfen | 可以 |
| mögen | 喜欢 |

### 2. 情态动词过去时

| 动词 | 过去时 |
|------|--------|
| können | konnte |
| müssen | musste |
| wollen | wollte |
| sollen | sollte |
| dürfen | durfte |
| mögen | mochte |

**人称变化：**

| 人称 | 词尾 |
|------|------|
| ich | - |
| du | -st |
| er/sie/es | - |
| wir | -en |
| ihr | -t |
| sie/Sie | -en |

**例词：können（能够）**

| 人称 | 过去时 |
|------|--------|
| ich | konnte |
| du | konnt**est** |
| er/sie/es | konnte |
| wir | konnt**en** |
| ihr | konnt**et** |
| sie/Sie | konnt**en** |

**例句：**
- Ich konnte gestern nicht kommen.（我昨天不能来。）
- Er musste nach Hause gehen.（他必须回家。）
- Sie wollte das Buch lesen.（她想读这本书。）
| Wir durften nicht ausgehen.（我们不能出去。）

### 3. 情态动词的现在完成时

情态动词在现在完成时中通常使用第二分词，但实际上更常见的是用不定式形式：

**形式：haben/sein + 第二分词**

| 动词 | 第二分词 |
|------|---------|
| können | gekonnt |
| müssen | gemusst |
| wollen | gewollt |
| sollen | gesollt |
| dürfen | gedurft |
| mögen | gemocht |

**例句：**
- Ich habe das gekonnt.（我能够做到这个。）
- Er hat das gewollt.（他想要这个。）

**但更常见的是用情态动词 + 不定式：**
- Ich habe das tun können.（我能够做这个。）
- Er hat das tun müssen.（他必须做这个。）

## 七、各过去时态的比较

### 1. 现在完成时 vs. 过去时

| 特点 | 现在完成时 | 过去时 |
|------|-----------|--------|
| 使用场合 | 口语，非正式 | 书面语，正式 |
| 助动词 | haben/sein + 第二分词 | 直接变位 |
| 情感动词 | 较少使用 | 常用 |
| 叙述 | 个人经历 | 故事、报道 |

**相同含义的表达：**
- Ich habe das Buch gelesen.（我读了这本书。）→ 口语
- Ich las das Buch.（我读了这本书。）→ 书面

### 2. 过去完成时 vs. 现在完成时

| 时态 | 用途 | 例句 |
|------|------|------|
| 现在完成时 | 现在时点看过去的动作 | Ich habe gegessen.（我吃过了。） |
| 过去完成时 | 过去时点看更早的动作 | Ich hatte gegessen, als er kam.（他来时我已经吃过饭了。） |

### 3. 时态选择指南

| 情况 | 推荐时态 |
|------|---------|
| 日常对话 | 现在完成时 |
| 写报告/论文 | 过去时 |
| 情感动词 haben, sein, werden | 过去时 |
| 情态动词 | 过去时 |
| 叙述过去的过去 | 过去完成时 |
| 表示从过去看将来 | 过去将来时 |

## 九、常见错误

| 错误 | 正确 | 说明 |
|------|------|------|
| Ich bin gegessen. | Ich habe gegessen. | essen 用 haben |
| Er hat nach Hause gefahren. | Er ist nach Hause gefahren. | 位置移动用 sein |
| Ich habe das Buch gelesen. | Ich las das Buch. | 正确，口语 vs 书面 |
| Wir haben angerufen. | Wir riefen an. | 可分动词分离 |
| Ich habe studiert. | Ich habe studiert. | -ieren 不加 ge- |
| Ich habe gegesst. | Ich habe gegessen. | 不规则变化 |
| Er hat gemacht. | Er hat gemacht. | 规则动词 |
| Sie sind gegangen. | Sie sind gegangen. | 位置移动用 sein |

## 十、练习提示

### 1. 判断助动词
- 位置移动、状态改变 → sein
- 其他 → haben

### 2. 记忆不规则动词
- 强变化动词：词干变化 + -en
- 混合变化动词：词干变化 + -t

### 3. 可分动词
- 前缀与 ge- 分离：aufstehen → aufgestanden

### 4. 不可分前缀
- 不加 ge-：besuchen → besucht

### 5. -ieren 结尾动词
- 不加 ge-：studieren → studiert

## 总结要点

1. **现在完成时**：haben/sein + 第二分词，口语最常用
2. **过去时**：直接变位，书面语常用
3. **过去完成时**：war/hatte + 第二分词，表示过去的过去
4. **过去将来时**：würde + 不定式，从过去看将来
5. **助动词选择**：位置移动/状态改变用 sein，其他用 haben
6. **第二分词规则**：规则动词加 -t，不规则动词加 -en
7. **可分动词**：前缀与 ge- 分离
8. **不可分前缀**：be-, er-, ver-, ent-, zer-, ge-, miss- 不加 ge-
