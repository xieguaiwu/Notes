# Präpositionalverben und Satzverbindung

> [!abstract] 概述
> 本篇涵盖德语学术写作中的三大核心工具：**正式写作引导词**（用于议论文/分析文篇章衔接）、**学术句式模板**（观点表达/论证/举例/总结）、**介词固定搭配**（动词/形容词/名词+介词）。搭配 [[口语表达]]（口语连接词）和 [[Clause]]（从句引导词）使用。

---

## 一、正式写作引导词 (Konnektoren für formelles Schreiben)

与口语连接词不同，以下引导词适用于议论文 (Erörterung)、评论 (Kommentar)、报告 (Bericht) 等正式文体。

### 1. 篇章结构 (Textstruktur)

| 引导词 | 语序 | 用法 |
|--------|------|------|
| **Zunächst / Zuerst** | 反语序 | 引出第一个论点 |
| **Erstens / Zweitens / Drittens** | 反语序 | 枚举论点（正式） |
| **Des Weiteren / Ferner** | 反语序 | 进一步论述（书面语） |
| **Darüber hinaus** | 反语序 | 除此之外（递进） |
| **Nicht zuletzt** | 反语序 | 最后同样重要的是 |
| **Abschließend / Schlussendlich** | 反语序 | 最后 |
| **Im Folgenden** | 反语序 | 接下来 |
| **Einleitend sei gesagt, dass...** | 从句末动词 | 首先需要说明... |

### 2. 逻辑关系 (Logische Relationen)

| 类别 | 引导词 | 语序 | 说明 |
|------|--------|------|------|
| **因果** | **nämlich** | 不占位 | 后置解释，相当于"也就是因为" |
| | **demzufolge / folglich** | 反语序 | 因此（强因果） |
| | **infolgedessen** | 反语序 | 结果（正式） |
| | **aus diesem Grund** | 反语序 | 出于这个原因 |
| **对比** | **hingegen / dagegen** | 反语序 | 然而、相反 |
| | **demgegenüber** | 反语序 | 与此相对（书面） |
| | **im Gegensatz dazu** | 反语序 | 与此相反 |
| | **während** (从句) | 从句末动词 | 而（对比两个事实） |
| **让步** | **zwar... aber...** | `zwar`反语序¹ | 虽然...但是... |
| | **allerdings** | 反语序 | 不过（带保留的转折） |
| | **trotzdem / dennoch** | 反语序 | 尽管如此 |
| | **obwohl / obgleich** (从句) | 从句末动词 | 尽管（正式） |
| **条件** | **andernfalls / sonst** | 反语序 | 否则 |
| | **unter dieser Voraussetzung** | 反语序 | 在此条件下 |
| **目的** | **zu diesem Zweck** | 反语序 | 为此目的 |
| | **mit dem Ziel** |  —  | 以...为目标 |

> [!tip] 语序速查
> - **反语序**：连接词位于句首时，动词→主语（V→S）
> - **不占位**：连接词不占用第一位，主语仍在动词前
> - **从句末动词**：从属连词引导从句，动词放末尾
> 
> ¹ `zwar` 位于句首时反语序（动词提前），位于句中时不改变语序。

### 3. 强调与限定 (Hervorhebung & Einschränkung)

| 引导词 | 语序 | 含义 |
|--------|------|------|
| **insbesondere / vor allem** | 反语序 | 特别是 |
| **vorwiegend / hauptsächlich** | 反语序 | 主要是 |
| **in erster Linie** | 反语序 | 首要地 |
| **zumindest / wenigstens** | 反语序 | 至少 |
| **allerdings / freilich** | 反语序 | 当然（带保留） |
| **keineswegs / keinesfalls** | 反语序 | 绝不 |
| **in gewisser Hinsicht** | 反语序 | 从某种程度上说 |
| **was ... betrifft / angeht** | 从句末动词 | 就...而言 |

> [!example] 正式引导词示例句
> - **Zunächst** ist zu klären, worum es sich bei diesem Phänomen handelt.
> - **Des Weiteren** spielt die wirtschaftliche Lage eine entscheidende Rolle.
> - **Keineswegs** darf man davon ausgehen, dass dies die einzige Lösung ist.
> - **Was** die finanziellen Aspekte **betrifft**, ist noch eine Prüfung erforderlich.

## 二、学术写作句式 (Satzmuster für wissenschaftliches Schreiben)

### 1. 引入主题 (Thema einleiten)

```
In der heutigen Debatte / Diskussion spielt [Thema] eine zentrale Rolle.
Die Frage, ob / inwiefern ..., ist von großer Bedeutung / Relevanz.
In den letzten Jahren hat [Thema] zunehmend an Aufmerksamkeit gewonnen.
Gegenstand dieser Erörterung / Analyse ist ...
```

### 2. 表达观点 (Standpunkt vertreten)

| 句式 | 注册 |
|------|------|
| **Ich bin der Auffassung / Ansicht / Meinung, dass...** | 标准 |
| **Ich vertrete den Standpunkt / die These, dass...** | 标准/正式 |
| **Es lässt sich behaupten / feststellen / konstatieren, dass...** | 正式 |
| **Es steht außer Frage / Zweifel, dass...** | 强调 |
| **Daran besteht kein Zweifel, dass...** | 强调 |
| **Ich möchte die Behauptung aufstellen, dass...** | 正式（大胆断言） |
| **Vieles spricht für die Annahme, dass...** | 谨慎表述 |
| **Es ist davon auszugehen, dass...** | 正式（推测） |
| **Es kann nicht ausgeschlossen werden, dass...** | 谨慎（不排除） |

### 3. 论证 (Argumentieren)

| 句式 | 功能 |
|------|------|
| **Einerseits... andererseits...** | 双面论证 |
| **Dafür spricht / Dagegen spricht, dass...** | 正反论据 |
| **Ein gewichtiges / starkes Argument für/gegen ... ist ...** | 重要论据 |
| **Es ist zu bedenken / berücksichtigen, dass...** | 提醒考虑 |
| **Hinzu kommt, dass...** | 补充论点 |
| **Nicht zu unterschätzen ist...** | 强调某因素 |
| **Ein weiterer Aspekt / Gesichtspunkt ist...** | 额外角度 |
| **Besonders hervorzuheben ist...** | 特别强调 |
| **Dies zeigt sich besonders deutlich an...** | 例证 |

### 4. 引用他人观点 (Fremde Positionen referieren)

| 句式 | 含义 |
|------|------|
| **Nach [Autor] ist ...** | 根据某人的观点 |
| **Wie [Autor] bereits festgestellt / betont hat, ...** | 正如某人已指出的... |
| **Laut [Quelle] lässt sich ... feststellen** | 据某来源... |
| **In der Forschung besteht Einigkeit darüber, dass...** | 学界共识... |
| **Die vorliegende Studie / Untersuchung zeigt / belegt, dass...** | 研究显示... |

### 5. 举例 (Beispiele anführen)

| 句式 | 正式程度 |
|------|---------|
| **Zum Beispiel / Beispielsweise...** | 标准 |
| **Ein Beispiel / Beleg hierfür ist...** | 标准 |
| **Dies lässt sich an ... veranschaulichen / verdeutlichen** | 正式 |
| **Exemplarisch sei hier ... genannt / angeführt** | 正式 |
| **Als Beispiel dient / mag ... gelten** | 正式 |
| **Dies wird besonders deutlich in / am Beispiel von...** | 正式 |

### 6. 对比 (Vergleichen & Gegenüberstellen)

| 句式 | 结构 |
|------|------|
| **Während ..., ...** | 从句+主句对比 |
| **Im Gegensatz / Unterschied zu ...** | 介词短语+名词 |
| **Im Vergleich zu / mit ...** | 与...相比 |
| **... hingegen / dagegen ...** | 句中副词对比 |
| **... unterscheidet sich von ... dadurch, dass ...** | 区别在于 |
| **... weist im Vergleich zu ... eine ... auf** | 某方面具有...特征 |

### 7. 强调因果关系 (Kausalzusammenhänge betonen)

| 句式 | 说明 |
|------|------|
| **Die Ursache / Der Grund hierfür liegt in ...** | 原因在后 |
| **Dies ist auf ... zurückzuführen** | 归因于 |
| **... zieht ... nach sich** | 带来后果 |
| **... hat ... zur Folge** | 导致 |
| **Daraus ergibt sich / resultiert / folgt, dass...** | 推理结果 |
| **... steht in engem Zusammenhang mit ...** | 密切相关 |
| **Dies lässt sich dadurch erklären, dass...** | 解释原因 |

### 8. 总结与结论 (Zusammenfassen & Schlussfolgern)

| 句式 | 说明 |
|------|------|
| **Zusammenfassend lässt sich sagen / festhalten, dass...** | 总结 |
| **Abschließend ist festzustellen / zu betonen, dass...** | 结尾强调 |
| **Es lässt sich das Fazit ziehen, dass...** | 得出结论 |
| **Aus den genannten Gründen bin ich der Ansicht, dass...** | 基于前述原因 |
| **Alles in allem / Im Großen und Ganzen ...** | 总体而言 |
| **Somit kann festgehalten werden, dass...** | 书面总结 |
| **Es bleibt zu hoffen / zu wünschen, dass...** | 展望 |

## 三、介词固定搭配 (Präpositionale Ausdrücke)

> [!warning] 高频错误来源
> 介词搭配是德语写作中**出错率最高**的知识点之一。学生常犯的错误是直接套用英语或中文的介词习惯。学**每个动词/形容词时必须同时记忆其介词 + 格**。

### 1. 动词 + 固定介词 (Verben mit festen Präpositionen)

| 动词 | 介词+格 | 含义 | 例句 |
|------|---------|------|------|
| **achten** | **auf** (+Akk) | 注意 | Achten Sie **auf** die Feinheiten. |
| **ankommen** | **auf** (+Akk) | 取决于 | Es kommt **auf** die Situation **an**. |
| **antworten** | **auf** (+Akk) | 回答 | Er antwortet **auf** die Frage. |
| **sich ärgern** | **über** (+Akk) | 对...生气 | Sie ärgert sich **über** die Verspätung. |
| **aufhören** | **mit** (+Dat) | 停止 | Hör **mit** dem Lärm **auf**! |
| **bestehen** | **aus** (+Dat) | 由...组成 | Das Team besteht **aus** fünf Mitgliedern. |
| **bestehen** | **auf** (+Dat) | 坚持 | Er besteht **auf** seiner Forderung. |
| **sich bewerben** | **um** (+Akk) / **bei** (+Dat) | 申请 | Sie bewirbt sich **um** die Stelle **bei** der Firma. |
| **denken** | **an** (+Akk) | 想到 | Ich denke oft **an** meine Kindheit. |
| **diskutieren** | **über** (+Akk) | 讨论 | Wir diskutieren **über** das Thema. |
| **sich entscheiden** | **für** (+Akk) / **gegen** (+Akk) | 决定选/反对 | Er entscheidet sich **für** den Job. |
| **sich entschuldigen** | **für** (+Akk) / **bei** (+Dat) | 为...向...道歉 | Ich entschuldige mich **für** den Fehler **bei** ihr. |
| **sich erinnern** | **an** (+Akk) | 回忆 | Erinnern Sie sich **an** unseren ersten Treff? |
| **sich erkundigen** | **nach** (+Dat) | 询问 | Ich erkundige mich **nach** den Öffnungszeiten. |
| **sich freuen** | **auf** (+Akk) | 期待（未来） | Wir freuen uns **auf** den Urlaub. |
| **sich freuen** | **über** (+Akk) | 对...感到高兴（已发生） | Sie freut sich **über** das Geschenk. |
| **sich fürchten** | **vor** (+Dat) | 害怕 | Das Kind fürchtet sich **vor** dem Hund. |
| **gelten** | **als** (+Nom) | 被视为 | Er gilt **als** Experte auf diesem Gebiet. |
| **sich gewöhnen** | **an** (+Akk) | 习惯于 | Ich habe mich **an** das Klima gewöhnt. |
| **glauben** | **an** (+Akk) | 相信（存在/理念） | Glaubst du **an** die Zukunft? |
| **sich handeln** | **um** (+Akk) | 涉及 | Es handelt sich **um** ein komplexes Problem. |
| **hindern** | **an** (+Dat) | 阻碍 | Nichts hindert mich **an** der Verwirklichung. |
| **hinweisen** | **auf** (+Akk) | 指出 | Ich weise **auf** die Risiken **hin**. |
| **sich informieren** | **über** (+Akk) | 了解 | Informieren Sie sich **über** unsere Angebote. |
| **sich interessieren** | **für** (+Akk) | 对...感兴趣 | Sie interessiert sich **für** Kunstgeschichte. |
| **sich kümmern** | **um** (+Akk) | 照顾 | Er kümmert sich **um** seine Eltern. |
| **leiden** | **an** (+Dat) / **unter** (+Dat) | 患（病）/受...之苦 | Er leidet **an** Diabetes / **unter** dem Druck. |
| **nachdenken** | **über** (+Akk) | 思考 | Ich denke **über** das Problem **nach**. |
| **profitieren** | **von** (+Dat) | 从...获益 | Wir profitieren **von** der Zusammenarbeit. |
| **rechnen** | **mit** (+Dat) | 预计 | Wir rechnen **mit** steigenden Kosten. |
| **sprechen / reden** | **über** (+Akk) | 谈论 | Wir sprechen **über** das Projekt. |
| **sprechen / reden** | **von** (+Dat) | 谈及 | Er spricht **von** seinen Erfahrungen. |
| **sterben** | **an** (+Dat) | 死于（疾病） | Er starb **an** Krebs. |
| **sich streiten** | **um** (+Akk) / **über** (+Akk) | 争吵 | Sie streiten sich **um** die Zuständigkeit. |
| **teilnehmen** | **an** (+Dat) | 参加 | Ich nehme **an** der Konferenz **teil**. |
| **sich treffen** | **mit** (+Dat) | 见面 | Ich treffe mich **mit** Kollegen. |
| **sich trennen** | **von** (+Dat) | 与...分离 | Er trennt sich **von** seiner Frau. |
| **sich unterscheiden** | **von** (+Dat) | 与...不同 | A unterscheidet sich **von** B. |
| **verfügen** | **über** (+Akk) | 拥有（正式） | Die Stadt verfügt **über** ein modernes Museum. |
| **vergleichen** | **mit** (+Dat) | 与...比较 | Man vergleicht das **mit** einer früheren Studie. |
| **sich verlassen** | **auf** (+Akk) | 信赖 | Ich verlasse mich **auf** deine Unterstützung. |
| **verzichten** | **auf** (+Akk) | 放弃 | Er verzichtet **auf** sein Recht. |
| **sich vorbereiten** | **auf** (+Akk) | 做准备 | Ich bereite mich **auf** die Prüfung **vor**. |
| **warnen** | **vor** (+Dat) | 警告 | Die Behörde warnt **vor** der Gefahr. |
| **warten** | **auf** (+Akk) | 等待 | Ich warte **auf** den Bus. |
| **sich wundern** | **über** (+Akk) | 惊讶 | Ich wundere mich **über** seine Entscheidung. |
| **zweifeln** | **an** (+Dat) | 怀疑 | Ich zweifle **an** seiner Ehrlichkeit. |

> [!note] 可分动词的介词位置
> 可分动词在句子中使用时，前缀与介词不冲突。如 `teilnehmen an`：Ich **nehme** an der Sitzung **teil**。

### 2. 形容词 + 固定介词 (Adjektive mit festen Präpositionen)

| 形容词 | 介词+格 | 含义 | 例句 |
|--------|---------|------|------|
| **abhängig** | **von** (+Dat) | 依赖于 | Das Ergebnis ist **abhängig von** vielen Faktoren. |
| **angewiesen** | **auf** (+Akk) | 依赖于 | Er ist **auf** Hilfe **angewiesen**. |
| **ärgerlich** | **über** (+Akk) | 对...生气的 | Sie ist **ärgerlich über** den Fehler. |
| **aufmerksam** | **auf** (+Akk) | 对...注意的 | Wir machen **auf** das Problem **aufmerksam**. |
| **begeistert** | **von** (+Dat) | 对...热衷的 | Sie ist **begeistert von** der Idee. |
| **bekannt** | **für** (+Akk) / **bei** (+Dat) | 以...出名 | Die Region ist **bekannt für** ihren Wein. |
| **beschäftigt** | **mit** (+Dat) | 忙于 | Er ist **mit** der Forschung **beschäftigt**. |
| **betroffen** | **von** (+Dat) | 受...影响的 | Die Branche ist stark **von** der Krise **betroffen**. |
| **böse / wütend** | **auf** (+Akk) | 对...生气 | Der Chef ist **böse auf** ihn. |
| **charakteristisch** | **für** (+Akk) | 是...的特征 | Dies ist **charakteristisch für** die Epoche. |
| **dankbar** | **für** (+Akk) | 对...感激 | Ich bin **dankbar für** Ihre Unterstützung. |
| **enttäuscht** | **von** (+Dat) / **über** (+Akk) | 对...失望 | Wir sind **enttäuscht von** dem Ergebnis. |
| **entscheidend** | **für** (+Akk) | 对...关键的 | Dieser Faktor ist **entscheidend für** den Erfolg. |
| **erfahren** | **in** (+Dat) | 在...方面有经验的 | Sie ist **erfahren in** der Projektarbeit. |

| **frei** | **von** (+Dat) | 免于 | **frei von** Vorurteilen |
| **froh** | **über** (+Akk) | 对...高兴 | Ich bin **froh über** deinen Besuch. |
| **geeignet** | **für** (+Akk) | 适合 | Diese Methode ist **für** Anfänger **geeignet**. |
| **gespannt** | **auf** (+Akk) | 期待/好奇 | Wir sind **gespannt auf** die Ergebnisse. |
| **interessiert** | **an** (+Dat) | 对...感兴趣 | Er ist **interessiert an** Geschichte. |
| **neidisch** | **auf** (+Akk) | 嫉妒 | Sie ist **neidisch auf** ihren Erfolg. |
| **neugierig** | **auf** (+Akk) | 好奇 | Ich bin **neugierig auf** deine Meinung. |

> [!tip] 介词辨义：interessant an vs. interessiert an
> 注意区分 `interessant an` 和 `interessiert an`：
> - **interessant an** (+Dat) — 在某方面有趣的：*Was ist **interessant an** diesem Buch?*（这本书哪里有趣？）
> - **interessiert an** (+Dat) — 对...感兴趣的：*Ich bin **interessiert an** dem Thema.*（我对这个主题感兴趣。）

> 多数名词的介词与其同源动词/形容词一致：
> - `interessiert **an**` → `das Interesse **an**`

> 学习时可将动词-名词-形容词三者的介词搭配对照记忆。

## 四、写作模板 (Schreibvorlagen für Aufsätze)

### 议论文框架 (Erörterung)

```
**Einleitung**
In der heutigen Gesellschaft wird das Thema [Thema] kontrovers diskutiert.
Während die einen der Meinung sind, dass ..., vertreten die anderen den Standpunkt, dass ...
Im Folgenden sollen die wichtigsten Argumente für und gegen ... dargelegt werden.

**Hauptteil — Pro-Argumente**
Zunächst spricht für ... , dass ...
Ein weiteres Argument ist ...
Darüber hinaus ist zu bedenken, dass ...

**Hauptteil — Kontra-Argumente**
Auf der anderen Seite lässt sich nicht leugnen, dass ...
Allerdings muss man auch berücksichtigen, dass ...
Dagegen spricht ferner, dass ...

**Schluss**
Zusammenfassend lässt sich sagen, dass ...
Auch wenn ... nicht von der Hand zu weisen ist, bin ich letztlich zu dem Schluss gelangt, dass ...
```

### 图表描述 (Grafikbeschreibung)

```
**Einleitung**
Das vorliegende Diagramm / Die vorliegende Grafik / Die Tabelle gibt Auskunft über ...
Die Daten stammen aus dem Jahr / beziehen sich auf den Zeitraum von ... bis ...

**Überblick**
Auffällig / Interessant ist, dass ...
Besonders hervorzuheben ist der deutliche Anstieg / Rückgang von ...
Ein weiterer wichtiger Aspekt ist ...

**Details**
Im Jahr ... lag / betrug der Anteil / Wert noch bei ...
Im Gegensatz dazu ist im Jahr ... ein deutlicher Anstieg um ... zu verzeichnen.
Den höchsten / niedrigsten Wert erreichte ... im Jahr ...
Bemerkenswert ist auch die Entwicklung von ..., die sich von ... auf ... verändert hat.

**Schluss**
Zusammenfassend ist ein klarer Trend zu ... zu erkennen.
Die dargestellten Daten zeigen, dass sich ... in den letzten Jahren ... entwickelt hat.
```

### 论证句型综合

| 功能 | 德语 | 中文 |
|------|------|------|
| **引入论点** | Als erstes Argument ist zu nennen, dass... | 首先要提出的论点是... |
| **加强** | Dieses Argument wird dadurch gestützt, dass... | 这一论点得到...的支持 |
| **弱化** | Dieses Argument mag zwar zutreffen, allerdings... | 这个论点或许成立，但... |
| **反驳** | Dem ist entgegenzuhalten, dass... | 对此需要反驳的是... |
| **权衡** | Abwägend ist festzustellen, dass... | 经权衡可以确定... |
| **承认+反驳** | Zwar ist ... zuzustimmen, jedoch darf nicht übersehen werden, dass... | 虽然同意...但不能忽视... |

## 五、常见错误

### 1. 介词搭配错误

⚠️ 注意区分形近介词搭配：
- **sich freuen auf** (未来) vs. **sich freuen über** (已发生)
- **denken an** (想到) vs. **nachdenken über** (思考)
- **glauben an** (信念/存在) vs. **glauben + Dat** (相信某人说的话)

### 2. 介词+格错误

❌ *trotz dem Wetter* → ✅ trotz **des** Wetters (第二格)
❌ *wegen dem Regen* → ✅ wegen **des** Regens (第二格，口语可接受第三格)
❌ *aus mein Land* → ✅ aus **meinem** Land (第三格)

### 3. 从句连词 + 语序错误

❌ *Obwohl es regnet, aber wir gehen spazieren.* → 德语不双用 obwohl + aber
✅ **Obwohl** es regnet, gehen wir spazieren.

❌ *Er ist krank, weil er hat keine Zeit.* → weil 从句动词在句末
✅ Er ist krank, **weil** er keine Zeit **hat**.

### 4. 副词连接词语序错误

❌ *Außerdem, ich möchte hinzufügen...*
✅ **Außerdem** möchte ich hinzufügen...（反语序，V→S）

❌ *Deshalb, die Lösung ist einfach.*
✅ **Deshalb** ist die Lösung einfach.（反语序）

### 5. 句子框结构破坏

❌ *Ich habe teilgenommen an der Konferenz.* →
✅ Ich habe **an der Konferenz** teilgenommen.（句框：habe ... teilgenommen）

❌ *Wir müssen warten auf die Ergebnisse.*（口语可）
✅ Wir müssen **auf die Ergebnisse** warten.（句框：müssen ... warten）
