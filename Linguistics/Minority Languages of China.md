## 1. 引言

中国是世界上语言资源最丰富的国家之一，拥有**130多种语言**，分别属于五大语系：汉藏语系、阿尔泰语系、南岛语系、南亚语系和印欧语系。

### 1.1 语言濒危状况

| 使用人口 | 语言数量 | 状态 |
|----------|----------|------|
| <100人 | 7种 | 极度濒危 |
| 100-1000人 | 18种 | 濒危 |
| <5000人 | 48种 | 脆弱 |
| <10000人 | 68种 | 需关注 |

> **已消亡语言**：木佬语（2015年后）、满语（作为母语）、羿语、哈卡斯语

### 1.2 主要濒危语言列表

| 语言 | 使用人口 | 语系 | 分布地区 |
|------|----------|------|----------|

## 2. 汉藏语系少数民族语言

### 2.1 藏缅语族

#### 2.1.1 藏语支

**代表语言**：藏语、门巴语、嘉绒语

| 特征 | 描述 |
|------|------|
| **词序** | SOV（主语-宾语-动词） |
| **作格性** | 作格-绝对格对齐（经典作格语言） |
| **声调** | 安多藏语无声调；拉萨藏语有声调 |
| **形态** | 有格标记、动词人称变化 |
| **复辅音** | 古藏语有复杂复辅音（如 sgr-） |

**作格系统示例（藏语）**：
```
不及物：bod-pa 'gro (藏族人 走) —— S无标记
及物：bod-pa-s me-tog bzung (藏族人-作格 花 摘) —— A有标记-s
```

**嘉绒语特点**：
- 保留古藏语特征最多
- 复杂的形态变化系统
- 有方向前缀系统
- 名词可在关系句前或后

#### 2.1.2 羌语支

**代表语言**：羌语、普米语、木雅语、史兴语、尔苏语

| 特征 | 描述 |
|------|------|
| **声调** | 部分有声调（普米语：高低二调），部分无 |
| **词序** | SOV |
| **形态** | 黏着语，有丰富的格标记 |
| **音节** | 单音节语素为主 |
| **方向范畴** | 有复杂的方向前缀 |

**羌语特殊现象**：
- 语法化程度高的方向标记
- 名词修饰语位置灵活
- 有丰富的语气词

#### 2.1.3 彝语支

**代表语言**：彝语、哈尼语、傈僳语、纳西语、拉祜语

| 特征 | 描述 |
|------|------|
| **声调** | 都有声调，彝语有4个调 |
| **松紧元音** | 彝语有松紧元音对立 |
| **词序** | SVO或SOV |
| **量词** | 发达的量词系统 |
| **形态** | 分析语特征明显 |

#### 2.1.4 缅语支

**代表语言**：载瓦语、阿昌语、浪速语

| 特征 | 描述 |
|------|------|
| **声调** | 有声调系统 |
| **词序** | SOV |
| **形态** | 有少量屈折变化 |

### 2.2 壮侗语族

**代表语言**：壮语、布依语、傣语、侗语、水语、黎语

| 特征 | 描述 |
|------|------|
| **声调** | 四声八类系统，与汉语相近 |
| **词序** | SVO |
| **量词** | 发达的量词系统，具有冠词功能 |
| **音节结构** | 声母+韵母+声调 |
| **形态** | 分析语，缺乏形态变化 |

**壮语量词特点**：
```
结构：数词 + 量词 + 名词
例：sam boux vunz (三 个 人)
```

### 2.3 苗瑶语族

**代表语言**：苗语、布努语、勉语（瑶语）、畲语

| 特征 | 描述 |
|------|------|
| **声调** | 复杂的声调系统，可多达10+个 |
| **韵尾** | 苗语无塞音韵尾；瑶语有-m, -n, -ŋ, -p, -t, -k |
| **词序** | SVO；形容词后置 |
| **量词** | 丰富的量词系统 |
| **鼻音** | 有送气鼻音 |

**苗语声调系统**：
- 声调数量多（可达8个或以上）
- 有声调分化现象
- 声调与声母、韵母有对应关系

## 3. 阿尔泰语系少数民族语言

### 3.1 满-通古斯语族

**代表语言**：满语（濒危）、锡伯语、赫哲语、鄂温克语、鄂伦春语

| 特征 | 描述 |
|------|------|
| **词序** | SOV |
| **元音和谐** | 有元音和谐律 |
| **格系统** | 满语支5-7个格；通古斯语支多达16个格 |
| **形态** | 黏着语，使用后加成分 |
| **复元音** | 满语支复元音多；通古斯语支有长元音 |

**满语格系统**：
```
主格（无标记）、宾格-be、与格-de、属格-i、工具格-i
```

**赫哲语特点**：
- 极度濒危（约100人使用）
- 有丰富的动词形态变化
- 动词有时、体、态、式变化

### 3.2 蒙古语族

**代表语言**：蒙古语、达斡尔语、东乡语、土族语、保安语

| 特征 | 描述 |
|------|------|
| **词序** | SOV |
| **元音和谐** | 严格的元音和谐律 |
| **格系统** | 8个格 |
| **形态** | 黏着语 |
| **文字** | 传统蒙古文（竖写） |

**蒙古语元音和谐**：
```
阳性元音：a, o, u
阴性元音：e, ö, ü
中性元音：i
同一词内元音必须同性
```

### 3.3 突厥语族

**代表语言**：维吾尔语、哈萨克语、柯尔克孜语、撒拉语、塔塔尔语

| 特征 | 描述 |
|------|------|
| **词序** | SOV |
| **元音和谐** | 严格的元音和谐律 |
| **格系统** | 6个格 |
| **形态** | 黏着语 |
| **动词系统** | 有时、体、态、式、人称变化 |

## 4. 南岛语系（台湾原住民语言）

### 4.1 概述

台湾是南岛语系的**原乡**，拥有南岛语系四个分支中的三个，语言最为分歧，保留古语成分最高。

**官方认定的16种原住民语言**：
- 泰雅语、赛夏语、赛德克语、邵语
- 布农语、邹语、卡那卡那富语、拉阿鲁瓦语
- 鲁凯语、排湾语、阿美语、卑南语
- 噶玛兰语、撒奇莱雅语、雅美语（达悟语）、太鲁阁语

### 4.2 语法特点

| 特征 | 描述 |
|------|------|
| **焦点系统** | 最核心特征：动词标记句子焦点 |
| **词序** | VOS或VSO |
| **代词系统** | 复杂的代词系统（包含式/排除式） |
| **格标记** | 通过焦点系统而非格标记 |
| **音节** | 简单，多为CV或CVC |

**焦点系统示例**：

| 焦点类型 | 标记 | 功能 |
|----------|------|------|
| 施事焦点 | -um- | 主语被聚焦 |
| 受事焦点 | -in- | 宾语被聚焦 |
| 方位焦点 | -an | 地点被聚焦 |
| 工具焦点 | i- | 工具被聚焦 |

> **IOLC考点**：焦点系统分析是南岛语系语言的高频考点。

## 5. 南亚语系

### 5.1 概述

**代表语言**：佤语、布朗语、德昂语、京语（越南语）

### 5.2 语法特点

| 特征 | 描述 |
|------|------|
| **声调** | 部分有声调，部分无声调 |
| **词序** | SVO |
| **量词** | 有量词系统 |
| **音节** | 单音节语素为主 |
| **形态** | 分析语特征 |

## 6. IOLC高频考点总结

### 6.1 按语言特征的考点

| 特征类型 | 可能考点 | 典型语言 |
|----------|----------|----------|
| **作格系统** | 判断A、S、O的标记方式 | 藏语、嘉绒语 |
| **元音和谐** | 分析元音变化规则 | 蒙古语、满语、维吾尔语 |
| **焦点系统** | 识别焦点标记和功能 | 台湾原住民语言 |
| **声调系统** | 分析声调对应关系 | 苗语、壮语、侗语 |
| **量词系统** | 分析量词与名词的搭配 | 壮语、苗语、汉语方言 |
| **格系统** | 分析格标记的分布 | 满语、蒙古语、藏语 |

### 6.2 按语系的考点分布

| 语系 | 高频考点 | 难度 |
|------|----------|------|
| **藏缅语族** | 作格系统、格标记、SOV词序 | ⭐⭐⭐⭐ |
| **满-通古斯语族** | 元音和谐、格系统、黏着形态 | ⭐⭐⭐ |
| **壮侗语族** | 声调分析、量词系统、SVO词序 | ⭐⭐ |
| **苗瑶语族** | 复杂声调、形容词后置 | ⭐⭐⭐ |
| **台湾南岛语** | 焦点系统、VOS词序 | ⭐⭐⭐⭐ |

### 6.3 解题方法论

#### 作格语言分析

1. 找出不及物动词句子（确定S标记）
2. 找出及物动词句子（确定A和O标记）
3. 对比三者的标记方式
4. 判断对齐类型

#### 元音和谐分析

1. 列出词根元音
2. 列出后缀元音变体
3. 找出对应规律
4. 写出和谐规则

#### 焦点系统分析

1. 找出动词词根
2. 识别动词上的词缀
3. 分析论元的格标记
4. 确定焦点类型

### 6.4 常见陷阱

| 陷阱类型 | 描述 | 应对方法 |
|----------|------|----------|
| **词序灵活** | 有格标记的语言词序可能灵活 | 关注格标记而非词序 |
| **声调变化** | 声调可能因语法环境变化 | 注意声调的语法功能 |
| **形态融合** | 多个语素可能融合为一个音节 | 仔细分析音节结构 |
| **焦点与格的混淆** | 焦点系统可能看起来像格标记 | 分析论元与动词的关系 |
