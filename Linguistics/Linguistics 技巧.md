# Linguistics 技巧

## Phonological Match
1. 区分**开元音和闭元音**
2. 注意音节的位置：在**句首/句尾还是句中**
3. 观察辅音特征：**清浊、送气、发音部位**的变化规律
4. 检查**互补分布**：同一音位的条件变体往往出现在互补环境中
5. 注意**同化方向**：顺行（前影响后）vs 逆行（后影响前）
6. 识别**元音和谐**：后缀元音随词根元音特征变化
7. 常见音变类型：
   - 删除：词首/词中/词尾脱落
   - 插入：辅音丛间增音
   - 换位：相邻音素交换位置
   - 弱化：元音弱化、辅音擦音化

## Syntactic Match
1. 划分语素，区分名词的**词干和词缀**
2. 词缀
    - 公母：很多时候，野生动物会被用特殊的词缀标注公母，但不同性别的家畜却会有单独的名称
3. **词序分析**：确定 S、V、O 的基本顺序，注意修饰语位置
4. **格标记识别**：主格、宾格、作格等标记的分布规律
5. **一致关系**：动词与主语/宾语的人称、数、性一致
6. **代词系统**：区分包含式/排除式，注意聚焦代词
7. **动词变位**：时态、体、语气、人称的复合标记
8. **对齐类型判断**：
   - 主格-宾格：S=A≠O
   - 作格-绝对格：S=O≠A
9. 对比**长短句结构**：不及物句（短）vs 及物句（长），识别论元标记

## Number System
1. 最常出现者为基数，**主选为20**
2. 连接不同单元者**是算子**，具体是哪种四则运算并不重要
3. 将数字表示为$$20x\pm 10+b$$
4. 详细区分个位数字与基数的乘数，即区分$b$与$x$，其中：
    - $x\in [1,4]$，大概是一个较小的数字
5. 在$20x\pm 10+b$中，以$5$作为分界线，可以取不同的原点将数字表示为原点$\pm c$
6. 个位数字往往$\in [1,9]$，因为$0$常常不会直接出现于数字的书写中
