# Verb and verb phrase

## 1. 引言

### 1.1 基本概念

| 概念 | 定义 |
|------|------|
| **动词（Verb）** | 表示动作、存在或状态 |
| **主语（Subject）** | 通常表示执行动作的人或事物 |
| **直接宾语（Direct Object）** | 通常表示被动作作用的人或事物 |
| **间接宾语（Indirect Object）** | 表示动作间接影响的对象 |

### 1.2 及物性与不及物性

| 类型 | 定义 | 例子 |
|------|------|------|
| **及物动词（Transitive）** | 有直接宾语 | I broke the glass |
| **不及物动词（Intransitive）** | 无直接宾语 | The baby yawned |

> **注意**：有些动词可以兼作及物和不及物，如 *eat*：
> - She is eating a pizza（及物）
> - She is eating（不及物）

## 2. 动词的变量

动词问题中的变量可分为三类：

| 类别 | 内容 |
|------|------|
| **TAM** | 时（Tense）、体（Aspect）、式（Mood） |
| **论元** | S（主语）、S+O（主语+宾语）、S+O+O（双宾语） |
| **其他** | 否定、疑问等 |

## 3. TAM（时、体、式）

### 3.1 时（Tense）

时是与时间概念相关的语法范畴。

#### 3.1.1 基本时态

| 时态 | 描述 |
|------|------|
| **过去时** | 动作发生在说话之前 |
| **现在时** | 动作正在发生 |
| **将来时** | 动作将在说话之后发生 |

#### 3.1.2 语言的时态系统差异

| 系统 | 语言示例 |
|------|----------|
| 过去/非过去 | 阿拉伯语、日语 |
| 将来/非将来 | 格陵兰语、尼夫赫语 |
| 现在/非现在 | Ithkuil（人造语言） |
| 无时态区分 | 汉语、Dyirbal语 |

#### 3.1.3 特殊时态

| 时态名称 | 描述 |
|----------|------|
| **Hesternal** | 昨天（前一天） |
| **Crastinal** | 明天（后一天） |
| **Hodiernal** | 今天 |
| **Pre-hesternal** | 前天 |
| **Post-crastinal** | 后天 |

**Yagua语的过去时系统**：
- -jásiy：几小时前（当天）
- -jay：一天前
- siy：一周到一个月前
- tíy：一到两年前
- -jada：两年以上（远过去/传说过去）

### 3.2 体（Aspect）

体表示动作、状态或事件在时间上的演进，与说话时刻无关。

#### 3.2.1 主要体类型

| 类型 | 描述 |
|------|------|
| **完整体（Perfective）** | 事件是有界的 |
| **非完整体（Imperfective）** | 事件正在进行或习惯性发生 |

#### 3.2.2 其他体类型

| 类型 | 描述 | 英语表达 |
|------|------|----------|
| **进行体（Progressive）** | 动作正在进行 | be V-ing |
| **一次体（Semelfactive）** | 短暂动作 | - |
| **偶然体（Accidental）** | 意外动作 | - |
| **持续体（Durative）** | 持续一段时间 | for a while |
| **潜能体（Potential）** | 可能发生 | probably |
| **前瞻体（Prospective）** | 准备做某事 | I'm getting ready to... |

> **注意**：英语"时态"实际上是时和体的组合：
> - 完成体：I have/had eaten a sandwich
> - 进行体：I am/was eating a sandwich

### 3.3 式（Mood）

式是与情态相关的屈折范畴，反映说话者对信息的态度。

#### 3.3.1 现实式（Realis）

| 类型 | 描述 |
|------|------|
| **陈述式（Indicative）** | 最常见，用于陈述事实 |
| **普遍真理式** | 用于普遍真理（如 Fish swim） |

#### 3.3.2 非现实式（Irrealis）

| 类型 | 描述 |
|------|------|
| **虚拟式（Subjunctive）** | 虚拟/假设事件、观点、情感 |
| **条件式（Conditional）** | 动作受条件制约 |
| **愿望式（Optative）** | 表示愿望 |
| **命令式（Imperative）** | 直接命令/请求/禁止 |
| **指令式（Jussive）** | 对第三人称的命令 |

### 3.4 情态的语言表达

在许多语言中，情态动词用于表达不同类型的情态。

**英语示例**：
- I must leave（必须）
- you should eat（应该）
- I want to go（想要）
- I need to keep swimming（需要）

**链式动词（Catenative verbs）**：可以形成动词链的动词
- I want to try to swim（want + try + swim）

### 3.5 示证性（Evidentiality）

示证性表示信息来源的方式。

#### 3.5.1 Pomo语的示证系统

| 类型 | 描述 |
|------|------|
| **视觉（Visual）** | 说话人亲眼目睹 |
| **感官非视觉（Sensorial non-visual）** | 通过听觉、嗅觉等感知 |
| **推论（Inferential）** | 通过结果或后果推断 |
| **传闻（Reportative）** | 从他人处得知 |

#### 3.5.2 土耳其语的示证系统

| 类型 | 名称 | 描述 |
|------|------|------|
| 见证过去 | görülen geçmiş zaman | 亲眼目睹 |
| 听说过去 | duyulan geçmiş zaman | 非亲眼目睹 |

#### 3.5.3 信息传递层级

| 层级 | 描述 |
|------|------|
| **第一手** | 直接观察（见证） |
| **第二手** | 从见证者处得知 |
| **第三手** | 从间接来源得知 |

## 4. 论元（Arguments）

### 4.1 问题难度分类

| 难度 | 特点 |
|------|------|
| **简单** | 主语和宾语由独立的词缀标记 |
| **中等** | S和O合并为单个词缀，或发生语音变化 |
| **困难** | S和O通过词缀组合表达 |

### 4.2 Swahili语示例

**动词结构**：S–Tense–O–V

**人称标记**：

| | 1 | 2 | 3 |
|--|---|---|---|
| sg | ni | wu | a |
| pl | - | - | wa |

**时态标记**：
- na = 现在
- li = 过去
- taka = 将来

**示例**：
- ninawuona = ni-na-wu-ona = 'I see you'
- aliniona = a-li-ni-ona = 'She saw me'

## 5. 切分（Segmenting）

切分是将动词分解为其组成词素的过程。

### 5.1 切分步骤

1. 识别重复出现的字符串
2. 分离特殊字符
3. 建立频率表
4. 推导动词结构

### 5.2 Dabida语示例

**动词结构**：S-cha-O-V-(na)

**人称标记（S=O）**：

| | 1 | 2 | 3 |
|--|---|---|---|
| sg | ni | ku | - |
| pl | di | mu | βi |

**动词词干**：
- rasha = 'to argue'（不及物）/ 'to curse'（及物）
- kunda = 'to fall in love'
- kaδa = 'to fight'（不及物）/ 'to beat'（及物）

**不及物标记**：-na

## 6. 论元模式

### 6.1 分裂模式 vs 合并模式

| 模式 | 描述 |
|------|------|
| **分裂模式** | 单一概念用多个词素表达（如人称+数） |
| **合并模式** | 多个概念合并为单个词素（如主语+宾语） |

### 6.2 Ge'ez语示例

**动词结构**：V-S-O

**主语和宾语标记**：

| | 1 | 2 | 3M | 3F |
|--|---|---|-----|-----|
| sg S | -ku | -ka | -a, O: -o | - |
| pl S | -na | -kǝmu | -u, O: -omu | O: -on |

**语音规则**：添加宾语标记时，主语词尾元音变化：
- a → ∅
- u → ǝw

### 6.3 Itelmen语示例

**动词结构**：

| 位置 | 功能 |
|------|------|
| I | 主语（n- = 3pl, ∅ = 3sg） |
| II | 式（-k'- = 条件式, ∅ = 陈述式） |
| III | 词干 |
| IV | 时/体 |
| V | 宾语人称 |
| VI | 宾语数 |

**时/体标记**：
- -ķzo- = 非完整体
- -ķzoz- = 现在时
- ∅ = 完整体

## 7. 代词层级（Pronoun Hierarchy）

某些语言中，代词词素的顺序不取决于主/宾角色，而是取决于预定义的层级。

### 7.1 常见层级

| 层级 | 特点 |
|------|------|
| **1>2>3** | 说话者优先（说话者导向语言） |
| **2>1>3** | 听者优先（听众导向语言） |

### 7.2 层级标记机制

| 情况 | 标记 |
|------|------|
| 层级被遵守（S > O） | 形式X |
| 层级未被遵守（S < O） | 形式X' |

### 7.3 Proto-Algonquian语示例

**代词层级**：2>1>3

**层级信息标记**：

| | ∃ 3 | ∄ 3 |
|--|-----|------|
| S>O | -a:- | -i- |
| S<O | -ekw- | -eθ- |

## 8. 动词语义

### 8.1 影响动词语义的词素

| 类型 | 描述 | 例子 |
|------|------|------|
| **增强词（Intensifier）** | 动作"过度"或"很多" | eat → devour |
| **减弱词（Mitigator）** | 动作强度减弱 | eat → taste |
| **致使标记（Causative）** | "使某人做..." | eat → feed |

### 8.2 静态动词 vs 动态动词

| 类型 | 特点 | 例子 |
|------|------|------|
| **动态动词** | 涉及动作、运动 | hit, kill, look, say, eat |
| **静态动词** | 表示状态或感觉 | love, hate, like, be good, be ill |

某些语言对这两类动词使用不同的TAM标记或词素顺序。

### 8.3 施事者（Agent）与受事者（Patient）

| 概念 | 定义 | 性质 |
|------|------|------|
| **施事者（A）** | 执行动作的论元 | 语义论元 |
| **受事者（P）** | 被动作作用的论元 | 语义论元 |
| **主语/宾语** | 句法角色 | 句法论元 |

**示例**：
- The cat eats the fish：S=A, O=P
- The fish is eaten by the cat：S=P, 介词宾语=A

### 8.4 流动-S对齐（Fluid-S Alignment）

不及物动词的论元可标记为施事者或受事者。

| 类型 | 默认标记 | 特殊标记含义 |
|------|----------|--------------|
| **施事者默认** | 施事标记 | 强调缺乏意愿/控制 |
| **受事者默认** | 受事标记 | 强调有一定意愿/控制 |

**英语示例**：
- get lost, get drowned（受事标记）
- fall, trip（意外动作，倾向受事标记）

## 9. 解题方法论

### 9.1 切分动词的步骤

1. 识别重复字符串
2. 分离词素
3. 建立频率表
4. 推导动词结构
5. 匹配翻译

### 9.2 混乱-有序问题（Chaos-and-Order）

1. 切分动词
2. 推导动词结构
3. 显示频率表

### 9.3 规则书写建议

**选项1**：分别列出结构和各词素含义
```
• Structure: S–Tense–O–V
• S: 1sg = ni, 2sg = wu...
• Tense: present = na...
```

**选项2**：用表格展示
```
S      Tense  O      V
ni=wu  na=现  ni=1sg sema=speak
```

**选项3**：用表格显示人称标记
```
    1   2   3
sg  ni  wu  a
pl  -   -   wa
```
