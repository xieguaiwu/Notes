# Comparative and Historical Linguistics（比较语言学与历史语言学）

## 1. 引言

比较语言学和历史语言学是研究语言演变和语言关系的两个密切相关的学科。

[[Language Families]]
[[Phonology]]

### 1.1 学科定义

| 学科 | 定义 | 核心问题 |
|------|------|----------|
| **比较语言学** | 通过比较不同语言的特征来研究语言之间的关系 | 语言之间有何关系？ |
| **历史语言学** | 研究语言随时间的演变过程 | 语言如何变化？ |

### 1.2 两者关系

比较语言学和历史语言学常常交织在一起：

| 研究方向 | 方法 |
|----------|------|
| **共时比较** | 比较同一时期的语言（比较语言学） |
| **历时比较** | 比较不同时期的语言状态（历史语言学） |
| **历史比较法** | 通过比较重构语言的历史状态（两者结合） |

## 2. 语言的分化

### 2.1 基本概念

| 术语 | 定义 |
|------|------|
| **语言分化** | 一种语言分裂为多种语言的过程 |
| **祖语/原始语（Proto-language）** | 分化前的共同祖先语言 |
| **子语/派生语（Daughter language）** | 从祖语分化出来的语言 |
| **亲属语言（Cognate languages）** | 有共同祖语的语言 |
| **语系（Language Family）** | 由亲属语言组成的群体 |

### 2.2 分化模式

```
           祖语（Proto-language）
              │
     ┌────────┼────────┐
     │        │        │
   子语A    子语B    子语C
     │        │
   ┌─┴─┐    ┌─┴─┐
   │   │    │   │
  A1  A2   B1  B2
```

### 2.3 分化的原因

| 原因类型 | 描述 | 示例 |
|----------|------|------|
| **地理隔离** | 语言社区被地理障碍分隔 | 拉丁语分化为罗曼语族 |
| **政治分裂** | 政治边界导致语言分离 | 德语方言分化 |
| **人口迁移** | 语言使用者迁移到不同地区 | 波利尼西亚语言分化 |
| **社会分化** | 社会群体之间的隔离 | 方言形成 |

### 2.4 分化的时间尺度

| 分化程度 | 时间跨度 | 示例 |
|----------|----------|------|
| **方言差异** | 数百年 | 汉语方言 |
| **语言差异** | 一千年以上 | 日耳曼语言 |
| **语族差异** | 两千年以上 | 印欧语系各语族 |
| **语系差异** | 五千年以上 | 不同语系之间 |

### 2.5 同源词（Cognates）

同源词是来自同一祖语、具有相同词源的词。

**识别同源词的标准**：

| 标准 | 描述 |
|------|------|
| **语义相似** | 意义相同或相近 |
| **语音对应** | 语音有规律对应关系 |
| **非借用** | 不是后来借用的词 |

**印欧语系同源词示例**：

| 英语 | 德语 | 拉丁语 | 希腊语 | 梵语 | 重建形式 |
|------|------|--------|--------|------|----------|
| mother | Mutter | māter | mḗtēr | mātár | *meh₂tér |
| father | Vater | pater | patḗr | pitár | *ph₂tér |
| three | drei | trēs | treîs | tráyas | *tréyes |
| new | neu | novus | néos | náva | *néwos |
| night | Nacht | nox | núx | nákt | *nókʷts |

## 3. 历史比较法

### 3.1 定义与目的

历史比较法是比较语言学和历史语言学的核心方法，通过比较亲属语言的对应形式，重构祖语的形式和特征。

| 目的 | 描述 |
|------|------|
| **重构祖语** | 推断原始语言的形式 |
| **确立亲属关系** | 证明语言之间的亲属关系 |
| **建立演变规律** | 发现音变、形态变化的规律 |
| **确定分化顺序** | 判断语言分化的先后顺序 |

### 3.2 历史比较法的步骤

| 步骤 | 操作 | 说明 |
|------|------|------|
| **Step 1** | 收集同源词 | 收集各亲属语言中意义相近、形式相似的词 |
| **Step 2** | 建立语音对应 | 找出各语言间有规律的语音对应关系 |
| **Step 3** | 重构祖语形式 | 基于对应关系推断原始形式（用*标记） |
| **Step 4** | 确立音变规则 | 描述从祖语到子语的演变规则 |
| **Step 5** | 检验重构 | 用新发现检验重构的合理性 |

### 3.3 语音对应原则

> **核心原则**：如果语言A和语言B是亲属语言，它们之间的语音差异不是随机的，而是有规律的。

**示例：印欧语 *p 的对应**

| 语言 | 形式 | 对应音 |
|------|------|--------|
| 梵语 | pitár | p |
| 希腊语 | patḗr | p |
| 拉丁语 | pater | p |
| 英语 | father | f |
| 德语 | Vater | f |

**规律**：原始印欧语 *p 在日耳曼语言中变为 f（格林定律）

### 3.4 重构原则

| 原则 | 描述 |
|------|------|
| **多数原则** | 如果多数子语有某特征，假设祖语也有该特征 |
| **自然演变原则** | 重构的形式应能自然演变为各子语形式 |
| **经济原则** | 选择需要最少假设的重构 |
| **方向性原则** | 音变通常是单向的（如 p→f 比 f→p 更自然） |

### 3.5 重构标记法

| 符号 | 含义 |
|------|------|
| **\*** | 重构形式（未直接证实） |
| **\*\*** | 错误的重构形式 |
| **→** | 演变为 |
| **<** | 来源于 |

**示例**：
```
*p → f / #_  （词首p变为f）
英语 father < *ph₂tér
```

### 3.6 历史比较法的局限

| 局限 | 描述 |
|------|------|
| **时间深度限制** | 难以重构超过6000-8000年的语言 |
| **借用干扰** | 借词可能被误认为同源词 |
| **平行演变** | 相似变化可能独立发生 |
| **证据缺失** | 许多语言没有文字记录 |

## 4. 音位对立关系的演变

### 4.1 音位对立的概念

| 术语 | 定义 |
|------|------|
| **音位对立** | 两个音位在相同环境中能区别意义 |
| **对立关系** | 音位之间的区别性关系 |

**示例**：英语 /p/ 和 /b/ 对立（pat vs bat）

### 4.2 对立关系的演变类型

| 演变类型 | 描述 | 结果 |
|----------|------|------|
| **对立的获得** | 原本相同的音位变得对立 | 增加音位数量 |
| **对立的丧失** | 原本对立的音位合并 | 减少音位数量 |
| **对立的转移** | 对立条件发生变化 | 对立环境改变 |

### 4.3 对立的获得

#### 合并-分裂模式

当一个音位在不同条件下发生不同变化时，可能产生新的对立。

**示例：英语元音大推移**

| 中古英语 | 现代英语 | 条件 |
|----------|----------|------|
| /iː/ | /aɪ/ | bite |
| /eː/ | /iː/ | beet |

中古英语 /eː/ 变为 /iː/，与原有的 /iː/（变为 /aɪ/）形成新的对立。

#### 借词引入新对立

| 语言 | 过程 |
|------|------|
| 日语 | 借词引入了 /v/ 音位（如 video） |
| 汉语 | 借词可能引入新的辅音丛 |

### 4.4 对立的丧失

#### 音位合并

两个原本对立的音位合并为一个。

**示例**：

| 语言 | 合并过程 | 结果 |
|------|----------|------|
| 汉语（官话） | /pʰ/ /p/ → /p/（部分环境） | 浊塞音清化 |
| 英语方言 | /w/ /ʍ/ → /w/ | wh-words 与 w-words 合并 |
| 法语 | /ɛ/ /e/ → /e/（词末） | 部分元音合并 |

#### 条件合并

只在特定条件下合并。

**示例**：德语元音合并

| 环境 | 合并 |
|------|------|
| 词末开音节 | /eː/ /ɛː/ → /eː/ |
| 其他位置 | 保持对立 |

### 4.5 对立的转移

对立从一个位置转移到另一个位置。

**补偿性分化示例**：

当一个音丢失时，相邻的音可能发生分化以保持区别。

```
原始形式：*V₁CV₂
演变：V₁ 发生分化，补偿 C 或 V₂ 的变化
```

## 5. 音位的链移

### 5.1 定义

链移（Chain Shift）是一系列相互关联的音变，一个音位的变化引发另一个音位的变化，形成连锁反应。

| 特点 | 描述 |
|------|------|
| **系统性** | 多个音位按系统方式变化 |
| **连锁性** | 一个变化引发另一个变化 |
| **保持对立** | 变化后通常保持原有的对立关系 |

### 5.2 链移的类型

#### 推链（Push Chain）

一个音位进入另一个音位的空间，将其"推走"。

```
音位A → 音位B的位置 → 音位B被推向音位C的位置 → 音位C被推向...
```

**示例**：

```
      原始状态          推链后
高   i      u      i      u
      │             ↑
中   e      o      e      o
      ↑             │
低   a              a

/e/ 上移进入 /i/ 的空间，迫使 /i/ 发生变化
```

#### 拉链/拖链（Drag Chain / Pull Chain）

一个音位移走，留下空位，其他音位来填补。

```
音位A移走 → 留下空位 → 音位B移入填补 → 音位B的位置空出 → ...
```

**示例**：

```
      原始状态          拉链后
高   i      u      ?      u
      ↓             ↑
中   e      o      i      o
                    │
低   a              e

/i/ 下移，/e/ 上移填补空位，/a/ 上移填补 /e/ 的空位
```

### 5.3 经典链移案例

#### 英语元音大推移（Great Vowel Shift）

英语历史上最重要的链移之一：

| 中古英语 | 演变方向 | 现代英语 | 示例 |
|----------|----------|----------|------|
| /iː/ | → /aɪ/ | /aɪ/ | bite, time |
| /uː/ | → /aʊ/ | /aʊ/ | house, mouse |
| /eː/ | → /iː/ | /iː/ | beet, meet |
| /oː/ | → /uː/ | /uː/ | boot, root |
| /ɛː/ | → /eː/ → /iː/ | /iː/ | see, sea |
| /ɔː/ | → /oː/ → /oʊ/ | /oʊ/ | boat, goat |

**图示**：

```
          中古英语              现代英语

高闭     i:      u:         ai      au
          ↓       ↓
高开                             i:      u:
                              
中闭     e:      o:    →         e:      o:
          ↓       ↓
中开     ɛ:      ɔ:              (并入其他)

低       a:                    a:
```

#### 北日耳曼语元音链移

| 原始形式 | 演变 | 现代瑞典语 |
|----------|------|------------|
| *a | → /ɔ/ | ola |
| *ɔ | → /u/ | ful |
| *u | → /ʉ/ | hus |
| *ɔ̃ | → /ɑ/ | håla |

#### 格林定律（Grimm's Law）

印欧语到日耳曼语的辅音链移：

| 第一阶段 | 塞音 → 擦音 |
|----------|-------------|
| *p → f | *pater → father |
| *t → θ | *tres → three |
| *k → h/x | *kʷod → what |

| 第二阶段 | 浊塞音 → 清塞音 |
|----------|-----------------|
| *b → p | - |
| *d → t | *duwo → two |
| *g → k | *gene- → kin |

| 第三阶段 | 浊送气塞音 → 浊擦音 |
|----------|---------------------|
| *bʰ → b | *bʰrāter → brother |
| *dʰ → d | *dʰwer- → door |
| *gʰ → g | *gʰans → goose |

### 5.4 链移的原因

| 理论 | 描述 |
|------|------|
| **对称性原则** | 语言倾向于保持音系对称 |
| **避免合并** | 避免音位合并，保持区别功能 |
| **空间填充** | 填补音系空间中的空白 |
| **声学因素** | 声学空间的调整 |

### 5.5 链移的识别

| 识别标准 | 描述 |
|----------|------|
| **系统性** | 多个音位按规律变化 |
| **方向一致** | 变化方向一致（如都向上移） |
| **时空对应** | 在不同方言中呈现不同阶段 |
| **文献证据** | 文献记录显示演变阶段 |

## 6. 语言演变的其他现象

### 6.1 语音演变的特点

| 特点 | 描述 |
|------|------|
| **规律性** | 同一音位在相同条件下有相同变化 |
| **无意识** | 变化是逐渐的，使用者无意识 |
| **方向性** | 某些变化方向更常见 |
| **可预测性** | 基于已知的音变规律可预测变化 |

### 6.2 常见音变类型

| 类型 | 描述 | 示例 |
|------|------|------|
| **同化** | 变得与相邻音相似 | n → m / _b |
| **异化** | 变得与相邻音不同 | r...r → l...r |
| **弱化** | 发音减弱 | 元音央化、辅音浊化 |
| **强化** | 发音加强 | 塞音送气 |
| **删除** | 音被省略 | 词末辅音删除 |
| **插入** | 增加音 | 辅音丛间插入元音 |

### 6.3 类比演变

类比演变是形态变化而非语音变化，基于模式类推。

| 类型 | 描述 | 示例 |
|------|------|------|
| **类比扩展** | 将规则扩展到新词 | 英语：help-helped（原为holp） |
| **类比拉平** | 消除不规则形式 | climb 的过去式从 clomb 变为 climbed |
| **比例类比** | A:B = C:X | cow:cows = kine:→ cows（kine被淘汰） |

## 7. 语言的亲属关系判定

### 7.1 判定标准

| 证据类型 | 可靠性 | 说明 |
|----------|--------|------|
| **基本词汇同源** | 高 | 基本词汇较少借用 |
| **语音对应规律** | 高 | 系统性对应是亲属关系的强证据 |
| **形态对应** | 高 | 屈折词缀的同源是强证据 |
| **语法结构相似** | 中 | 可能是类型学相似或接触 |
| **文化词汇相似** | 低 | 容易借用 |

### 7.3 语言年代学

通过同源词保留率估算分化时间：

$$t = \frac{\ln(c)}{\ln(r)}$$

其中：
- t = 分化时间（千年）
- c = 同源词保留比例
- r = 保留率常数（约0.86）

**局限性**：
- 假设变化速率恒定（实际并不恒定）
- 不适用于超过6000年的分化
- 忽略借词和接触影响

## 8. 解题方法论

### 8.1 历史比较问题分析步骤

| 步骤 | 操作 |
|------|------|
| **Step 1** | 识别同源词（意义相近、形式相似） |
| **Step 2** | 建立语音对应表 |
| **Step 3** | 找出对应规律 |
| **Step 4** | 推断演变方向和规则 |
| **Step 5** | 重构原始形式 |
| **Step 6** | 验证重构的一致性 |

### 8.2 语音对应表构建

**示例**：比较三种语言的"三"

| 语言 | 形式 | 首辅音 | 元音 | 尾辅音 |
|------|------|--------|------|--------|
| 语言A | tris | t | i | s |
| 语言B | tis | t | i | s |
| 语言C | tis | t | i | s |

**对应规则**：
- 首辅音：t = t = t（无变化）
- 元音：i = i = i（无变化）
- 尾辅音：s = s = s（无变化）

### 8.3 判断演变方向的技巧

| 技巧 | 描述 |
|------|------|
| **分布分析** | 出现环境更广的形式可能是原始形式 |
| **自然性判断** | 更自然的演变方向更可能发生 |
| **多数原则** | 多数语言保留的形式可能是原始形式 |
| **类型学参照** | 参照已知的音变规律 |

### 8.4 常见音变方向

| 变化方向 | 自然程度 | 示例 |
|----------|----------|------|
| 塞音 → 擦音 | 高 | p → f |
| 清音 → 浊音（元音间） | 高 | t → d / V_V |
| 后元音 → 前元音（i影响） | 高 | u → y / _i |
| 词末辅音删除 | 高 | -t → ∅ / _# |
| 元音间辅音弱化 | 高 | t → d → ð → ∅ / V_V |

### 8.5 注意事项

1. **排除借词**：借词可能干扰同源词识别
2. **考虑条件**：音变可能在特定条件下发生
3. **检验一致性**：重构的规则应适用于所有相关词
4. **注意例外**：例外可能有特殊原因（类比、借用）
