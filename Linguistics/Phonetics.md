# Phonetics
## 1. Classification of sounds
元音与辅音
- Vowels: 从肺部上升的气流在声道中不会遇到任何障碍
- consonants: 在存在某种障碍或狭窄的情况下形成的

### 1.1.1 辅音发音部位 (Place of articulation)
| 部位分类             | 具体部位               | 描述        | 例子                           |
| ---------------- | ------------------ | --------- | ---------------------------- |
| **唇音（Labial）**   | 双唇（Bilabial）       | 使用双唇      | [b], [p], [m]                |
|                  | 唇齿（Labiodental）    | 下唇与上齿     | [f], [v]                     |
| **舌尖音（Coronal）** | 齿间（Interdental）    | 舌尖放在齿间或齿上 | [θ] (thin), [ð] (that)       |
|                  | 齿龈（Alveolar）       | 舌尖放在齿龈    | [t], [d], [s], [z], [n], [l] |
|                  | 后齿龈（Post-alveolar） | 舌尖在齿龈后方   | [ʃ] (shop), [ʒ], [tʃ], [dʒ]  |
| **舌背音（Dorsal）**  | 硬腭（Palatal）        | 舌背抵住硬腭    | [j], [c], [ɟ], [ɲ], [ʎ]      |
|                  | 软腭（Velar）          | 舌后部抵住软腭   | [k], [ɡ], [ŋ], [x]           |
| **喉音（Glottal）**  | 声门（Glottal）        | 气流通过声带    | [h], [ʔ]（喉塞音）                |

### 1.1.2 辅音发音方式 (Manner of articulation)
| 类型 | 描述 | 例子 |
|------|------|------|
| **塞音/爆破音（Plosive/Stop）** | 声道完全闭塞后突然释放，类似爆炸 | [p], [b], [t], [d], [k], [ɡ] |
| **擦音（Fricative）** | 声道部分收缩，气流产生湍流噪音 | [f], [v], [s], [z], [θ], [ð], [x], [ɣ] |
| **塞擦音（Affricate）** | 塞音与擦音的结合，闭塞后逐渐释放 | [t͡s], [d͡z], [t͡ʃ], [d͡ʒ] |
| **鼻音（Nasal）** | 口腔闭合，气流从鼻腔释放 | [n], [m], [ŋ], [ɳ] |
| **流音（Liquid）** | 总称，包括 [l] 和 [r] 类音 | [l], [r], [ɫ], [ʟ], [ʎ] |
| **滑音/半元音（Glide）** | 部分收缩，介于擦音和元音之间 | [w], [j] |

#### 重要术语
- **阻碍音（Obstruent）**：塞音、擦音、塞擦音的总称
- **响音（Sonorant）**：鼻音、流音、滑音的总称
- **咝音（Sibilant）**：齿龈和后齿龈的擦音和塞擦音

### 1.1.3 辅音浊化（Voicing）

| 类型 | 特点 | 例子 |
|------|------|------|
| **清音（Voiceless）** | 声带不振动 | [p], [s], [ts], [k] |
| **浊音（Voiced）** | 声带振动 | [b], [z], [dz], [ɡ] |

> **注意**：响音（鼻音、流音、滑音）几乎总是浊音。

### 1.1.4 辅音的其他特征

| 特征 | 描述 | 标记方式 |
|------|------|----------|
| **送气（Aspiration）** | 发音时伴随小股气流 | 上标 h：[pʰ] |
| **唇化（Labialisation）** | 发音时嘴唇圆起 | 上标 w：[ɡʷ] |
| **腭化（Palatalisation）** | 舌后部向上抬起靠近硬腭 | 上标 j：[tʲ] 或撇号 [t'] |
| **清化（Voicelessness for sonorants）** | 使响音变为清音 | 下标圆圈：[n̥] |

### 1.2.1 前后位置 (Backness)

**舌位前后**是元音分类的三个维度之一（另外两个是舌位高低和唇形圆展）。

| 类型 | 定义 | 发音时舌头位置 |
|------|------|----------------|
| **前元音（Front）** | 舌体前移，舌面前部隆起 | 舌头向前伸展，靠近前硬腭 |
| **后元音（Back）** | 舌体后缩，舌面后部隆起 | 舌头向后收缩，靠近软腭 |
| **央元音（Central）** | 舌体居中 | 舌头处于自然中立位置 |

#### 主要语言中的前/后元音

**汉语普通话**：

| 类型 | 元音 | 例字 |
|------|------|------|
| 前元音 | i, ü, e | 衣、迂、耶 |
| 央元音 | a, e（轻声） | 啊、的 |
| 后元音 | u, o, a | 乌、哦、大 |

> 注：汉语中的 a 实际上是央元音 [ä]，但在音系分析中通常归为后元音系列。

**英语**：

| 类型 | 元音 | 示例 |
|------|------|------|
| 前元音 | iː, ɪ, e, æ | see, sit, bed, cat |
| 央元音 | ə, ʌ, ɜː | about, cup, bird |
| 后元音 | uː, ʊ, ɔː, ɑː | food, put, thought, car |

**土耳其语**（IOLC常考，元音和谐系统）：

| 类型 | 不圆唇 | 圆唇 |
|------|--------|------|
| 前元音 | i, e | ü, ö |
| 后元音 | ı, a | u, o |

#### 如何判断前/后元音

**发音测试**：
```
发 [i] 时：
  - 舌头向前伸
  - 舌尖抵下齿
  - 舌面前部隆起
  - 感觉声音在口腔前部

发 [u] 时：
  - 舌头向后缩
  - 舌尖远离下齿
  - 舌面后部隆起
  - 感觉声音在口腔后部
```

**快速记忆技巧**：
| 技巧 | 前元音 | 后元音 |
|------|--------|--------|
| 感受 | 笑容状，嘴角向两侧拉开 | 圆唇状，嘴角向中间收缩 |
| 声音 | 明亮、尖锐 | 低沉、浑厚 |
| 例子 | "衣"、"一" | "乌"、"五" |

#### IOLC应用：元音和谐

在土耳其语、芬兰语等语言中，前/后元音的对立是**元音和谐**的基础：

```
词根元音         后缀元音选择
─────────────────────────────
前元音 (e,i,ö,ü) → 前元音后缀
后元音 (a,ı,o,u) → 后元音后缀
```

**土耳其语示例**：
| 词根 | 元音类型 | 复数后缀 |
|------|----------|----------|
| ev（房子） | e（前元音） | ev**ler** |
| kız（女孩） | ı（后元音） | kız**lar** |

**音系规则书写**：
```
前元音 → [+front]
后元音 → [-front]

后缀元音 → [α front] / [α front]_
```

#### 前/后元音对比总结

| 特征 | 前元音 | 后元音 |
|------|--------|--------|
| 舌位 | 舌体前移 | 舌体后缩 |
| 舌面隆起 | 舌面前部 | 舌面后部 |
| 口腔形状 | 较扁平 | 较圆拱 |
| 声学特征 | F2较高（频率） | F2较低 |
| 典型元音 | i, e, æ, y, ø | u, o, ɑ, ɯ, ɤ |

## 2. Syllable
### 2.1 Syllabic Structure
音节由三部分组成：

```
音节（σ）
├── 节首（Onset）───── 音节开头部分
└── 韵基（Rhyme）
    ├── 核（Nucleus）── 音节核心（必须成分）
    └── 节尾（Coda）─── 音节结尾部分
```

**示例**：`priest [pri:st]`
- 节首：`pr`
- 核心：`iː`
- 节尾：`st`

### 2.2 音节划分规则

| 模式 | 规则 | 例子 |
|------|------|------|
| VV | 两个相邻元音分属不同音节 | `V.V` |
| VCV | 单辅音归属于后一音节 | `V.CV` |
| VCCV | 两辅音分别作节尾和节首 | `VC.CV` |

> **基本原则**：辅音更倾向于作节首而非节尾。

## 3. 诗律学（Versification）
### 3.1 解题步骤
1. **音节划分**：对每行诗句进行音节划分
2. **确定音节类型**：区分轻音节与重音节
3. **确定格律**：找出支配诗句结构的规则

### 3.2 音节轻重分类
| 类型 | 基于元音长度 | 基于节尾 | 综合标准 |
|------|-------------|---------|---------|
| 轻音节 | (C)V(C) 短元音 | (C)V(V) 开音节 | (C)V |
| 重音节 | (C)VV(C) 长元音 | (C)V(V)C 闭音节 | (C)VV 或 (C)VC |
| 超重音节 | - | - | (C)VVC |

### 3.3 重要规律
- 一个重音节通常等价于两个轻音节
- 诗句音节数可变往往暗示存在这种等价关系

## 4. 重音（Stress）
### 4.1 重音类型
| 类型 | 特点 |
|------|------|
| **固定重音** | 重音位置固定（首、尾、倒数第二、第三音节等） |
| **自由重音** | 每个词有固定模式但无法预测 |
| **可预测重音** | 根据明确规则确定重音位置 |

### 4.2 重音窗口（Stress Window）
- 重音通常落在特定的音节范围内
- 最常见的窗口：**前三音节**或**后三音节**

### 4.3 重音标记
| 类型 | 符号 | 位置 |
|------|------|------|
| 主重音 | ˈ | 重读音节前 |
| 次重音 | ˌ | 重读音节前（下方） |

### 4.4 影响重音的因素
- 元音长度
- 音节是否有节尾（闭音节）
- 元音音质（前/后、开/闭等）
- 节首类型（清音/浊音/无节首）

**复杂示例——Pirahã语**：
音节层级：`CVV > GVV > VV > CV > GV > V`
（C = 清辅音，G = 浊辅音，V = 短元音，VV = 长元音）

## 5. 声调（Tone）
### 5.1 基本概念
- **音高（Pitch）**：声带振动的频率（基频）
- **语调（Intonation）**：用音高表达语气（陈述/疑问等）
- **声调（Tone）**：用音高区分词汇意义或语法意义

### 5.2 声调语言
在声调语言中，声调可区分词义。

### 5.3 超音段特征

声调和重音都属于**超音段（Suprasegmental）**现象：
- 超音段特征描述的是音段（辅音和元音）组合的特性
- 通常依附于音节
