## 5. 正确的语言规则

### 5.1 动词结构
Guarani 动词由以下**五个成分**按顺序构成：

| 位置 | 成分 | 说明 |
|------|------|------|
| 1 | 否定前缀 | `n(d)(a)-`，仅否定句存在 |
| 2 | 人称前缀 | 表示主语的人称和数 |
| 3 | 词根 | 动词的核心 |
| 4 | 否定后缀 | `-(r)i`，仅否定句存在 |
| 5 | 时态后缀 | `-ma`(过去) / `-ta`(将来) |

### 5.2 人称前缀

| 前缀 | 含义 |
|------|------|
| `a-` | 我 (第一人称单数) |
| `o-` | 他/她 (第三人称单数) |
| `ja-` | 我们 (第一人称复数) |
| `pe-` | 你们 (第二人称复数) |

**注意：人称用前缀标注，不是后缀，也不是中缀。**

### 5.3 否定前缀的变体

基本形式为 `nda-`，但有两条变化规则：

**规则 A：鼻音同化**
- 如果动词词根包含**任何鼻音**（ñ, ã, ẽ, ĩ, õ, ũ 等），否定前缀变为 `na-`（而非 `nda-`）
- 例：kororõ (哭泣) → napekororõi (你们不哭)，词根含鼻音 õ，故用 na-

**规则 B：元音省略**
- 如果人称前缀以**元音开头**，否定前缀中的元音 `a` 被省略
- `nda-` + 元音开头前缀 → `nd-`
- `na-` + 元音开头前缀 → `n-`

```
例：a- (我) → nd- + a- = nda- (我...不)
    o- (他) → nd- + o- = ndo- (他...不)
    ja- (我们) → n- + ja- = nja- (我们...不) [若词根含鼻音则用 na]
    pe- (你们) → nd- + pe- = ndape- (你们...不)
```

**关键点：否定前缀的不同形式（na/nda/no/ndo/nja/ndape等）是由语音规则决定的，而非由人称决定。**

### 5.4 否定后缀 `-ri` vs `-i`

- 词根以 **i 结尾** → 否定后缀用 **`-ri`**
- 词根以 **其他元音结尾** → 否定后缀用 **`-i`**

```
例：purahei (唱歌) → ndopuraheiri (他不唱歌) [词根以 i 结尾，用 -ri]
    karu (吃) → ndokarui (他不吃) [词根不以 i 结尾，用 -i]
```

### 5.5 否定+将来时：特殊后缀 `-mo'ãi`

否定将来时不使用 `*-rita`，而是使用**特殊后缀 `-mo'ãi`**

```
例：karu (吃) + 将来 + 否定 → ndokarumo'ãi (他将不吃)
    而非 *ndokarurita
```

### 5.6 时态后缀（肯定句）

| 后缀 | 时态 |
|------|------|
| `-ta` | 将来时 |
| `-ma` | 过去时 |
| 无后缀 | 现在时 |

```
例：akaruta (我将吃)    akaruma (我吃了)    akaru (我吃/正在吃)
```

### 5.7 完整示例解析

| Guarani | 分析 | 翻译 |
|---------|------|------|
| `akaruma` | a- (我) + karu (吃) + -ma (过去) | 我吃了 |
| `ojupita` | o- (他) + jupi (醒来) + -ta (将来) | 他将醒来 |
| `ndavo'omo'ãi` | nd- (否定) + a- (我) + vo'o (拿) + -mo'ãi (否定将来) | 我将不拿 |
| `napekororõi` | na- (否定+鼻音) + pe- (你们) + kororõ (哭) + -i (否定) | 你们不哭 |
| `ndapyhyima` | nd- (否定) + a- (我) + pyhy (抓) + -i (否定) + -ma (过去) | 我没抓 |

### 5.8 否定句构造流程

```
┌─────────────────────────────────────────────────────┐
│ 1. 词根含鼻音？                                      │
│    是 → 否定前缀 = na-                              │
│    否 → 否定前缀 = nda-                             │
├─────────────────────────────────────────────────────┤
│ 2. 人称前缀以元音开头？                              │
│    是 → 省去否定前缀的 a                            │
│         (nda- → nd-, na- → n-)                      │
├─────────────────────────────────────────────────────┤
│ 3. 将来时？                                         │
│    是 → 否定后缀 = -mo'ãi                           │
│    否 → 词根以i结尾？                               │
│         是 → 否定后缀 = -ri                         │
│         否 → 否定后缀 = -i                          │
└─────────────────────────────────────────────────────┘
```

## 6. Mistake
我的核心错误在于：
1. **误认为人称用后缀标注** → 实际上人称用**前缀**标注
2. **误认为否定前缀的不同形式标记不同人称** → 实际上否定前缀的形式由**语音规则**（鼻音同化、元音省略）决定
3. **误将人称标记称为"中缀"** → 实际上人称标记是**前缀**，直接位于词根之前
