# Vowel Harmony
[[Phonology]]
[[Minority Languages of China]]

## 1. 引言

元音和谐（Vowel Harmony）是一种特殊的**远距离同化**现象，指同一单词内的元音必须共享某些特征，使元音在听觉上保持协调一致。

| 性质 | 描述 |
|------|------|
| **类型** | 远距离同化（非相邻声音的同化） |
| **作用范围** | 通常在词根与词缀之间 |
| **核心机制** | 词根元音"决定"词缀元音的形式 |

**基本公式**：
$$\text{词根}_{[\alpha F]} + \text{词缀} \rightarrow \text{词缀}_{[\alpha F]}$$

其中 $[\alpha F]$ 表示某特征（如[±前]、[±圆唇]）具有相同值。

## 2. 元音和谐的类型

### 2.1 基于舌位前后（Palatal Harmony）

| 类型 | 规则 | 代表语言 |
|------|------|----------|
| **前-后和谐** | 词根与词缀元音必须同为前元音或同为后元音 | 土耳其语、蒙古语、芬兰语 |

**土耳其语元音系统**：

| | 前元音 | | 后元音 | |
|--|--------|--------|--------|--------|
| | 不圆唇 | 圆唇 | 不圆唇 | 圆唇 |
| **闭** | i | ü | ı | u |
| **开** | e | ö | a | o |

**示例（土耳其语复数后缀）**：

| 词根 | 元音类型 | 复数后缀 | 结果 |
|------|----------|----------|------|
| ev | 前元音 (e) | -ler | evler（房子们）|
| kız | 后元音 (ı) | -lar | kızlar（女孩们）|
| göz | 前圆唇 (ö) | -ler | gözler（眼睛们）|
| yol | 后元音 (o) | -lar | yollar（路们）|

### 2.2 基于唇形圆展（Labial Harmony）

| 类型 | 规则 | 代表语言 |
|------|------|----------|
| **圆唇和谐** | 词根与词缀元音必须同为圆唇或同为不圆唇 | 土耳其语（部分后缀）、蒙古语 |

**土耳其语圆唇和谐示例（1sg所有格后缀）**：

| 词根最后元音 | 元音特征 | 所有格后缀 |
|--------------|----------|------------|
| e, i | 前不圆唇 | -im |
| a, ı | 后不圆唇 | -ım |
| ö, ü | 前圆唇 | -üm |
| o, u | 后圆唇 | -um |

**完整示例**：
```
ev-im   我的房子     göz-üm   我的眼睛
kız-ım  我的女孩     kol-um   我的胳膊
```

### 2.3 基于舌位高低（高度和谐）

| 类型 | 规则 | 代表语言 |
|------|------|----------|
| **高-低和谐** | 元音必须同为高元音或同为低元音 | 部分尼罗-撒哈拉语言 |

### 2.4 基于舌根位置（ATR和谐）

| 类型 | 规则 | 代表语言 |
|------|------|----------|
| **ATR和谐** | 元音必须同为[+ATR]或[-ATR] | 尼日尔-刚果语系语言 |

**ATR（Advanced Tongue Root）**：

| 特征 | 描述 | 音质特点 |
|------|------|----------|
| [+ATR] | 舌根前伸 | 元音更"紧"、更清晰 |
| [-ATR] | 舌根后缩 | 元音更"松"、略沉闷 |

**约鲁巴语ATR和谐示例**：
```
[+ATR] 元音：i, e, u, o
[-ATR] 元音：ɪ, ɛ, ʊ, ɔ, a
```

## 3. 中性元音（Neutral Vowels）

某些元音**既不触发也不参与和谐**，可在任何和谐环境中出现。

| 语言 | 中性元音 | 行为特点 |
|------|----------|----------|
| **土耳其语** | i | 不影响后缀元音选择 |
| **芬兰语** | i, e | 不触发后元音后缀，但自身可出现在前/后元音环境中 |
| **匈牙利语** | i, é | 可出现在任何环境中，后缀根据前面非中性元音选择 |

**中性元音的影响**：

```
情况1：词根只有中性元音
  → 后缀通常采用默认形式（常为前元音形式）

情况2：词根有中性元音 + 和谐元音
  → 后缀根据非中性元音选择
```

## 5. 蒙古语元音和谐详解

蒙古语是世界上元音和谐系统最完整的语言之一。

### 5.1 蒙古语元音系统

| 性别 | 阳性（后元音） | 阴性（前元音） |
|------|----------------|----------------|
| **阴性** | a, o, u | e, ö, ü |
| **中性** | - | i |

### 5.2 和谐规则

**基本规则**：阳性词根 + 阳性后缀；阴性词根 + 阴性后缀

| 后缀类型 | 阳性形式 | 阴性形式 |
|----------|----------|----------|
| 复数 | -nar | -ner |
| 与格 | -a/-e | -e |
| 工具格 | -aar/-eer | -eer |

**示例**：
```
 bagš（老师）+ nar → bagšnar（老师们）  阳性
 ger（房子）+ ner → gerner（房子们）    阴性
```

### 5.3 圆唇和谐（蒙古语特有）

当词根含圆唇元音时，后缀元音也倾向圆唇化：
```
nöxör（朋友）+ ör → nöxörör（和朋友）
```

## 6. IOLC解题方法论

### 6.1 识别线索

| 线索 | 说明 |
|------|------|
| **同一后缀多种形式** | 如 -ler/-lar, -im/-ım/-üm/-um |
| **词根元音与后缀存在规律对应** | 可列表观察 |
| **语言属于已知有和谐的语系** | 阿尔泰、乌拉尔、尼日尔-刚果 |

### 6.2 常见题型分类

#### 6.2.1 基础后缀变体题

**特征**：单一后缀有2种形式（前后和谐）

**出题模式**：
```
给定：8-12组 词根 + 词缀配对
问题：预测新词的词缀形式
```

**解题要点**：
- 仅涉及前后二元对立
- 后缀变体数量 = 2

#### 6.2.2 复合规则题

**特征**：同一后缀有4种形式（前后 + 圆唇和谐叠加）

**土耳其语所有格后缀示例**：

| 词根元音 | 前后特征 | 圆唇特征 | 后缀形式 |
|----------|----------|----------|----------|
| e, i | 前 | 不圆唇 | -im |
| a, ı | 后 | 不圆唇 | -ım |
| ö, ü | 前 | 圆唇 | -üm |
| o, u | 后 | 圆唇 | -um |

**解题要点**：
- 需同时判断两个特征
- 建立2×2特征矩阵

#### 6.2.3 中性元音处理题

**特征**：词根含中性元音时的特殊情况

**两种情况**：

| 情况 | 描述 | 处理方式 |
|------|------|----------|
| 中性元音 + 和谐元音 | 词根同时含两种元音 | 根据**非中性元音**选择 |
| 仅含中性元音 | 词根只有中性元音 | 采用**默认形式**（通常为前元音形式） |

**芬兰语示例**：
```
väki（人民）→ väet（复数）  词根含非中性元音ä，根据ä选择
tiivi（密封）→ tiiveet（复数）  词根仅含中性元音i、e，采用默认前元音形式
```

#### 6.2.4 多级派生题

**特征**：词根 → 一级派生 → 二级派生

**结构**：
```
词根 → [派生词缀] → [屈折词缀]
       ↑              ↑
    遵循和谐       继续遵循和谐（基于派生后的元音）
```

**示例**：
```
土耳其语：
ev（房子）→ ev-ci（建房者）→ ev-ci-ler（建房者们）
           ↑               ↑
        前元音 → -ci     前元音 → -ler

kol（胳膊）→ kol-cu（手臂师）→ kol-cu-lar（手臂师们）
            ↑                 ↑
         后元音 → -cu       后元音 → -lar
```

**解题要点**：
- 每一级派生都独立触发和谐
- 后级派生基于**前一级派生结果**的元音特征

#### 6.2.5 词内和谐题

**特征**：词根本身需遵循和谐规则

**蒙古语示例**：
```
正确：nöxör（朋友）、ger（房子）
错误：*nöxar、*gar
      ↑       ↑
   前元音   后元音
   后元音   前元音（冲突！）
```

**解题要点**：
- 检查词根内部元音是否同组
- 判断词根的"性别"（阳性/阴性）

### 6.3 分析步骤

```
步骤1：收集词根 + 后缀配对数据
步骤2：提取词根最后元音和后缀元音
步骤3：建立对应关系表
步骤4：分析元音特征（前后/圆唇/高低）
步骤5：找出规律，写出规则
步骤6：验证规则（检查是否有反例）
```

### 6.4 形式化表达（计算语言学视角）

#### 6.4.1 有限状态机建模

元音和谐可用 **FSM** 形式化建模：

**前后和谐 FSM**：

$$M = (Q, \Sigma, \delta, q_0, F)$$

| 元素 | 取值                   |
| -------- | -------------------- |
| $Q$ | {q前, q后, q默认}        |
| $\Sigma$ | {前元音, 后元音, 中性元音, 边界} |
| $q_0$ | q默认                  |
| $F$ | {q前, q后}             |

**状态转移表**：

| 当前状态 | 输入   | 下一状态    |
| ---- | ---- | ------- |
| q默认  | 前元音  | q前      |
| q默认  | 后元音  | q后      |
| q前   | 前元音  | q前      |
| q前   | 后元音  | q后      |
| q前   | 中性元音 | q前      |
| q后   | 前元音  | q前      |
| q后   | 后元音  | q后      |
| q后   | 中性元音 | q后      |
| q默认  | 边界   | 输出默认后缀  |
| q前   | 边界   | 输出前元音后缀 |
| q后   | 边界   | 输出后元音后缀 |

**状态转移图**：

```
                    前元音
              ┌──────────────┐
              │              │
              ▼              │
           (q前)             │
          ↗     ↖            │
   前元音/         \后元音    │
        /           \        │
   (q默认)           \       │
        \           /        │
   后元音\         /前元音    │
          ↘     ↙            │
           (q后)             │
              │              │
              │后元音        │中性元音
              └──────────────┘
```

#### 6.4.2 规则形式化

**使用特征符号**：

$$\text{后缀} \to \begin{cases}
\text{Suffix}_1 / \text{词根}_{[\alpha F]} \_ \\
\text{Suffix}_2 / \text{词根}_{[-\alpha F]} \_
\end{cases}$$

其中 $[\alpha F]$ 表示某特征（如 [±前]、[±圆唇]）。

**示例（前后和谐）**：
$$\text{Plural} \to \begin{cases}
\text{-ler} / \text{词根}_{[+前]} \_ \\
\text{-lar} / \text{词根}_{[-前]} \_
\end{cases}$$

### 6.5 规则书写格式

**标准格式**：
```
后缀选择规则：
条件1 → 后缀形式1
条件2 → 后缀形式2
...

中性元音：...
默认值：...（词根仅含中性元音时）
```

**完整示例**：
```
土耳其语所有格后缀规则：
词根最后元音为前不圆唇 (e, i) → -im
词根最后元音为后不圆唇 (a, ı) → -ım
词根最后元音为前圆唇 (ö, ü) → -üm
词根最后元音为后圆唇 (o, u) → -um
```

### 6.6 IOLC真题分析

#### 6.6.1 IOL 2007 土耳其语题（经典）

**题目类型**：基础后缀变体题 + 复合规则

**给定数据片段**：

| 词根 | 含义 | 复数 | 所有格 |
|------|------|------|--------|
| el | 手 | eller | elim |
| kız | 女孩 | kızlar | kızım |
| göz | 眼睛 | gözler | gözüm |
| yol | 路 | yollar | yolun |
| ev | 房子 | evler | evim |

**分析过程**：

**复数后缀分析**：

| 词根元音 | 元音特征 | 复数后缀 |
|----------|----------|----------|
| e | 前 | -ler |
| ı | 后 | -lar |
| ö | 前 | -ler |
| o | 后 | -lar |

**规则**：
```
复数后缀：
前元音 (e, i, ö, ü) → -ler
后元音 (a, ı, o, u) → -lar
```

**所有格后缀分析**：

| 词根元音 | 前后 | 圆唇 | 所有格后缀 |
|----------|------|------|------------|
| e | 前 | 不圆唇 | -im |
| ı | 后 | 不圆唇 | -ım |
| ö | 前 | 圆唇 | -üm |
| o | 后 | 圆唇 | -un |

**规则**：
```
所有格后缀：
前不圆唇 (e, i) → -im
后不圆唇 (a, ı) → -ım
前圆唇 (ö, ü) → -üm
后圆唇 (o, u) → -un
```

**注意**：题目中 `yol` 的所有格是 `yolun` 而非 `*yolun`，体现了圆唇和谐的特殊性。

#### 6.6.2 IOL 2011 芬兰语题

**题目类型**：中性元音处理题

**关键数据**：

| 词根 | 复数 |
|------|------|
| väki | väet |
| lampi | lammet |
| kieli | kielet |
| nainen | naiset |

**芬兰语元音系统**：

| 组别 | 元音 |
|------|------|
| 前元音 | ä, ö, y |
| 后元音 | a, o, u |
| 中性元音 | i, e |

**分析要点**：
- 中性元音 i, e 不参与和谐
- 词根含前元音时，后缀用前元音形式 (-et)
- 词根仅含中性元音时，默认用前元音形式

**规则**：
```
芬兰语复数后缀：
词根含后元音 (a, o, u) → -at/-ot/-ut
词根含前元音 (ä, ö, y) 或仅含中性元音 (i, e) → -ät/-öt/-yt
```

#### 6.6.3 IOL 2013 匈牙利语题

**题目类型**：多级派生题

**数据示例**：

| 词根 | 含义 | 复数 | 所有格复数 |
|------|------|------|------------|
| ház | 房子 | házak | házaim |
| könyv | 书 | könyvek | könyveim |
| gyümölcs | 水果 | gyümölcsök | gyümölcseim |

**分析要点**：
- 匈牙利语元音和谐较为复杂
- 存在"隐性"和谐现象
- 多级派生时每级独立遵循和谐

### 6.7 常见陷阱

| 陷阱 | 描述 | 应对策略 |
|------|------|----------|
| **中性元音干扰** | 中性元音不触发和谐 | 先识别中性元音，再找非中性元音 |
| **借词例外** | 外来词可能不遵循和谐 | 注意题目是否给出例外说明 |
| **词根元音变化** | 派生时词根元音可能变化 | 区分词根变化和词缀和谐 |
| **同形异源** | 同形式后缀可能有不同来源 | 检查是否有多个后缀变体 |
| **隐性和谐** | 某些和谐仅在一部分词中体现 | 扩大数据样本，找规律 |

### 6.8 高频语言与出题趋势

| 语言 | 语系 | 和谐类型 | IOLC频率 | 难度 |
|------|------|----------|----------|------|
| **土耳其语** | 突厥 | 前后+圆唇 | ⭐⭐⭐⭐⭐ | 中等 |
| **芬兰语** | 乌拉尔 | 前后+中性元音 | ⭐⭐⭐⭐ | 中等 |
| **匈牙利语** | 乌拉尔 | 前后（复杂） | ⭐⭐⭐ | 较难 |
| **蒙古语** | 蒙古 | 前后+圆唇 | ⭐⭐⭐ | 中等 |
| **维吾尔语** | 突厥 | 前后 | ⭐⭐ | 简单 |
| **哈萨克语** | 突厥 | 前后 | ⭐⭐ | 简单 |
| **满语** | 满-通古斯 | 前后（弱化） | ⭐ | 较难 |

## 7. 特殊现象

### 7.1 不透明元音（Opaque Vowels）

某些元音会"阻断"和谐的传播。

| 现象 | 描述 | 示例 |
|------|------|------|
| 阻断 | 特定元音使后缀选择基于它而非更早的元音 | 某些土耳其语借词 |

### 7.2 透明元音（Transparent Vowels）

某些元音"跳过"，不影响和谐传递。

```
中性元音 i 在土耳其语中：
词根元音 → i（跳过）→ 后缀元音根据词根选择
```

### 7.3 借词例外

外来借词可能不遵循和谐规则：

| 语言 | 现象 |
|------|------|
| 土耳其语 | 阿拉伯语借词有时保留原元音 |
| 蒙古语 | 汉语借词可能打破和谐 |

### 7.4 和谐弱化

现代语言中和谐规则可能逐渐弱化：

| 语言 | 弱化表现 |
|------|----------|
| 满语 | 年轻使用者元音和谐意识减弱 |
| 部分土耳其语方言 | 圆唇和谐在某些词中失效 |

## 8. 与其他音系现象的对比

| 现象 | 作用范围 | 性质 |
|------|----------|------|
| **元音和谐** | 词内，词根→词缀 | 远距离同化 |
| **辅音和谐** | 词内，辅音特征传播 | 远距离同化 |
| **同化** | 相邻声音 | 接触同化 |
| **元音变化** | 历时演变 | 音变规律 |

---

## 9. 总结

| 要点 | 内容 |
|------|------|
| **本质** | 远距离同化，使词内元音特征一致 |
| **主要类型** | 前后和谐、圆唇和谐、ATR和谐 |
| **分布** | 阿尔泰语系、乌拉尔语系、尼日尔-刚果语系 |
| **中国境内** | 蒙古语、维吾尔语、哈萨克语、满语、赫哲语等 |
| **IOLC考点** | 后缀变体分析、规则推导 |
