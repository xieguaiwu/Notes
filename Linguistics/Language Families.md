## 一、汉藏语系

### 1.1 概述
- **地理分布**：东亚、东南亚、喜马拉雅地区
- **主要分支**：汉语族、藏缅语族
- **使用人口**：约14亿（世界第二大）

### 1.2 语音/音系特点

| 特征       | 描述                              |
| -------- | ------------------------------- |
| **声调系统** | 几乎所有语言都有声调，汉语4-9个调，苗瑶语可达10+个    |
| **音节结构** | 简单，多为 CV 或 CVC，极少复辅音（藏语除外）      |
| **声母系统** | 送气/不送气对立重要（汉语：p/pʰ, t/tʰ, k/kʰ） |
| **韵尾简化** | 汉语方言韵尾减少趋势（粤语 -ptk > 官话 -∅）     |

**藏缅语支特殊现象**：
- 藏语有复辅音
- 缅甸语有声调系统（但古藏语无声调）
- 彝语有松紧元音对立

### 1.3 句法特点

| 特征         | 描述                   |
| ---------- | -------------------- |
| **词序**     | SVO（汉语）或 SOV（藏缅语）    |
| **修饰语位置**  | 修饰语在中心词之前（定语前置）      |
| **话题优先**   | 汉语是典型的话题优先语言         |
| **量词系统**   | 发达的量词系统（汉语、藏语、缅甸语都有） |
| **缺乏形态变化** | 分析语特征明显，缺少屈折变化       |

**汉语句法特点**：
```
话题-评论结构：这本书，我看过了
把字句：我把书看了
被字句：书被我看了
```

### 1.4 形态学特点

- **孤立语**：几乎无屈折变化
- **缺乏时态标记**：通过虚词表达
- **缺乏格标记**：靠词序和介词表达语法关系
- **丰富的复合词**：通过合成造词

## 二、印欧语系

### 2.1 概述
- **地理分布**：欧洲、南亚、美洲、大洋洲
- **主要分支**：
  - 日耳曼语族（英语、德语、荷兰语、北欧语言）
  - 罗曼语族（拉丁语、法语、西班牙语、意大利语、葡萄牙语、罗马尼亚语）
  - 斯拉夫语族（俄语、波兰语、捷克语、塞尔维亚语）
  - 凯尔特语族（爱尔兰语、威尔士语、布列塔尼语）
  - 印度-伊朗语族（印地语、乌尔都语、波斯语、孟加拉语）
  - 希腊语族、阿尔巴尼亚语族、亚美尼亚语族

### 2.2 语音/音系特点

| 分支          | 特点                                                        |
| ----------- | --------------------------------------------------------- |
| **日耳曼语族**   | 格林定律（Grimm's Law）：p→f, t→θ, k→h；元音变化丰富（英语 sing-sang-sung） |
| **罗曼语族**    | 拉丁语有长短元音对立；现代语言元音较简单；西班牙语有颤音r                             |
| **斯拉夫语族**   | 丰富的辅音丛；软硬辅音对立（腭化）；重音可移动                                   |
| **凯尔特语族**   | 词首辅音变化；词序VSO                                              |
| **印度-伊朗语族** | 保留较多古印欧特征；印地语有送气塞音系列；波斯语有咽音                               |

**重要音变规律**：

**格林定律（日耳曼语）**：
```
*p → f (pater → father)
*t → θ (tres → three)
*k → h (canis → hound)
*b → p, *d → t, *g → k（浊音清化）
```

### 2.3 句法特点

| 分支          | 词序  | 特殊特点                    |
| ----------- | --- | ----------------------- |
| **日耳曼语族**   | SVO | 德语V2规则（动词第二位）；英语固定词序    |
| **罗曼语族**    | SVO | 法语有二十进制残留；西班牙语/意大利语屈折丰富 |
| **斯拉夫语族**   | SVO | 自由词序（丰富的格系统支持）；否定对格     |
| **凯尔特语族**   | VSO | 词首辅音变化；爱尔兰语、威尔士语典型      |
| **印度-伊朗语族** | SOV | 印地语SOV；波斯语SVO           |


**德语V2规则**：
```
Ich sehe den Mann.（我看见那个男人）
Den Mann sehe ich.（那个男人我看见）
Sehe ich den Mann?（我看见那个男人吗？）
```

### 2.4 形态学特点

| 分支 | 特点 |
|------|------|
| **英语** | 屈折系统大幅简化；分析语倾向强 |
| **德语** | 4格系统（主格、宾格、与格、属格）；3性；动词变位 |
| **俄语** | 6格系统；3性；丰富的动词体对立（完成体/未完成体） |
| **拉丁语** | 6格；5种变格法；动词时态语态丰富 |
| **印地语** | 有后置词系统；分词结构复杂；SOV词序 |

## 三、闪含语系

### 3.1 概述
- **地理分布**：中东、北非
- **主要分支**：
  - 闪语族（阿拉伯语、希伯来语、阿拉米语、阿姆哈拉语）
  - 柏柏尔语族
  - 库什语族（索马里语）
  - 乍得语族（豪萨语）
  - 埃及语族（科普特语，已消亡）

### 3.2 语音/音系特点

| 特征 | 描述 |
|------|------|
| **辅音根系统** | 最核心特征：由3个辅音构成词根，插入不同元音派生词汇 |
| **喉音/咽音** | 阿拉伯语有 /ħ, ʕ, x, ɣ/ 等喉音 |
| **强调辅音** | 咽化辅音（阿拉伯语：ṭ, ṣ, ḍ, ẓ） |
| **元音系统简单** | 通常只有 a, i, u 三个短元音和对应长元音 |
| **声门塞音** | 阿拉伯语有 /ʔ/（hamza） |

**辅音根示例（阿拉伯语 k-t-b "写"）**：
```
kataba   他写了
kitāb    书
kātib    作家
maktaba  图书馆
maktūb   被写的/命运
```

### 3.3 句法特点

| 特征 | 描述 |
|------|------|
| **VSO词序** | 古典阿拉伯语、希伯来语典型 |
| **名词短语** | 名词-形容词顺序（形容词后置） |
| **定冠词** | 阿拉伯语 al-；希伯来语 ha- |
| **状态构造** | 名词的"状态"（定指/非定指）通过后缀表达 |
| **性数一致** | 动词与主语在性、数上一致 |

**阿拉伯语名词状态**：
```
kitābun    一本书（非定指）
al-kitābu  那本书（定指）
kitābu     ...的书（构造状态）
```

### 3.4 形态学特点

| 特征       | 描述                       |
| -------- | ------------------------ |
| **非连接语** | 词内变化通过内部元音变化实现，而非词缀      |
| **派生模式** | 固定的音节模板（CVCCVC, CVCVVC等） |
| **性对立**  | 阳性/阴性，阴性常以 -a(t) 结尾      |
| **数系统**  | 单数、双数、复数（完整/破碎复数）        |
| **动词系统** | 完成体/未完成体；词根+模板+词缀        |

**阿拉伯语破碎复数**：
```
kitāb（书）→ kutub（书籍）
rajul（男人）→ rijāl（男人们）
```

## 四、南岛语系

### 4.1 概述
- **地理分布**：东南亚岛屿、太平洋岛屿、马达加斯加
- **主要分支**：
  - 台湾南岛语言（泰雅语、阿美语、排湾语等）
  - 马来-波利尼西亚语族（马来语、印尼语、他加禄语、马来亚拉姆语、斐济语、毛利语、夏威夷语等）
- **特点**：世界上分布最广的语系（从马达加斯加到复活节岛）

### 4.2 语音/音系特点

| 特征 | 描述 |
|------|------|
| **音节简单** | 多为 CV 或 CVC 结构 |
| **元音系统** | 通常5元音系统 |
| **辅音系统** | 相对简单，少见复杂辅音丛 |
| **重音** | 多为固定重音（倒数第一或第二音节） |
| **音位对立少** | 波利尼西亚语言音位很少（夏威夷语只有13个音位） |

**波利尼西亚语言音位（夏威夷语）**：
```
辅音：p, k, ʔ, h, m, n, l, w（8个）
元音：a, e, i, o, u（5个）
```

**菲律宾语言特点**：
- 他加禄语有喉塞音
- 有元音长度对立

### 4.3 句法特点

| 特征 | 描述 |
|------|------|
| **焦点系统** | 最核心特征：动词标记句子焦点（施事、受事、方位等） |
| **VSO/VOS词序** | 动词常居首 |
| **代词系统** | 复杂的代词系统（包含、排除式） |
| **缺乏格标记** | 通过焦点系统而非格标记表达语法关系 |

**菲律宾语焦点系统（他加禄语）**：
```
焦点类型    标记    示例
施事焦点    -um-    bumibili（买...的人是...）
受事焦点    -in-    binibili（被买的是...）
方位焦点    -an     bilhian（买的地方是...）
工具焦点    i-      ipambili（用来买的是...）
```

**印尼/马来语**：
- 较简化的焦点系统
- SVO词序
- 大量词缀系统

### 4.4 形态学特点

| 特征 | 描述 |
|------|------|
| **黏着语** | 通过词缀派生和屈折 |
| **前缀丰富** | 主动前缀、被动前缀、使动前缀等 |
| **中缀** | 部分语言有中缀（他加禄语 -um-） |
| **重复** | 常见词根重复表示复数、强调等 |
| **缺乏屈折变化** | 无格、性、数变化（通过词序和词缀表达） |

**印尼语词缀示例**：
```
meN-    主动前缀      membaca（读）
di-     被动前缀      dibaca（被读）
ber-    状态前缀      berdiri（站立）
pe-     施事名词      pembaca（读者）
-an     受事名词      bacaan（读物）
```

## 五、尼日尔-刚果语系

### 5.1 概述
- **地理分布**：撒哈拉以南非洲
- **主要分支**：
  - 大西洋-刚果语族
  - **班图语支**（最重要：斯瓦希里语、祖鲁语、科萨语、卢干达语、约鲁巴语等）
  - 曼德语族
  - 沃尔特-尼日尔语族
- **特点**：世界上语言数量最多的语系（1500+种语言）

### 5.2 语音/音系特点

| 特征        | 描述                   |
| --------- | -------------------- |
| **声调语言**  | 大多数语言有声调（高/低，或高/低/降） |
| **鼻辅音丰富** | 前置鼻化塞音常见             |
| **声母交替**  | 班图语有系统的声母变化          |
| **吸气音**   | 南非语言（祖鲁语、科萨语）有吸气音    |
| **元音和谐**  | 部分语言有ATR元音和谐         |

**科萨语吸气音**：
```
|    齿吸气音
ǁ    边吸气音
ǃ    齿龈吸气音
```

**声调示例（约鲁巴语）**：
```
ó kọ́    他建造了（高-高）
ó kọ    他学习（高-中）
ó kọ̀    他拒绝（高-低）
```

### 5.3 句法特点

| 特征 | 描述 |
|------|------|
| **SVO词序** | 班图语典型 |
| **名词类别系统** | 最核心特征：名词分为多个类别（"性"系统） |
| **一致关系** | 动词、形容词、代词与名词类别一致 |
| **代词系统** | 丰富的代词系统（主语/宾语标记在动词上） |

**班图语名词类别系统**：
```
类别    前缀（斯瓦希里语）    意义范围
1/2     m-/wa-               人物
3/4     m-/mi-               植物、自然物
5/6     ji-/ma-              水果、扩展物
7/8     ki-/vi-              物体
9/10    N-/N-                动物
11      u-                   抽象、长形物
15      ku-                  动名词
```

**一致关系示例（斯瓦希里语）**：
```
mtu mkubwa amekuja
人（类别1） 大（类别1一致） 已经来了（类别1主语标记）
"一个大人来了"

watu wakubwa wamekuja
人们（类别2） 大（类别2一致） 已经来了（类别2主语标记）
"大人们来了"
```

### 5.4 形态学特点

| 特征 | 描述 |
|------|------|
| **黏着语** | 通过前缀/后缀表达语法关系 |
| **多式综合倾向** | 单词可包含多个语素 |
| **动词复杂** | 时态、体、语气、主语/宾语标记都在动词上 |
| **缺乏格标记** | 通过词序和名词类别表达 |
| **派生丰富** | 使动、被动、应用动词等 |

**斯瓦希里语动词结构**：
```
a-li-m-saidia
他-过去时-他(宾语)-帮助
"他帮助了他"

ni-ta-ku-ona
我-将来时-你(宾语)-看见
"我将看见你"
```

## 六、突厥语系

### 6.1 概述
- **语系归属**：阿尔泰语系（与蒙古语族、满-通古斯语族并列）
- **地理分布**：土耳其、中亚、高加索、西伯利亚、中国新疆
- **语言数量**：约40种语言
- **使用人口**：约2亿人
- **特点**：语族内各语言高度相似，可一定程度互通

**主要分支**：

| 语支 | 代表语言 |
|------|----------|
| **乌古斯语支（西南）** | 土耳其语、阿塞拜疆语、土库曼语、撒拉语 |
| **克普恰克语支（西北）** | 哈萨克语、吉尔吉斯语、鞑靼语、巴什基尔语 |
| **葛逻禄语支（东南）** | 维吾尔语、乌兹别克语 |
| **西伯利亚语支（东北）** | 雅库特语、图瓦语、哈卡斯语 |
| **布尔加尔语支** | 楚瓦什语（与其他突厥语言差异最大） |

### 6.2 语音/音系特点

#### 6.2.1 元音系统

**8元音系统**（以土耳其语为例）：

| | 前元音 | | 后元音 | |
|--|--------|--------|--------|--------|
| | 不圆唇 | 圆唇 | 不圆唇 | 圆唇 |
| **高** | i | ü (y) | ı (ɯ) | u |
| **低** | e | ö (ø) | a | o |

**元音和谐（核心特征）**：
- **前后和谐**：后缀元音必须与词根元音在前后性上一致
- **圆唇和谐**：部分后缀还受圆唇性影响

```
词根元音类型 → 后缀选择
──────────────────────────
前元音 (e, i, ö, ü) → 前元音后缀
后元音 (a, ı, o, u) → 后元音后缀
```

**土耳其语复数后缀示例**：

| 词根 | 元音 | 复数形式 |
|------|------|----------|
| ev（房子） | e（前） | ev**ler** |
| kız（女孩） | ı（后） | kız**lar** |
| göz（眼） | ö（前圆） | göz**ler** |
| kol（臂） | o（后圆） | kol**lar** |

#### 6.2.2 辅音系统

| 特征 | 描述 |
|------|------|
| **词首限制** | 词首少出现浊流辅音 r, l, n, m |
| **辅音丛限制** | 词首、词尾不出现相邻辅音（外来词除外） |
| **清浊对立** | p/b, t/d, k/g, ç/c |
| **送气** | 塞音通常不送气 |

**维吾尔语特殊音变**：
- 元音弱化：词中元音弱化为更闭、更央的元音
- 辅音弱化：送气音非送气化、清辅音浊化、塞音擦音化

#### 6.2.3 重音系统

| 特征 | 描述 |
|------|------|
| **位置** | 多数语言重音在末位音节 |
| **例外** | 部分后缀不吸引重音 |
| **功能** | 重音无辨义功能 |

### 6.3 句法特点

#### 6.3.1 词序

**基本词序：SOV**

```
Ben    kitab-ı   oku-du-m
我     书(宾格)  读-过去时-1sg
"我读了书"

Subject - Object - Verb
```

**修饰语位置**：

| 类型 | 位置 | 示例 |
|------|------|------|
| 形容词 | 名词前 | büyük ev（大房子） |
| 属格 | 名词前 | Ali-nin ev-i（阿里的房子） |
| 关系从句 | 名词前 | oku-yan adam（读书的人） |
| 副词 | 动词前 | hızlı koş（快跑） |

#### 6.3.2 后置词系统

突厥语使用**后置词**而非前置词：

| 土耳其语 | 功能 | 示例 |
|----------|------|------|
| -de/-da | 方位格（在...） | ev**de**（在家里） |
| -e/-a | 向格（向...） | ev**e**（向家） |
| -den/-dan | 从格（从...） | ev**den**（从家里） |
| için | 为了 | ben **için**（为了我） |
| ile | 和、用 | sen **ile**（和你） |

#### 6.3.3 从句结构

**特点：使用动名词/分词而非限定从句**

```
英语：I know that he came.
土耳其语：Onun geldiğini biliyorum.
         他  来-动名词-宾格  知道-现在时-1sg
         字面：我知道他的来。
```

### 6.4 形态学特点

#### 6.4.1 黏着语特征

**核心特点**：每个后缀表达一个语法意义，依次附加

```
Türk-çe-leş-tir-e-me-di-kler-i-miz-den-siniz
土耳其-语-化-使动-可能-否定-过去时-分词-复数-1pl所有格-从格-是-2pl

"你们是那些我们没能土耳其化的人之中的"
（展示极端黏着性的造句）
```

#### 6.4.2 格系统（土耳其语）

| 格 | 后缀 | 功能 | 示例 |
|------|------|------|------|
| **主格** | -∅ | 主语 | adam（人） |
| **宾格** | -ı/-i/-u/-ü | 定指宾语 | adam**ı**（那个人） |
| **与格** | -a/-e | 间接宾语、方向 | adam**a**（给人） |
| **方位格** | -da/-de | 在... | adam**da**（在人那里） |
| **从格** | -dan/-den | 从... | adam**dan**（从人） |
| **属格** | -ın/-in/-un/-ün | 所有者 | adam**ın**（人的） |

#### 6.4.3 动词形态

**时态-体-语气系统**：

| 复合后缀 | 含义 |
|----------|------|
| -di-/-tı- | 过去时（亲眼见证） |
| -miş-/-mış- | 过去时（非亲眼见证/传闻） |
| -iyor- | 现在进行时 |
| -ecek-/-acak- | 将来时 |
| -meli-/-malı- | 必要语气 |
| -se-/-sa- | 条件语气 |

**动词变位示例**（gel-mek "来"）：

| 人称 | 过去时 | 现在时 | 将来时 |
|------|--------|--------|--------|
| 1sg | geldim | geliyorum | geleceğim |

| 3sg | geldi | geliyor | gelecek |

| 2pl | geldiniz | geliyorsunuz | geleceksiniz |

### 6.6 IOLC常见考点

| 考点类型 | 具体内容 |
|----------|----------|
| **元音和谐** | 分析后缀变化规律、判断前后元音 |
| **格系统** | 识别格标记、分析语法关系 |
| **动词变位** | 时态-体-人称为复合后缀 |
| **词序分析** | SOV + 修饰语前置 |
| **派生构词** | 识别派生后缀的叠加顺序 |

**解题提示**：
1. 遇到后缀变化丰富、元音交替规则 → 优先考虑突厥语系特征
2. 注意元音和谐的"逆向推导"：从后缀形式推断词根元音特征
3. 格标记是定指/不定指的重要线索（宾格只用于定指宾语）

## 八、语言学竞赛常见考点

### 8.1 按语系的常考现象

| 语系 | 高频考点 |
|------|----------|
| **汉藏** | 声调分析、量词搭配、复合词结构 |
| **印欧** | 格系统分析、动词变位、历史音变 |
| **闪含** | 辅音根识别、派生模式、名词状态 |
| **南岛** | 焦点系统分析、词缀识别、代词系统 |
| **尼日尔-刚果** | 名词类别识别、声调分析、动词复杂结构 |
| **突厥** | 元音和谐、格标记分析、动词变位、后缀叠加 |

### 8.2 解题提示

1. **识别语系特征**：快速判断语系有助于缩小分析范围
2. **注意特殊现象**：
   - 凯尔特语：词首辅音变化
   - 班图语：名词类别一致
   - 菲律宾语：焦点系统
   - 闪语：辅音根
3. **利用类型学规律**：
   - SOV语言常有后置词
   - VSO语言常有前置词
   - 作格语言多为SOV或VSO

### 8.3 语系识别快速指南

| 观察点 | 指示语系 |
|--------|----------|
| 辅音根 + 简单元音 | 闪含语系 |
| 名词类别系统 + 声调 | 尼日尔-刚果语系 |
| 焦点标记 + 动词居首 | 南岛语系 |
| 声调 + 孤立特征 | 汉藏语系 |
| 丰富的格系统 + 动词变位 | 印欧语系 |
| 元音和谐 + SOV + 后缀叠加 | 突厥语系 |
