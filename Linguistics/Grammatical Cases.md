# Grammatical Cases

## 特殊格位详解

### 1 与格（Dative）

#### 定义与功能

| 项目 | 描述 |
|------|------|
| **核心功能** | 标记间接宾语（动作的接受者） |
| **英语对应** | to, for |
| **常见语境** | 给予、告诉、展示类动词 |

#### 典型用法

| 用法类型 | 描述 | 示例 |
|----------|------|------|
| **间接宾语** | 动作的接受者 | 我给**他**书 |
| **受益者** | 动作的受益方 | 我为**你**买 |
| **体验者** | 感觉/心理状态的主体 | 他觉得冷 |

#### 语言实例

**德语**：
```
Ich gebe dem Mann das Buch.
我     给     男人(与格)   书(宾格)
"我给那个男人书"
```

**拉丁语**：
```
Puer puellae librum dat.
男孩  女孩(与格) 书(宾格) 给
"男孩给女孩书"
```

**俄语**：
```
Я пишу письмо другу.
我 写   信    朋友(与格)
"我给朋友写信"
```

**古希腊语**：
```
δῶρον τῷ ἀνδρί
礼物    男人(与格)
"给男人的礼物"
```

#### 与格标记形式

| 语言 | 单数标记 | 复数标记 |
|------|----------|----------|
| 德语（阳性） | dem | den |
| 拉丁语（第一变格） | -ae | -īs |
| 俄语（阳性） | -у/-ю | -ам/-ям |
| 古希腊语 | -ῳ | -οις |

### 2 具格/工具格（Instrumental）

#### 定义与功能

| 项目 | 描述 |
|------|------|
| **核心功能** | 标记完成动作所使用的工具或手段 |
| **英语对应** | with, by, by means of |
| **常见语境** | 工具、手段、伴随 |

#### 典型用法

| 用法类型 | 描述 | 示例 |
|----------|------|------|
| **工具** | 动作使用的工具 | 用**笔**写字 |
| **手段** | 完成动作的方式 | 以**欺骗**获得 |
| **伴随** | 一起进行动作 | 与**朋友**同行 |
| **方式** | 动作进行的方式 | 以**快速**移动 |

#### 语言实例

**俄语**：
```
Он пишет карандашом.
他 写    铅笔(工具格)
"他用铅笔写字"

Я иду с другом.
我 走  和 朋友(工具格)
"我和朋友一起走"
```

**芬兰语**（工具格已并入具格）：
```
Kirjoitan kynällä.
写      笔(具格)
"我用笔写"
```

**梵语**：
```
लेखनेन लिखामि。
笔(工具格) 写
"我用笔写"
```

#### 具格与其他格的关系

在许多语言中，具格与其他格合并：

| 语言 | 具格合并情况 |
|------|--------------|
| **俄语** | 工具格 + 伴随格 |
| **芬兰语** | 工具格 + 手段格 |
| **梵语** | 独立工具格 |
| **拉丁语** | 无独立具格（用介词 cum + 夺格） |

### 3 夺格/离格（Ablative）

#### 定义与功能

| 项目 | 描述 |
|------|------|
| **核心功能** | 标记动作的来源、起点、原因 |
| **英语对应** | from, out of, by, with |
| **原始功能** | 表示"分离"、"夺走" |

#### 典型用法

| 用法类型 | 描述 | 示例 |
|----------|------|------|
| **来源** | 动作的起点 | 从**家里**来 |
| **原因** | 动作的原因 | 因**恐惧**逃跑 |
| **比较** | 比较的对象 | 比**他**高 |
| **手段** | 完成动作的方式 | 以**努力**成功 |

#### 语言实例

**拉丁语**：
```
Rōmā vēnit.
罗马(夺格) 来了
"他从罗马来"

Timōre fugit.
恐惧(夺格) 逃跑
"他因恐惧而逃跑"
```

**梵语**：
```
ग्रामात् आगच्छति。
村子(夺格) 来
"他从村子来"
```

**芬兰语**：
```
Tulen kotoa.
来   家(夺格)
"我从家来"
```

#### 夺格的历史演变

| 阶段 | 描述 |
|------|------|
| **原始印欧语** | 有独立夺格 |
| **拉丁语** | 保留夺格，融合工具格、方位格功能 |
| **罗曼语族** | 夺格消失，由介词替代（de, da, de） |
| **斯拉夫语族** | 部分语言保留夺格功能 |

#### 拉丁语夺格的多功能合并

拉丁语夺格合并了三种原始功能：

| 原始格 | 功能 | 拉丁语示例 |
|--------|------|------------|
| **工具格** | 工具 | ferrō（用铁） |
| **方位格** | 位置 | Rōmae（在罗马） |
| **夺格** | 来源 | Rōmā（从罗马） |

### 4 位格/方位格（Locative）

#### 定义与功能

| 项目 | 描述 |
|------|------|
| **核心功能** | 标记动作发生的地点或位置 |
| **英语对应** | in, on, at |
| **常见语境** | 位置、场所 |

#### 典型用法

| 用法类型 | 描述 | 示例 |
|----------|------|------|
| **位置** | 动作发生的地点 | 在**家里** |
| **时间** | 动作发生的时间 | 在**夏天** |
| **领域** | 活动的范围 | 在**音乐方面** |

#### 语言实例

**拉丁语**（仅限地名和时间词）：
```
Rōmae habitat.
罗马(位格) 居住
"他住在罗马"

domī manet.
家(位格) 留
"他留在家"
```

**俄语**：
```
Я живу в Москве.
我 住  在 莫斯科(位格)
"我住在莫斯科"
```

**芬兰语**（丰富的方位格系统）：

| 格位 | 功能 | 示例 |
|------|------|------|
| **内格（Inessive）** | 在...里面 | talossa（在房子里） |
| **出格（Elative）** | 从...里面出来 | talosta（从房子里） |
| **入格（Illative）** | 进入...里面 | taloon（进入房子） |
| **位格（Adessive）** | 在...上面/旁边 | talolla（在房子旁） |
| **离格（Ablative）** | 从...上面/旁边 | talolta（从房子旁） |
| **向格（Allative）** | 向...上面/旁边 | talolle（向房子旁） |

#### 匈牙利语的方位格系统

| 格位 | 后缀 | 示例 |
|------|------|------|
| **内格** | -ban/-ben | házban（在房子里） |
| **出格** | -ból/-ből | házból（从房子里） |
| **入格** | -ba/-be | házba（进入房子） |
| **上位格** | -n | házon（在房子上） |
| **上出格** | -ról/-ről | házról（从房子上） |
| **上入格** | -ra/-re | házra（到房子上） |

### 呼格（Vocative）

#### 定义与功能

| 项目 | 描述 |
|------|------|
| **核心功能** | 标记直接称呼或呼唤的对象 |
| **英语对应** | 无专门标记（使用语调） |
| **常见语境** | 称呼、呼唤、祈使 |

#### 典型用法

| 用法类型 | 描述 | 示例 |
|----------|------|------|
| **称呼** | 直接称呼某人 | "**约翰**，你来" |
| **呼唤** | 呼唤某人 | "**妈妈**！" |
| **祈使** | 命令或请求的对象 | "**先生**，请帮我" |

#### 语言实例

**拉丁语**：
```
Marce, veni!
马库斯(呼格) 来
"马库斯，来！"

Brute, fili mi!
布鲁图斯(呼格) 儿子 我的
"布鲁图斯，我的儿子！"
```

**俄语**：
```
Иван! Иди сюда!
伊万(呼格) 来 这里
"伊万！来这里！"

Мама!
妈妈(呼格)
"妈妈！"
```

**波兰语**：
```
Panie profesorze!
先生(呼格) 教授(呼格)
"教授先生！"
```

**古希腊语**：
```
ὦ ἄνθρωπε
哦 人(呼格)
"哦，人啊！"
```

#### 呼格的形式特点

| 语言 | 呼格形式特点 |
|------|--------------|
| **拉丁语** | 名词有专门呼格形式（第二变格：-us → -e） |
| **俄语** | 部分名词有专门呼格形式 |
| **波兰语** | 有完整呼格系统 |
| **英语** | 无专门呼格形式 |
| **汉语** | 无专门呼格形式 |

#### 呼格标记示例

| 语言 | 主格 | 呼格 |
|------|------|------|
| 拉丁语 | Marcus | Marce |
| 拉丁语 | filius | fili |
| 俄语 | друг（朋友） | друге |
| 波兰语 | pan（先生） | panie |
| 波兰语 | profesor | profesorze |

### 作格（Ergative）

#### 定义与功能

| 项目 | 描述 |
|------|------|
| **核心功能** | 在作格-绝对格系统中，标记及物动词的施事者（A） |
| **对齐类型** | 作格-绝对格对齐 |
| **与主格的区别** | 主格标记S和A；作格只标记A |

> 详见 [[Syntax]] 第4-"形态句法对齐"

#### 对齐类型对比

```
对齐类型        S     A     O
主格-宾格      █     █     ░    （S和A相同）
作格-绝对格    █     ░     █    （S和O相同）

█ = 同色表示标记相同
S = 不及物动词主语
A = 及物动词施事者
O = 及物动词宾语
```

#### 语言实例

**Dyirbal语**：
```
不及物：Bayi yaŗa buŗan.
        男孩(S) 走
        "男孩走了"

及物：Bayi yaŗa baŋgul jugumbil-ŋgu buŗan.
      男孩(O)   女人(A)-作格      看见
      "女人看见了男孩"
```

**巴斯克语**：
```
不及物：Gizona etorri da.
        男人(S) 来了
        "男人来了"

及物：Gizonak emakumea ikusi du.
      男人(A)-作格 女人(O)  看见
      "男人看见了女人"
```

#### 作格标记形式

| 语言 | 作格标记 | 绝对格标记 |
|------|----------|------------|
| Dyirbal | -ŋgu | 零标记 |
| 巴斯克语 | -k | 零标记（单数）/-ak（复数） |
| 格鲁吉亚语 | -ma（仅用于特定时态） | -i（主格形式） |
| 因纽特语 | -up | 零标记 |

#### 作格语言的类型学特点

| 特点 | 描述 |
|------|------|
| **词序倾向** | 作格语言几乎都是动词居首（VSO）或动词居末（SOV） |
| **罕见词序** | 作格语言极少是SVO语序 |
| **分布比例** | 约占世界语言的四分之一 |

### 7 通格/绝对格（Absolutive）

#### 定义与功能

| 项目 | 描述 |
|------|------|
| **核心功能** | 在作格-绝对格系统中，标记不及物动词主语（S）和及物动词宾语（O） |
| **对齐类型** | 作格-绝对格对齐 |
| **典型标记** | 常为零标记（无词缀） |

#### 通格与其他格的关系

| 对齐系统 | S | A | O |
|----------|---|---|---|
| 主格-宾格 | 主格 | 主格 | 宾格 |
| 作格-绝对格 | **绝对格** | 作格 | **绝对格** |

#### 语言实例

**Dyirbal语**：
```
Bayi yaŗa buŗan.
标记 绝对格(S) 走
"男孩走了"

Bayi yaŗa baŋgul jugumbil-ŋgu buŗan.
标记 绝对格(O)  女人-作格    看见
"女人看见了男孩"
```
- yaŗa（男孩）在两句中都使用绝对格形式（无标记）

**巴斯克语**：
```
Gizona etorri da.
男人(绝对格) 来了
"男人来了"

Emakumea gizonak ikusi du.
女人(绝对格) 男人(作格) 看见
"男人看见了女人"
```

#### 通格标记形式

| 语言 | 通格标记 | 特点 |
|------|----------|------|
| Dyirbal | 零标记 | 无显性标记 |
| 巴斯克语 | 零标记（单数） | 复数有标记 |
| 因纽特语 | 零标记 | 无显性标记 |
| 格鲁吉亚语 | -i（单数） | 有显性标记 |

## 2. 格位系统对比

### 1 主格-宾格系统 vs 作格-绝对格系统

| 系统 | S | A | O | 代表语言 |
|------|---|---|---|----------|
| **主格-宾格** | 主格 | 主格 | 宾格 | 英语、德语、拉丁语、日语 |
| **作格-绝对格** | 绝对格 | 作格 | 绝对格 | Dyirbal、巴斯克语、格鲁吉亚语 |

### 2 常见语言的格系统

| 语言 | 格数量 | 主要格位 |
|------|--------|----------|
| **汉语** | 0 | 无格（靠词序） |
| **英语** | 2（代词）/ 0（名词） | 主格、宾格 |
| **德语** | 4 | 主格、宾格、与格、属格 |
| **拉丁语** | | 主格、宾格、与格、属格、夺格、呼格 |
| **俄语** | | 主格、宾格、与格、属格、工具格、位格 |
| **芬兰语** | 1| 包含丰富的方位格 |
| **匈牙利语** | 18 | 包含丰富的方位格 |
| **巴斯克语** | 12+ | 绝对格、作格、与格等 |

### 3 格位语义功能矩阵

| 格位 | 句法角色 | 语义角色 | 典型介词 |
|------|----------|----------|----------|
| 主格 | 主语 | 施事、体验者 | - |
| 宾格 | 直接宾语 | 受事、客体 | - |
| 与格 | 间接宾语 | 接受者、受益者 | to, for |
| 属格 | 定语 | 所有者、整体的一部分 | of |
| 具格 | 状语 | 工具、手段 | with, by |
| 夺格 | 状语 | 来源、原因 | from |
| 位格 | 状语 | 地点、位置 | in, on, at |
| 呼格 | 独立成分 | 被称呼者 | - |
| 作格 | 及物主语 | 施事 | - |
| 绝对格 | 不及物主语/宾语 | 受事、客体 | - |

## 3. 分裂对齐现象

某些语言可有两种或多种对齐类型，根据特定条件切换。

### 1 分裂条件

| 条件 | 示例语言 |
|------|----------|
| **时态** | 普什图语：过去时用作格，现在时用主格-宾格 |
| **人称** | 某些语言第一、二人称用主格-宾格，第三人称用作格-绝对格 |
| **论元类型** | 名词vs代词的不同对齐 |
| **动词类型** | 某些动词触发不同对齐 |

### 2 普什图语示例

| 时态 | 对齐类型 | 1sg主格/作格 | 1sg宾格/绝对格 |
|------|----------|--------------|----------------|
| **过去时** | 作格-绝对格 | me | ze |
| **现在时** | 主格-宾格 | ze | me |

**过去时**（作格-绝对格）：
```
Ze wlarrem.（我走了）—— ze = S（绝对格）
Ze me woleed.（我看见了他）—— me = A（作格）, ze = O（绝对格）
```

**现在时**（主格-宾格）：
```
Ze dzem.（我走）—— ze = S（主格）
Ze yay weenem.（我看见他）—— ze = A（主格）, yay = O（宾格）
```

## 4. 解题方法论

### 1 识别格系统类型

| 步骤 | 操作 |
|------|------|
| **Step 1** | 找出不及物动词句子（通常较短） |
| **Step 2** | 找出及物动词句子（通常较长） |
| **Step 3** | 比较S、A、O的标记形式 |
| **Step 4** | 判断对齐类型（主格-宾格 vs 作格-绝对格） |

### 2 格标记分析技巧

| 技巧 | 描述 |
|------|------|
| **对比分析** | 比较相同名词在不同句子中的形式变化 |
| **频率统计** | 统计不同词尾的出现频率和语境 |
| **语义角色映射** | 根据翻译推断名词的语义角色 |
| **排除法** | 排除已知格位，确定剩余格位功能 |

### 3 常见格标记形式

| 格位 | 常见标记特征 |
|------|--------------|
| 主格 | 常为零标记或基础形式 |
| 宾格 | 常为鼻音结尾或元音变化 |
| 与格 | 常有专门后缀 |
| 属格 | 常有专门后缀 |
| 作格 | 常为特殊后缀（如-ŋgu, -k） |
| 绝对格 | 常为零标记 |

### 4 注意事项

1. **多义格位**：同一格位可能有多种功能（如拉丁语夺格）
2. **合并格位**：多种格功能可能合并为一个格
3. **分裂系统**：注意不同语境下的对齐变化
4. **语序配合**：格系统与语序密切相关
