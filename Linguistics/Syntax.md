# Syntax
## 1. 引言
[[Chomsky - Automata]]
[[Chomsky - Formal Grammar in English]]
句法研究句子和短语的结构。句子通常由名词短语（NP）和动词短语（VP）组成：

$$Sentence = NP + VP$$

句法问题本质上是名词形态学问题与动词形态学问题的结合，但由于NP和VP在同一结构中相关联，会出现额外的相互作用：

- 名词短语可包含指示名词在句中角色的词素（主语、宾语、施事者、受事者等）
- 论元可通过名词表达（不仅限于代词），因此动词短语中可有更多区分
- 可添加 ±human（人类与非人类）等区分

## 2. 词序（Word Order）

### 2.1 基本词序类型

根据主语（S）、动词（V）、宾语（O）的顺序，有六种可能模式：

| 词序 | 频率 | 语言示例 |
|------|------|----------|
| **SOV** | ~45% | 日语、土耳其语、印地语 |
| **SVO** | ~42% | 英语、汉语、法语 |
| **VSO** | ~8% | 古典阿拉伯语、他加禄语、凯尔特语 |
| **VOS** | 罕见 | 马尔加什语 |
| **OVS** | 极罕见 | Hixkaryana语 |
| **OSV** | <0.5% | Apurinã语 |

> **统计规律**：主语在宾语之前的模式（SOV、SVO、VOS、OSV）最常见，约占95%以上的语言。

### 2.2 固定词序 vs 灵活词序

| 类型 | 特点 | 形态系统 |
|------|------|----------|
| **固定词序** | 几乎每句话都遵循同一词序 | 较简单的屈折系统 |
| **灵活词序** | 多种词序都可使用 | 较丰富的屈折系统 |

**英语示例（固定词序）**：
- The cat saw the boy（猫看见男孩）
- The boy saw the cat（男孩看见猫）
- 仅词序不同，主语和宾语角色随之改变

**罗马尼亚语示例（灵活词序）**：
- Uriașul îl dă pe copil tatălui（巨人把孩子给父亲）
- Tatălui îl dă uriașul pe copil（父亲给孩子被巨人给）
- Pe copil îl dă uriașul tatălui（孩子被巨人给父亲）
- 三句意义相同，通过形态标记区分角色

### 2.3 词序描述原则

**句子层面**：描述主要成分的相对位置
- 例如：SVOInstr（主语-动词-宾语-工具格）

**短语内部**：描述名词与修饰语的关系
- 例如：Adj.–Noun 或 Modifier–Noun

> **类型学规律**：许多语言中，句子的词序与名词短语的词序镜像对应。如果句子是动词后置，名词短语也倾向于中心词后置。

### 2.4 Nung语示例

**否定命令式词序**：Da V O
**陈述式词序**：S[...]V O / 'likethat' (mi = Question)

**副词/情态动词顺序**：
```
时间副词 → 'then' → 'was about to' → 'continue to' → 'also' → 'not' → 'truly' → 情态动词 → 'alone'
```

## 3. 聚焦（Focusing）

聚焦是强调句子某一部分的句法过程。

### 3.1 英语中的聚焦方式

| 方式 | 示例 | 强调部分 |
|------|------|----------|
| 语调 | He **hit** the dog vs He hit the **dog** | 不同成分 |
| 分裂句 | I saw a cat vs **It is a cat** that I saw | 宾语 |

### 3.2 Wolof语的聚焦系统

Wolof语有四组代词，用于不同的聚焦情况：

| 类型 | 1sg | 2sg | 用途 |
|------|-----|-----|------|
| 主语（S） | man | yow | 默认 |
| 宾语（O） | ma | la | 默认 |
| 动词聚焦 | damay | dangay | 动词被强调时替代主语 |
| 宾语聚焦 | laa | nga | 宾语被强调时替代宾语 |

**示例**（动词：gisoon = 'see'）：

| 聚焦类型 | 句子 | 翻译 |
|----------|------|------|
| 中性（无聚焦） | Man la gisoon | I saw you |
| 动词聚焦 | Damay gisoon la | I **saw** you |
| 宾语聚焦 | Nga man gisoon | I saw **you** |

## 4. 形态句法对齐（Morphosyntactic Alignment）

### 4.1 基本概念

形态句法对齐涉及三个动词论元的标记方式：

| 符号 | 含义 |
|------|------|
| **S** | 不及物动词的主语 |
| **A** | 及物动词的主语（施事者） |
| **O** | 及物动词的宾语（受事者） |

### 4.2 对齐类型

| 对齐类型 | 标记方式 | 描述 |
|----------|----------|------|
| **主格-宾格（Nominative-Accusative）** | S = A ≠ O | 最常见，S和A标记相同 |
| **作格-绝对格（Ergative-Absolutive）** | S = O ≠ A | S和O标记相同，A单独标记 |
| **三分（Tripartite）** | S ≠ A ≠ O | 三者标记各不相同 |
| **直接（Direct）** | S = A = O | 三者标记相同（或均无标记） |
| **及物（Transitive）** | S ≠ A = O | 极罕见，A和O标记相同 |

### 4.3 对齐类型图示

```
对齐类型        S     A     O
主格-宾格      █     █     ░
作格-绝对格    █     ░     █
三分          █     ▓     ░
直接          █     █     █
及物          █     ▓     ▓

█ = 同色表示标记相同
```

### 4.4 四种语言的示例

| 语言 | 短句结构 | 长句结构 | 对齐类型 |
|------|----------|----------|----------|
| **Dyirbal** | Bayi S buŗan | Bayi O baŋgul A-ŋgu buŗan | 作格-绝对格 |
| **Tahitian** | 'uahi'o te S | 'uahi'o te A 'ite O | 直接 |
| **Nez-Perce** | S peexne | A-nim peexne O-ne | 三分 |
| **Wappo** | S-i nawta | A-i O nawta | 主格-宾格 |

**Dyirbal语详解**：
- 不及物动词：Bayi yaŗa buŗan（男孩走）
- 及物动词：Bayi yaŗa baŋgul jugumbil-ŋgu buŗan（男孩被女人看见）
- S和O使用相同标记（无标记或零词素）
- A使用后缀 -ŋgu（作格标记）

### 4.5 英语中的对齐

英语展示了一种有趣的混合情况：
- **代词**：主格-宾格对齐（he – him, I – me）
- **名词**：直接对齐（无标记）

## 5. 分裂对齐（Split Alignment）

某些语言可有两种或多种对齐类型，每种出现在特定语言环境中。

### 5.1 普什图语（Pashto）示例

普什图语根据时态分裂对齐：

| 时态 | 对齐类型 | 1sg主格/作格 | 1sg宾格/绝对格 |
|------|----------|--------------|----------------|
| **过去时** | 作格-绝对格 | me | ze |
| **现在时** | 主格-宾格 | ze | me |

**过去时示例**：
- Ze wlarrem.（我走了）—— ze = S
- Ze me woleed.（我看见了他）—— me = A, ze = O
- Ze me yay woleedelem.（他看见了我）—— yay = A, ze = O

**现在时示例**：
- Ze dzem.（我走）—— ze = S
- Ze yay weenem.（我看见他）—— ze = A, yay = O
- Dai me weenee.（他看见我）—— dai = A, me = O

### 5.2 分裂对齐的条件

分裂对齐可由多种因素触发：

| 条件 | 示例 |
|------|------|
| **时态** | 普什图语 |
| **人称** | 某些语言第一、二人称用主格-宾格，第三人称用作格-绝对格 |
| **论元类型** | 名词vs代词的不同对齐 |
| **话语显著性** | 有定vs无定论元 |

### 5.3 重要提示

> 作格语言约占世界语言的四分之一，在语言学问题中比例较高。作格语言几乎都是动词居首或动词居末，极少是SVO语序。

## 6. 解题方法论

### 6.1 句法问题的分析步骤

1. **识别主语和宾语标记**
2. **确定词序**
3. **分析不及物句和及物句的论元标记**
4. **判断对齐类型**
5. **写出完整规则**

### 6.2 词序分析技巧

1. 找出重复出现的关键词（通常与动词相关）
2. 通过句子长度差异识别动词位置
3. 对比长短句结构推断论元位置
4. 使用频率表匹配翻译

### 6.3 对齐类型的判断

| 步骤 | 操作 |
|------|------|
| 1 | 找出不及物动词句子（通常较短） |
| 2 | 找出及物动词句子（通常较长） |
| 3 | 比较S、A、O的标记 |
| 4 | 判断对齐类型 |

### 6.4 规则书写格式

```
Word order: SVO / SOV / VSO...

Morphosyntactic alignment: Nominative-Accusative / Ergative-Absolutive...

Subject markers: ...
Object markers: ...
Verb: ...
```
