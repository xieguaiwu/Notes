# Number Systems

## 1. 引言

数字系统问题涉及**进位制（base）**的概念和特征。

### 1.1 罗马尼亚语示例

数字432写作 *patrusutetreizecișidoi*，可分解为：

| 词素 | 含义 | 类型 |
|------|------|------|
| patru | 'four' | 数码（digit） |
| sute | 'hundreds' | 位数（order） |
| trei | 'three' | 数码 |
| zeci | 'tens' | 位数 |
| și | 'and' | 加法标记 |
| doi | 'two' | 数码 |

**结构规则**：
- 词序：从大到小（百位→十位→个位）
- 乘数放在位数之前
- 十位的词素与乘数合成一词
- 十位与个位用 *și* 分隔

**一般结构**：
$100X + 10Y + Z = X\ sute\ Y\text{-}zeci\ și\ Z$

### 1.2 进位制的类型

| 进制 | 常见程度 | 语言示例 |
|------|----------|----------|
| **10（十进制）** | 最常见 | 大多数语言 |
| **6, 20** | 较常见 | - |
| **4, 5, 12, 60** | 较少见 | - |
| **22–28** | 特殊（基于身体部位） | Oksapmin, Kaugel |
| **7, 9, 11, 13** | 极罕见 | 目前无已知自然语言使用 |

### 1.3 不同进制的表示

**六进制示例**（base = 6）：

| 十进制数 | 六进制表示 |
|----------|------------|
| 10 | 6 + 4 |
| 25 | (4 × 6) + 1 |
| 100 | (2 × 36) + (4 × 6) + 4 |

六进制的位数：6¹=6, 6²=36, 6³=216...

## 2. 数字系统的两个关键特征

### 2.1 词序（方向）

| 类型 | 描述 | 示例 |
|------|------|------|
| **大-小** | 百位→十位→个位 | 英语、罗马尼亚语 |
| **小-大** | 个位→十位→百位 | Quenya语 |
| **混合** | 非严格顺序 | 德语 |

**德语示例**：432 = *vierhundertzweiunddreißig*
- 结构：4-百位-2-加法-3-十位
- 词序：百位-个位-十位

### 2.2 加法与乘法的标记

| 语言 | 加法标记 | 乘法标记 | 示例 |
|------|----------|----------|------|
| **法语** | 隐含 | 隐含 | 2,510 = (2×1000)+(5×100)+10 |
| **罗马尼亚语** | 显性（și） | 隐含 | patrusute（四百） |
| **德语** | 显性（und） | 隐含 | vierhundert（四百） |

**法语规则**：
- 数码放在位数之前 → 乘法
- 放在位数之后 → 加法

## 3. Quenya语示例（十进制）

### 3.1 语料

| Quenya | 数值 |
|--------|------|
| neldë | 3 |
| canta | 4 |
| lempë | 5 |
| otso | 7 |
| tolto | 8 |
| nelcëa | 13 |
| encëa | 16 |
| enquëyucainen | 26 |
| minënelcainen | 31 |
| cancainen | 40 |
| attatolcainen | 82 |
| attatolcainentuxa | 182 |
| nertënelcainenlemtuxa | 539 |

### 3.2 分析

**结构发现**：
- 13和16都以 *-cëa* 结尾 → 10+X 的标记
- 82和182只差一个 *tuxa* → tuxa = 100
- 右列第二个词都以 *cainen* 结尾 → 十位标记
- 词序：**个位-十位-百位**

### 3.3 数码变化表

| 数码 | 个位（X） | 10+X | 10X | 100X |
|------|-----------|------|-----|------|
| 1 | minë | ∅ | - | - |
| 2 | atta | yu- | - | - |
| 3 | neldë | nel- | nel- | nel- |
| 4 | canta | can- | can- | - |
| 5 | lempë | lem- | lem- | lem- |
| 6 | enquë | en- | en- | - |
| 7 | otso | - | - | - |
| 8 | tolto | tol- | tol- | - |
| 9 | nertë | - | - | - |

> **规律**：复合词中使用数码的第一音节，数字2有不规则形式 *yu-*

### 3.4 规则

```
10 + X = X'-cëa
100X + 10Y + Z = Z Y'-cainen X'-tuxa
```

（X'、Y'表示数码的缩略形式，即第一音节）

## 4. Embera Chami语示例（五进制）

### 4.1 问题

已知两个等式：
1. umbea + huasomakwimane = ommehuasomaomme
2. ommehuasomakwimane + huasomaabba = kwimanehuasoma

### 4.2 分析

**步骤1**：识别结构
- 单词类型：单独词 或 X huasoma Y
- *huasoma* 出现频繁 → 可能是基数

**步骤2**：数数码个数
- 只有四个数码词：umbea, kwimane, omme, abba
- 推测 base = 5，huasoma = 5

**步骤3**：确定词序
- huasoma X = 5 + X（不是 5 × X）
- X huasoma Y = 5X + Y

**步骤4**：利用进位推断数码
- 从等式(1)推导：omme = 2
- 从等式(2)推导：abba = 1, kwimane = 4, umbea = 3

### 4.3 规则

| 数码 | 值 |
|------|-----|
| abba | 1 |
| omme | 2 |
| umbea | 3 |
| kwimane | 4 |
| huasoma | 5 |

**结构**：5X + Y = X huasoma Y

### 4.4 翻译

| 任务 | 答案 |
|------|------|
| 17 | 3×5+2 = umbeahuasomaomme |
| 23 | 4×5+3 = kwimanehuasomaumbea |

## 5. Yup'ik语示例

### 5.1 身体部位计数

Yup'ik数字词可分解为有意义的部分，与身体部位相关：
- talliman (5) = 'arm'（手臂）
- arvinlegen (6) = 'crossover'（交叉，换手继续数）

### 5.2 魔方阵问题

**魔方阵规则**：每行、每列、每对角线之和相等（对于1-9，和为15）

**解答步骤**：
1. 中间数必为5
2. 利用已知信息填充
3. 根据Yup'ik线索验证

**结果**：
```
4 9 2
3 5 7
8 1 6
```

## 6. 解题方法论

### 6.1 确定进制的方法

1. **数数码个数**：若数出11个数码词，进制可能是10或12
2. **误差范围**：结果应考虑±2的误差
3. **只数数码**：不要把位数词、加法/乘法标记算进去

### 6.2 关键观察点

| 观察点 | 用途 |
|--------|------|
| 重复出现的词 | 可能是基数 |
| 词的结构变化 | 区分数码和位数 |
| 进位（carryover） | 推断数码2和3 |
| 词序模式 | 确定大-小或小-大 |

### 6.3 解题步骤

1. 识别数字的结构模式
2. 区分数码词和位数词
3. 数数码词的数量，估计进制
4. 确定词序方向
5. 分析加法和乘法的标记方式
6. 利用等式或进位推断具体数码值

### 6.4 常见陷阱

- 不同语境下数码可能有不同形式
- 1和2常有不规则变化
- 词序可能不严格（如德语）
- 加法和乘法可能隐含不标
