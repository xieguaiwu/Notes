# Phonology

## 1. 引言

音系学（Phonology）研究声音的**功能**及其在语言系统中的**组合模式**，与语音学（Phonetics[[Phonetics]]）关注声音的物理方面形成对比。


| 学科 | 研究对象 | 核心问题 |
|------|----------|----------|
| 语音学 | 声音的物理属性 | 声音如何产生、传播、感知？ |
| 音系学 | 声音的功能与模式 | 声音如何在语言中组合、变化？ |

## 2. 音系学标记法

### 2.1 特征标记 `[±A]`

表示声音具有（+）或不具有（-）某特征A。

| 例子 | 特征描述 |
|------|----------|
| [m] | `[+bilabial +nasal]` |
| [s] | `[+alveolar –voice +fricative]` |

### 2.2 音系规则 `A → B / C_V`

读作："声音A变成B，如果它出现在某环境中"

| 符号 | 含义 |
|------|------|
| `→` | 变成 |
| `/` | 条件环境 |
| `_` | 受影响声音的位置 |
| `#` | 词界 |
| `.` | 音节边界 |
| `C⁰` | 辅音序列（或无辅音） |
| `{}` | 多种可能性 |

**示例：**
- `k → g / _a`：在a之前k变成g
- `k → g / {#, V}_`：在词首或在元音之后，k变成g

### 2.3 希腊字母标记法

使用α、β、γ等代替±，表示某特征在规则多个部分具有相同值。

**同化规则示例：**
```
n → [αPLACE] / [+stop αPLACE]
```
读作："在塞音之前，n将改变其发音部位以匹配塞音的发音部位。"

## 3. 互补分布（Complementary Distribution）

互补分布是两个或多个声音之间的关系，其中每个声音只能出现在特定环境中。

### 3.1 分析步骤

1. 根据包含的声音将单词分成两组
2. 检查位置因素（词首/词末/元音前后）
3. 检查相邻声音的类型或特征
4. 找出可预测环境，写出音系规则

### 3.2 经典案例

**韩语 l/r 分布：**
- 字符ﾩ在两元音之间发音为r，否则为l
- 规则：`l → r / V_V`

**西班牙语 d/ð 分布：**
| d音 | ð音 |
|-----|-----|
| 词首或辅音后 | 元音后 |
| [d]ecir, aban[d]onar | la[ð]o, senti[ð]o |

- 规则：`d → ð / V_`

**卢干达语 l/r 分布：**
| l音 | r音 |
|-----|-----|
| 词首或元音o后 | 元音i和e后 |

- 规则：`l → r / [+front]_`（i和e都是前元音）

## 4. 音系过程（Phonological Processes）

### 4.1 声音的删除

| 类型 | 术语 | 描述 |
|------|------|------|
| 词首删除 | Aphaeresis | 词首的删除 |
| 词中删除 | Syncope | 词中的删除 |
| 词末删除 | Apocope | 词末的删除 |

**示例（阿伊努语）：**
- 后缀-re变成-e，如果添加到以-r结尾的词
- 规则：`r → ∅ / r_e`

### 4.2 声音的插入（Epenthesis）

增音通常发生在：
- 两个辅音之间
- 两个元音之间
- 词首（Prosthesis）

**示例：**
- 英语 horses 复数标记前有增音元音
- 英式英语连接r：drawing → draw-r-ing

### 4.3 换位（Metathesis）

两个或多个声音交换位置。

| 语言 | 原形式 | 换位后 |
|------|--------|--------|
| 古英语 | þridda | third |
| 拉丁语 | integrum | 罗马尼亚语 întreg |

### 4.4 同化（Assimilation）

最重要的音系过程类型，一个声音因附近另一个声音而转变。

#### 同化的分类

| 分类标准 | 类型 | 描述 |
|----------|------|------|
| 同化程度 | 完全同化 | 目标声音变得与触发声音完全相同 |
| | 部分同化 | 只改变某些特征 |
| 声音位置 | 接触同化 | 两个声音相邻 |
| | 远距离同化（和谐） | 声音不相邻 |
| 同化方向 | 顺行同化 | 后面的声音改变 |
| | 逆行同化 | 前面的声音改变 |

**示例（英语）：**
- can be → [kæmbi]（快速语流中n同化为m）

### 4.5 鼻辅音的行为

鼻辅音极易发生同化，常同化到相邻辅音的发音部位。

> **核心规则**：避免让鼻音直接跟在清辅音后面。

各语言的处理方式：

| 语言 | 处理方式 |
|------|----------|
| 印尼语 | 清塞音被删除 |
| Mandar语 | 鼻辅音完全同化 |
| 克丘亚语、Zoque语 | 清辅音变成浊音 |
| Lunyole语 | 鼻辅音被删除 |

## 5. 元音和谐（Vowel Harmony）

元音和谐是一种特殊的远距离同化，在突厥语系语言中常见。

### 5.1 土耳其语元音系统

| | 前元音 | | 后元音 | |
|--|--------|--------|--------|--------|
| | 不圆唇 | 圆唇 | 不圆唇 | 圆唇 |
| 闭 | i | ü | ı | u |
| 开 | e | ö | a | o |

### 5.2 两类元音和谐

**第一类：前后和谐**
- 复数后缀：-lar（后元音后）/ -ler（前元音后）
- 方位格后缀：-da（后元音后）/ -de（前元音后）

**第二类：圆唇和谐（考虑开闭）**
- 1sg所有格后缀：-im, -ım, -um, -üm

| 词的最后元音 | 所有格后缀 |
|--------------|------------|
| 前不圆唇（e, i） | -im |
| 后不圆唇（a, ı） | -ım |
| 前圆唇（ö, ü） | -üm |
| 后圆唇（o, u） | -um |

### 5.3 Evenki语的三种元音和谐

Evenki语展示了三种不同类型的元音和谐：

| 和谐类型 | 影响范围 | 应用场景 |
|----------|----------|----------|
| 完全同化 | 完全复制元音 | 主格复数后缀 |
| 圆唇和谐 | 只影响圆唇元音，基于前后性 | 方位格后缀 |
| 前后和谐 | 基于前后性，与圆唇无关 | 1pl所有格后缀 |

> **注意**：元音i是中性元音，既不触发也不影响元音和谐。

## 6. 词首辅音变化（Initial Consonant Mutation）

常见于凯尔特语言（爱尔兰语、威尔士语等），词的第一个辅音根据语法语境发生变化。

### 6.1 威尔士语示例

| 语境1 | 语境2 | 语境3 |
|-------|-------|-------|
| p | b | mh |
| c | g | ngh |
| t | d | nh |

**规律**：发音部位被保留，只有发音方式和浊音发生变化
- 清塞音 → 浊塞音 → 清鼻音

## 7. 音系学解题方法论

### 7.1 互补分布问题

1. 将单词按包含的声音分组
2. 排除不相关的位置因素
3. 找出声音出现的规律性环境
4. 用音系规则描述变化

### 7.2 方言/相关语言问题

1. 找出不同方言间的声音对应关系
2. 检查任务是否需要完整的规则概括
3. 根据对应关系填空

### 7.3 词形变化问题

1. 确定基本形式（通常是无标记形式）
2. 分离词干和语素
3. 找出语素变体的分布规律
4. 判断是否存在元音和谐或同化过程
