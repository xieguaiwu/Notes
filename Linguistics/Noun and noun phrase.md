# Noun and noun phrase

## 1. 引言

### 1.1 基本概念

传统语法定义：

| 概念 | 定义 | 例子 |
|------|------|------|
| **名词（Noun）** | 指称特定物体、地点、存在等的词 | - |
| **专有名词（Proper noun）** | 指代人名或地名 | Giovanni, London |
| **普通名词（Common noun）** | 描述一类实体 | city, class, girl, cat |
| **名词短语（Noun phrase）** | 以名词为中心，担任主语、宾语等语法功能 | the good boy, the boy inside the house |

### 1.2 句子结构
[[Chomsky - Automata]]
句子（S）可定义为名词短语（NP）和动词短语（VP）的组合：

$$S = NP + VP$$

**示例：**
- *Andrew eats.* → NP = Andrew, VP = eats
- *A boy eats those sandwiches.* → NP = A boy, VP = eats those sandwiches

> **注意**：名词短语包括名词（中心成分）和其他从属成分（定冠词、所有格、形容词等），统称为**修饰语（modifiers）**。

### 1.3 形态学基本术语

| 术语 | 定义 | 例子 |
|------|------|------|
| **词素（Morpheme）** | 具有意义或语法功能的最小语言单位 | desire, -able, un-, -ity |
| **语素变体（Allomorph）** | 同一词素的不同变体，基于音系条件选择 | im- (impossible), in- (intolerant) |
| **词缀（Affix）** | 添加到词干的词素 | - |
| **前缀（Prefix）** | 位于词首的词缀 | un- (unhappy) |
| **后缀（Suffix）** | 位于词尾的词缀 | -s (cats) |

### 1.4 其他词缀类型

英语只有前缀和后缀，但其他语言有更多类型：

| 类型 | 定义 | 例子 |
|------|------|------|
| **中缀（Infix）** | 插入词干内部的词缀 | Bontoc语：fikas → fumikas（-um-） |
| ** circumfix** | 前后两部分组成的词缀 | Chickasaw语：chokma → ikchokmo（ik--o） |
| **转缀（Transfix）** | 不连续词缀，常见于闪米特语 | 马耳他语：k-t-b → kiteb（-i-e-） |
| **缺缀（Disfix）** | 词素的一部分被删除 | 阿拉巴马语：tipsali → tipli |

## 2. 形态分析基本原则

### 2.1 原则一

> **如果一个语音形式有两个不同的意义或功能，它必须被分析为两个不同的词素。**

**示例**：英语 -s
- cats 中的 -s：名词复数标记
- sees 中的 -s：第三人称单数现在时标记

这两个 -s 功能不同，应视为两个不同的词素。

### 2.2 原则二

> **如果同一功能（或意义）与两个或多个语音形式相关联，这些不同形式代表同一个词素，选择通常可基于音系或其他考虑预测。**

**示例**：否定前缀 il-, im-, in-, ir-
- illegal, impossible, intolerant, irresponsible
- 都是同一个词素的语素变体
- 选择规则：若词干以流音开头，前缀为 iX-（X为词干首辅音）；若词干以塞音开头，前缀为 iN-（N为鼻音）

## 3. 名词的变量

### 3.1 变量类型

| 变量 | 描述 | 常见值 |
|------|------|--------|
| **数（Number）** | 名词所指事物的数量 | 单数、双数、复数、三数、少量数（paucal） |
| **性（Gender）** | 名词类别的语法化 | 阳性、阴性、中性等 |
| **格（Case）** | 名词在句中的语法角色 | 主格、宾格、与格、属格等 |

### 3.2 格（Case）

格指示名词在句中的角色。乌拉尔语系语言可有20多种格：

| 格类型 | 功能 | 英语对应 |
|--------|------|----------|
| 工具格（Instrumental） | 表示动作所用的工具 | with, by |
| 向格（Allative） | 表示方向（+移动） | on + M |
| 位格（Adessive） | 表示位置（-移动） | on - M |
| 内格（Illative） | 进入 | in + M |
| 出格（Elative） | 从...出来 | from |

### 3.3 Dagaare语的固有数

Dagaare语具有**固有复数性**：
- 通常单独存在的名词（"额头"、"帽子"）：无标记形式为单数
- 通常成对或成群存在的名词（"腿"、"蜜蜂"）：无标记形式为复数

## 4. 分类词（Classifiers）

分类词是伴随名词、对名词进行分类的词或词缀。

### 4.1 汉语示例

| 短语 | 分析 |
|------|------|
| 三只狗 | 三 + 只（小动物分类词）+ 狗 |
| 三只猫 | 三 + 只 + 猫 |
| 五只猫 | 五 + 只 + 猫 |

分类词通常基于语义考虑，如形状（长形物体、扁平物体、圆形物体等）。

### 4.2 日语分类词

| 分类词 | 适用对象 | 例子 |
|--------|----------|------|
| -nin | 人类 | isha-kyūnin（9医生） |
| -satsu | 书籍、杂志 | hon-yonsatsu（4本书） |
| -mai | 扁平物体 | kami-hachimai（8张纸） |
| -hiki | 小动物 | inu-kyūhiki（9只狗） |
| -tō | 大动物 | uma-rokutō（6匹马） |

**短语结构**：Noun + Number-Classifier

## 5. 重叠（Reduplication）

重叠是通过部分或全部重复词素或词的一部分来形成新词的过程。

### 5.1 重叠类型

| 类型 | 描述 | 例子 |
|------|------|------|
| **完全重叠** | 重复整个词 | 印尼语：rumah → rumahrumah（房子→房子们） |
| **部分重叠** | 重复词的一部分 | 见下文 |

### 5.2 部分重叠示例

**Marshallese语**：重复最后一个音节
- kagir → kagirgir（系腰带）
- takin → takinkin（穿袜子）

**Samoan语**：重复倒数第二个音节
- savali → savavali（他走→他们走）
- alofa → alolofa（他爱→他们爱）

## 6. 异根（Suppletion）

异根指同一个词素（具有语义成分）有两个或多个完全不同的形式。

### 6.1 常见例子

| 语言 | 例子 |
|------|------|
| 英语 | good – better, bad – worse, go – went |
| 法语 | être – sommes, aller – vais |
| 西班牙语 | ir – voy |
| 德语 | gut – besser |

> **注意**：异根不同于词干的元音变化（如 goose – geese），而是两个形式完全不同。

## 7. 表示所有关系

### 7.1 所有关系的类型

| 类型 | 定义 | 典型例子 |
|------|------|----------|
| **不可让渡所有（Inalienable）** | 不能借出/取走的所有物 | 身体部位、家庭成员 |
| **可让渡所有（Alienable）** | 可以转让的所有物 | 其他物品 |

### 7.2 斐济语（Fijian）示例

**不可让渡所有结构**：
```
na Noun-Poss
```
例子：na ulu-qu（我的头）

**可让渡所有结构**：
```
na C-Poss Noun
```
其中 C 为分类词。

**所有格后缀**：

| 人称 | 形式 |
|------|------|
| 1sg | -qu |
| 2sg | -mu |
| 3sg | -na |
| 1pl.incl | -da |
| 2du | -mudrau |
| 2pl | -munī |
| 3pl | -dra |

**所有分类词**：

| 分类词 | 用途 |
|--------|------|
| ke- | 食物；影响我们的非所有物 |
| me- | 饮料 |
| no- | 其他（拥有的物品） |

### 7.3 古希腊语示例

**短语结构**：
```
[Art.possessed] [Art.possessor] Possessor Possessed
```

**冠词形式**：

| | 单数 | 复数 |
|--|------|------|
| 所有人 | tu | tōn |
| 被所有物 | ho | hoi |

**名词词尾**：

| | 单数 | 复数 |
|--|------|------|
| 所有人 | -u | -ōn |
| 被所有物 | -os | -oi |

## 8. 颜色术语

### 8.1 Berlin & Kay 假说

Berlin & Kay（1969）研究了100多种语言的颜色术语，提出颜色感知受同一规律支配。

### 8.2 颜色术语演变阶段

使用符号：W=白，Bk=黑，Br=棕，R=红，G=绿，Y=黄，Bl=蓝

下划线表示该语言有独立词汇的颜色，后面跟随的是被视为相同颜色的变体。

| 阶段 | 颜色术语系统 |
|------|-------------|
| Stage I | W: Y, Br, R / Bk: Bl, G |
| Stage II | W / R: Y, Br / Bk: Bl, G |
| Stage IIIa | W / R: Br / Y / Bk: Bl, G |
| Stage IIIb | W / R: Y, Br / Bk / G: Bl |
| Stage IV | W / R: Br / Y / Bk / G: Bl |
| Stage V | W / R: Br / Y / Bk / G / Bl |
| Stage VI | W / R / Br / Y / Bk / G / Bl |

### 8.3 应用

- 语言的颜色术语数量决定其颜色感知方式
- 基本颜色词通常有完整的屈折变化
- 借用的颜色词可能屈折变化减少或无变化

**罗马尼亚语示例**：
- 基本颜色词有完整屈折：alb – albă – albi – albe（白）
- 借用颜色词无变化：roz（粉红）、maro（棕色）

## 9. 解题方法论

### 9.1 名词短语问题

1. 确定短语结构
2. 分离不同语素
3. 分析语素变体的分布规律
4. 找出支配变化的规则

### 9.2 常见分析技巧

| 技巧 | 描述 |
|------|------|
| 频率表 | 统计词干出现频率，匹配翻译 |
| 对比分析 | 比较包含相同成分的例子 |
| 假设验证 | 提出假设并用剩余数据验证 |
| 排除法 | 逐步排除不可能的对应关系 |

### 9.3 注意事项

- 缺缀（Disfix）极为罕见，需确保无其他解释可能
- 异根也很罕见，需排除语音变化的可能性
- 在确定词类分类时，可基于形态标记而非语义
