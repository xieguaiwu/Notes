# AP Physics 1 易错点总结

> 本文档整理了 AP Physics 1 各主题中的常见易错点和误区，帮助避免考试中的典型错误。

---

## 一、运动学易错点

### 1. 平衡力 vs 相互作用力
- **易错**：混淆两者概念
- **正确**：
  - 平衡力：作用在**同一物体**上，可抵消，合力为零
  - 相互作用力：作用在**不同物体**上，不可抵消，同时产生同时消失

### 2. 抛体运动最高点
- **易错**：认为最高点动能全部转化为势能
- **正确**：水平速度仍存在
  - 初始动能 = 最高点动能（水平）+ 势能
  - 最高点仍有水平方向的动能

### 3. 质量与下落加速度
- **易错**：认为质量大的物体下落更快
- **正确**：忽略空气阻力时，所有物体自由落体加速度相同（$g$），与质量无关

### 4. 渡河问题
- **易错**：混淆"最短时间"与"到达正对岸"
- **正确**：
  - 最短时间：船头垂直对岸，$t = \frac{d}{v_{船}}$
  - 到达正对岸：需抵消水流速度，船头偏向上游

### 5. 斜面下滑时间
- **易错**：认为高度相同则下落时间相同
- **正确**：斜面下滑时间 > 自由落体时间
  - 原因：路程更长，加速度更小（$a = g\sin\theta < g$）

### 6. v-t图转a-t图
- **易错**：忽略曲线斜率变化
- **正确**：$a = \frac{dv}{dt}$ = 切线斜率，需分析斜率变化趋势
  - 曲线变平 → 斜率减小 → 加速度减小
  - 曲线变陡 → 斜率增大 → 加速度增大

## 二、动力学与摩擦易错点

### 7. 静摩擦力 vs 动摩擦力
- **易错**：混淆两种摩擦力
- **正确**：
  - 静止或共同运动（无相对滑动）→ 静摩擦力
  - 有相对滑动 → 动摩擦力
  - 一般情况下 $\mu_s > \mu_k$

### 8. 静摩擦力存在条件
- **易错**：认为静止物体一定受静摩擦力
- **正确**：有相对运动趋势时才有静摩擦力
  - 共同匀速运动时摩擦力可能为零
  - 静止但无运动趋势时无摩擦力

### 9. 斜面支持力
- **易错**：支持力等于重力
- **正确**：$F_N = mg\cos\theta$（不是 $mg$）
  - 倾角越大，支持力越小

### 10. 有角度外力时的支持力
- **易错**：支持力仍等于 $mg$
- **正确**：
  - 斜向上拉时：$F_N = mg - F\sin\theta$
  - 斜向下压时：$F_N = mg + F\sin\theta$

### 11. 非接触力判断
- **易错**：认为不接触的物体之间有弹力/摩擦力
- **正确**：只有**直接接触**的物体间才有弹力和摩擦力
  - 力的传递需要通过中间物体

### 12. 摩擦力方向
- **易错**：摩擦力总是与运动方向相反
- **正确**：
  - 动摩擦力：与**相对运动**方向相反
  - 静摩擦力：与**相对运动趋势**方向相反
  - 摩擦力可以与运动方向相同（如走路时的静摩擦力）

## 三、圆周运动与引力易错点

### 13. 向心力
- **易错**：认为向心力是独立存在的力
- **正确**：向心力是**效果力**，由其他力（重力、弹力、摩擦力等）的合力提供
  - 受力分析时不要额外画"向心力"
  - 公式：$F_c = m\frac{v^2}{r} = m\omega^2 r$

### 14. 离心力
- **易错**：认为存在离心力
- **正确**：惯性系中**不存在离心力**
  - 离心现象是惯性的表现
  - 非惯性系中可引入惯性离心力（AP Physics 1 不涉及）

### 15. 线速度与角速度
- **易错**：匀速圆周运动中线速度恒定
- **正确**：
  - 速率恒定，但**方向时刻变化**
  - 线速度是矢量，所以是变化的
  - 角速度大小和方向都不变

### 16. 圆锥摆角速度
- **易错**：认为角速度与半径有关
- **正确**：$\omega = \sqrt{g/h}$，**只与悬挂高度 $h$ 有关**
  - 与半径、质量无关
  - 周期：$T = 2\pi\sqrt{h/g}$

### 17. 失重 ≠ 重力消失
- **易错**：认为失重时重力消失
- **正确**：重力依然存在，只是**视重（支持力）减小**
  - 完全失重：$a = g$，支持力为零
  - 重力 $mg$ 始终存在

### 18. 超重/失重判断
- **易错**：向上运动就超重，向下运动就失重
- **正确**：取决于**加速度方向**，不是速度方向
  - 加速度向上 → 超重（视重 $= m(g+a)$）
  - 加速度向下 → 失重（视重 $= m(g-a)$）

### 19. 三体引力势能
- **易错**：只计算部分势能
- **正确**：$n$ 个物体有 $\frac{n(n-1)}{2}$ 对相互作用势能
  - 三体系统：3 对势能
  - 每对只计算一次

## 四、能量与动量易错点

### 20. 动量守恒 vs 机械能守恒
- **易错**：混淆两种守恒条件
- **正确**：
  - 动量守恒：合外力为零（或内力远大于外力）
  - 机械能守恒：只有保守力做功
  - 碰撞时动量守恒，动能不一定守恒

### 21. 摩擦力与系统动量
- **易错**：认为摩擦力影响系统总动量
- **正确**：系统内部物体间的摩擦力是**内力**，不影响系统总动量
  - 只有外力才影响系统总动量

### 22. 弹簧下垂
- **易错**：只考虑弹性势能
- **正确**：**弹性势能和重力势能同时变化**
  - 弹簧伸长时：弹性势能增加，重力势能减少
  - 需同时考虑两种势能的变化

### 23. 功率的正负
- **易错**：忽略功率符号
- **正确**：
  - 摩擦力功率为负（做负功）
  - $P = \vec{F} \cdot \vec{v}$，注意力的方向

### 24. 碰撞类型判断
- **易错**：混淆弹性/非弹性碰撞
- **正确**：
  - 弹性碰撞：动量守恒 + 动能守恒（$e=1$）
  - 非弹性碰撞：动量守恒 + 动能损失（$0<e<1$）
  - 完全非弹性：动量守恒 + 碰后共速（$e=0$，动能损失最大）

### 25. 等质量弹性碰撞
- **易错**：不知道特殊结论
- **正确**：等质量弹性碰撞，**速度交换**
  - $m_1 = m_2$ 时：$v_1' = v_2$，$v_2' = v_1$
  - 例：台球碰撞

### 26. 二维碰撞
- **易错**：只考虑一个方向
- **正确**：在**两个相互垂直方向**分别应用动量守恒
  - $x$ 方向：$m_1 v_{1x} + m_2 v_{2x} = m_1 v_{1x}' + m_2 v_{2x}'$
  - $y$ 方向：$m_1 v_{1y} + m_2 v_{2y} = m_1 v_{1y}' + m_2 v_{2y}'$

## 五、转动易错点

### 27. 转动惯量
- **易错**：认为只与质量有关
- **正确**：与**质量分布**和**旋转轴位置**有关
  - $I = \sum m_i r_i^2$
  - 质量离轴越远，转动惯量越大

### 28. 旋转轴位置
- **易错**：忽略旋转轴对转动惯量的影响
- **正确**：同一物体绕不同轴的转动惯量完全不同
  - 平行轴定理：$I = I_{cm} + Md^2$

### 29. 纯滚动摩擦力
- **易错**：认为纯滚动时有动摩擦力
- **正确**：纯滚动时是**静摩擦力**，不做功
  - 接触点瞬时速度为零
  - 机械能守恒

### 30. 纯滚动速度关系
- **易错**：忽略转动动能
- **正确**：
  - $v_{cm} = \omega R$（纯滚动条件）
  - 总动能 $= \frac{1}{2}mv_{cm}^2 + \frac{1}{2}I\omega^2$
  - 不能只用平动动能

### 31. 刚体各点速度
- **易错**：认为刚体各点速度相同
- **正确**：只有**质心速度**代表整体平动速度
  - 接触点瞬时速度为零
  - 顶点速度为 $2v_{cm}$

## 六、振动易错点

### 32. 单摆周期与质量
- **易错**：认为周期与摆球质量有关
- **正确**：$T = 2\pi\sqrt{L/g}$，**与质量无关**
  - 也与振幅无关（小角度范围内）

### 33. 单摆小角度条件
- **易错**：忽略小角度近似条件
- **正确**：$\theta < 5°$ 时才近似为简谐运动
  - $\sin\theta \approx \theta$
  - 大角度时周期会增大

### 34. 简谐运动速度最大位置
- **易错**：认为最大位移处速度最大
- **正确**：**平衡位置**（$x=0$）速度最大
  - 最大位移处速度为零
  - $v_{max} = A\omega$

### 35. 简谐运动加速度方向
- **易错**：认为加速度方向与速度方向相同
- **正确**：加速度方向**始终指向平衡位置**
  - $a = -\omega^2 x$（与位移方向相反）
  - 平衡位置加速度为零

### 36. 简谐运动能量转换
- **易错**：混淆动能、势能最大位置
- **正确**：
  - 平衡位置：动能最大，势能为零
  - 最大位移：势能最大，动能为零
  - 总机械能守恒：$E = \frac{1}{2}kA^2$

## 七、弹簧与弹性易错点

### 37. 瞬时静止 ≠ 平衡
- **易错**：认为瞬时静止时力平衡
- **正确**：瞬时静止时可能仍有加速度
  - 如弹簧压缩到最低点：速度为零，加速度向上
  - 需分析受力情况，不能仅凭静止判断

### 38. 弹簧串联/并联
- **易错**：混淆有效劲度系数
- **正确**：
  - 串联：$\frac{1}{k_{eq}} = \frac{1}{k_1} + \frac{1}{k_2}$，$k_{eq} < k$
  - 并联：$k_{eq} = k_1 + k_2$，$k_{eq} > k$

## 八、实验与图像易错点

### 39. a-F图像截距
- **易错**：忽略截距的物理意义
- **正确**：不过原点说明存在**恒定阻力**
  - 负截距：存在与施加力方向相反的阻力（如摩擦力）
  - 正截距：存在与施加力方向相同的助力

### 40. 图像斜率变化
- **易错**：只看趋势，忽略斜率变化
- **正确**：
  - v-t图曲线的斜率 = 加速度
  - 需分析斜率的变化趋势
  - 曲线弯曲方向反映加速度的变化

## 解题检查清单

在做题时，建议按以下顺序检查：

1. **研究对象**：明确是对哪个物体/系统进行分析
2. **参考系**：确定正方向
3. **受力分析**：画出完整的受力图
4. **运动状态**：判断静止、匀速还是加速
5. **守恒条件**：检查动量/能量是否守恒
6. **公式选择**：选择正确的物理公式
7. **单位统一**：检查单位是否一致
8. **结果验证**：检查结果是否合理
