# Pendulum Problems
[[Conical Pendulum]]

## 1. 
>质量为 $m$ 的物体在水平圆周上运动，半径为 $r$，周期为 $T_1$ 将质量改为 $2m$，同样的半径 $r$，周期为 $T_2$ ，求$T_2$ 与 $T_1$ 的关系？

![[Enterprise_pendulum.png]]

圆锥摆的周期公式为：$$T = 2\pi \sqrt{\frac{L \cos\theta}{g}}$$
**周期与质量无关** — 这意味着改变质量不会改变摆动周期。因此 $T_2 = T_1$
