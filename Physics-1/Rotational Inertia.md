# Rotational Inertia (转动惯量) - AP Physics 1

## 1. 基本定义

**转动惯量 (Rotational Inertia / Moment of Inertia)** 是物体对旋转运动的惯性的量度。它取决于物体质量分布相对于旋转轴的位置。

### 核心公式
- **离散物体**：$I = \sum m_i r_i^2$
  - $m_i$：第$i$个质点的质量
  - $r_i$：第$i$个质点到旋转轴的距离

- **连续物体**：$I = \int r^2 dm$

---

## 2. 常见形状的转动惯量公式

| 形状 | 旋转轴 | 转动惯量公式 |
|------|--------|--------------|
| **薄圆环** | 通过中心垂直于平面 | $I = MR^2$ |
| **实心圆盘/圆柱** | 通过中心垂直于平面 | $I = \frac{1}{2}MR^2$ |
| **实心圆柱** | 沿中心轴（纵向） | $I = \frac{1}{2}MR^2$ |
| **实心球** | 通过中心 | $I = \frac{2}{5}MR^2$ |
| **薄球壳** | 通过中心 | $I = \frac{2}{3}MR^2$ |
| **细长杆** | 通过中心垂直于杆 | $I = \frac{1}{12}ML^2$ |
| **细长杆** | 通过一端垂直于杆 | $I = \frac{1}{3}ML^2$ |
| **矩形薄板** | 通过中心垂直于板面 | $I = \frac{1}{12}M(L^2 + W^2)$ |

其中：$M$ 为总质量，$R$ 为半径，$L$ 为长度，$W$ 为宽度

---

## 3. 重要定理

### 平行轴定理 (Parallel Axis Theorem)
当旋转轴从质心轴平移距离 $d$ 时：

$$I = I_{cm} + Md^2$$

- $I_{cm}$：通过质心的转动惯量
- $M$：物体总质量
- $d$：两平行轴之间的距离

**应用场景**：计算绕非质心轴旋转时的转动惯量

### 垂直轴定理 (Perpendicular Axis Theorem)
适用于平面薄板（二维物体），若物体在xy平面内：

$$I_z = I_x + I_y$$

- $I_z$：绕z轴（垂直于平面）的转动惯量
- $I_x, I_y$：绕x轴和y轴（在平面内）的转动惯量

---

## 4. 转动惯量的物理意义

### 与平动的类比

| 平动量 | 转动量 |
|--------|--------|
| 质量 $m$ | 转动惯量 $I$ |
| 力 $F$ | 力矩 $\tau$ |
| 加速度 $a$ | 角加速度 $\alpha$ |
| 牛顿第二定律 $F = ma$ | 转动牛顿第二定律 $\tau = I\alpha$ |

### 关系式

- **力矩与角加速度**：$\tau = I\alpha$
- **转动动能**：$K_{rot} = \frac{1}{2}I\omega^2$
- **角动量**：$L = I\omega$

---

## 5. 影响转动惯量的因素

1. **质量大小**：质量越大，转动惯量越大
2. **质量分布**：质量离旋转轴越远，转动惯量越大
3. **旋转轴位置**：旋转轴不同，转动惯量不同

> **重要概念**：对于相同质量的物体，如果质量分布离旋转轴更远，其转动惯量更大（例如：薄圆环的 $I$ 比实心圆盘的 $I$ 大）

---

## 6. 典型应用场景

### 1. 滚动运动
- **纯滚动**：物体无滑动地滚动
- **平动动能 + 转动动能**：$K = \frac{1}{2}mv^2 + \frac{1}{2}I\omega^2$
- 纯滚动时：$v = \omega R$

### 2. 刚体旋转
- 使用 $\tau = I\alpha$ 分析旋转动力学
- 类比牛顿第二定律应用于旋转系统

### 3. 滚动物体竞赛
- 不同形状的物体从斜坡滚下时
- $a = \frac{g\sin\theta}{1 + I/(MR^2)}$
- $I/(MR^2)$ 称为"形状因子"

**常见形状因子**：
- 实心圆柱/圆盘：$\frac{1}{2}$
- 实心球：$\frac{2}{5}$
- 薄圆环：$1$

**结论**：形状因子越小，滚下斜坡越快（实心球最快，薄圆环最慢）

## 7. AP Physics 1 考试重点

1. **理解转动惯量的物理意义**：与质量的类比
2. **记忆常见形状的公式**：薄圆环、实心圆盘、实心球、杆
3. **应用平行轴定理**：计算非质心轴旋转的 $I$
4. **$\tau = I\alpha$ 的应用**：旋转动力学的核心公式
5. **滚动问题的能量分析**：$K = \frac{1}{2}mv^2 + \frac{1}{2}I\omega^2$
6. **斜坡滚下比较**：不同形状物体的加速特性
7. **系统转动惯量的叠加**：$I_{total} = I_1 + I_2 + ...$

---

## 8. 典型例题类型

### 类型1：基础计算
给定物体形状、质量和尺寸，计算绕指定轴的转动惯量

### 类型2：平行轴定理应用
已知绕质心轴的转动惯量，求绕平行轴的转动惯量

### 类型3：力矩与角加速度
使用 $\tau = I\alpha$ 求解旋转问题

### 类型4：滚动能量问题
分析物体滚下斜坡时的能量转化，计算末速度或加速度

### 类型5：角动量守恒
涉及 $L = I\omega$ 的守恒问题（当 $\tau = 0$ 时）

---

## 9. 常见陷阱提醒

- ⚠️ 确认旋转轴的位置！同一物体绕不同轴的转动惯量完全不同
- ⚠️ 记忆公式时要明确是绕哪个轴旋转
- ⚠️ 滚动问题中，必须考虑转动动能，不能只用平动动能
- ⚠️ 平行轴定理只能用于从一个质心轴到平行平移轴
- ⚠️ 垂直轴定理只适用于平面薄板

---

## 10. 关键概念总结

| 概念 | 公式 | 物理意义 |
|------|------|----------|
| 转动惯量 | $I = \sum mr^2$ | 旋转运动的惯性 |
| 平行轴定理 | $I = I_{cm} + Md^2$ | 轴平移时的转动惯量 |
| 力矩方程 | $\tau = I\alpha$ | 转动动力学核心 |
| 转动动能 | $K = \frac{1}{2}I\omega^2$ | 旋转运动的能量 |
| 角动量 | $L = I\omega$ | 旋转运动的动量 |
| 滚动总动能 | $K = \frac{1}{2}mv^2 + \frac{1}{2}I\omega^2$ | 滚动物体的总能量 |

---
