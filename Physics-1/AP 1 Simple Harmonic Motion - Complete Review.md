## 一、简谐运动的定义与特征

### 知识点

**动力学定义：** 物体所受恢复力与位移大小成正比，方向相反。即：
$$F = -kx$$

其中 $k$ 为劲度系数或等效劲度系数。

**运动学特征：** 位移 $x$ 随时间按正弦或余弦规律变化。

**平衡位置：** 合力为零的位置，也是振动中心。

**振幅 $A$：** 离开平衡位置的**最大**距离。

**周期 $T$：** 完成一次全振动所需时间，单位：秒。

**频率 $f$：** 单位时间内完成全振动的次数：
$$f = \frac{1}{T}$$

单位：Hz。

**角频率 $\omega$：**
$$\omega = 2\pi f = \frac{2\pi}{T}$$

单位：rad/s。

## 二、弹簧振子

### 知识点

#### 水平弹簧振子

恢复力由弹簧提供：$F = -kx$

**周期：**
$$T = 2\pi\sqrt{\frac{m}{k}}$$

**角频率：**
$$\omega = \sqrt{\frac{k}{m}}$$

周期与振幅**无关**，与质量、劲度系数有关。

#### 竖直悬挂弹簧

平衡位置处：$mg = k\Delta l$（$\Delta l$ 为静伸长）

**振动周期：**
$$T = 2\pi\sqrt{\frac{m}{k}}$$

与水平情况相同！

**注意：** 重力只改变平衡位置，不影响周期和角频率。

## 三、单摆

### 知识点

**小角近似条件：** 摆角 $\theta \lesssim 10^\circ$，此时 $\sin\theta \approx \theta$（弧度制）。

**恢复力：** 重力切向分量
$$F_\theta = -mg\sin\theta \approx -mg\theta$$

**周期：**
$$T = 2\pi\sqrt{\frac{L}{g}}$$

**角频率：**
$$\omega = \sqrt{\frac{g}{L}}$$

**注意：**
- 周期与摆球质量**无关**
- 周期与振幅**无关**（在小角下）
- 只与摆长 $L$ 和重力加速度 $g$ 有关
- 摆长 $L$ 为悬点到摆球质心的距离

**重要：** 若单摆处于加速电梯中，则 $g$ 替换为等效重力加速度 $g_{eff}$。

## 四、SHM 中的能量

### 知识点

**总机械能：**
$$E_{total} = \frac{1}{2}kA^2$$

与振幅平方成正比。

**动能：**
$$K = \frac{1}{2}mv^2$$

**弹性势能：**
$$U = \frac{1}{2}kx^2$$

**动能与势能相互转换：**
- 当 $x=0$ 时，$K_{max} = \frac{1}{2}kA^2$，$U=0$
- 当 $x=\pm A$ 时，$K=0$，$U_{max}=\frac{1}{2}kA^2$

**当 $x = \pm \frac{A}{\sqrt{2}}$ 时：**
$$K = U = \frac{1}{4}kA^2$$

**能量法求速度：**
$$v = \pm \sqrt{\frac{k}{m}(A^2 - x^2)}$$

与运动学一致。

## 五、AP1 常考模型与题型

### 模型1：竖直弹簧振子的平衡位置与周期

将物体挂在弹簧下端，静止时弹簧伸长 $\Delta l = \frac{mg}{k}$。给物体一个扰动后，它将围绕该平衡位置做 SHM，周期 $T=2\pi\sqrt{m/k}$。

**解题时注意：** 坐标原点取在平衡位置，则回复力仍为 $-kx$。

### 模型2：弹簧串联与并联

**串联：** 等效劲度系数
$$k_{eq} = \frac{1}{\frac{1}{k_1} + \frac{1}{k_2}} = \frac{k_1k_2}{k_1+k_2}$$

总伸长量等于各弹簧伸长量之和。

**并联：** 等效劲度系数
$$k_{eq} = k_1 + k_2$$

两弹簧形变量相同。

**周期变化：**
- 串联后系统周期 $T = 2\pi\sqrt{\frac{m}{k_{eq}}}$，周期变**大**
- 并联后周期变**小**

### 模型3：子弹射入木块后与弹簧结合

子弹射入木块过程（完全非弹性碰撞）动量守恒，但机械能不守恒。碰后整体压缩弹簧做 SHM。

压缩弹簧的最大距离由碰后动能全部转化为弹性势能求得：
$$\frac{1}{2}(m+M)V^2 = \frac{1}{2}kA^2$$

其中 $V$ 为碰后共同速度。

**周期：**
$$T = 2\pi\sqrt{\frac{m+M}{k}}$$

### 模型4：两个弹簧连接一个物体（不同连接方式）

**两端固定弹簧：** 物体在中间，两侧弹簧都产生恢复力，等效 $k = k_1 + k_2$。

**一侧弹簧，另一侧轻绳：** 类似单弹簧。

**水平光滑面上，物体连接两个弹簧，另一端分别固定在墙上：** 等效 $k = k_1 + k_2$。

### 模型5：单摆与重力加速度的测量

由 $T = 2\pi\sqrt{L/g}$ 可得：
$$g = \frac{4\pi^2 L}{T^2}$$

实验常用改变摆长测多组数据，作 $L-T^2$ 图，斜率可求 $g$。

### 模型6：物理摆（AP1 可能涉及定性）

对于刚体绕固定轴摆动，周期：
$$T = 2\pi\sqrt{\frac{I}{mgd}}$$

其中 $d$ 为转轴到质心距离，$I$ 为转动惯量。

当刚体为细杆绕一端时，$I = \frac{1}{3}ML^2$，$d = L/2$，则：
$$T = 2\pi\sqrt{\frac{2L}{3g}}$$

## 六、位移-时间图像与相位

### 知识点

**振动方程：**
$$x(t) = A\cos(\omega t + \phi)$$

或 $A\sin(\omega t + \phi')$。

**相位 $\phi$：** 决定初始时刻物体的位置和运动方向。

**常见初始条件对应相位：**
- $t=0$ 时在最大位移处（$x_0=+A$，$v_0=0$）：$\phi = 0$（余弦形式）
- $t=0$ 时在平衡位置且向正方向运动：$\phi = -\frac{\pi}{2}$（或 $\frac{3\pi}{2}$）
- $t=0$ 时在平衡位置且向负方向运动：$\phi = \frac{\pi}{2}$

**速度-时间图像：** 速度相位比位移超前 $\frac{\pi}{2}$：
$$v(t) = -A\omega\sin(\omega t+\phi)$$

**加速度-时间图像：** 加速度相位比位移超前 $\pi$（反向）：
$$a(t) = -A\omega^2\cos(\omega t+\phi)$$

## ⚠️ 易错点

### AP1 简谐振动易错点

1. **竖直弹簧振子的平衡位置在弹簧被压缩或拉伸后的静止位置，不要与原长混淆。**

2. **单摆周期公式中的 $L$ 是摆长，不是摆线长度（摆球半径不可忽略时需加上半径）。**

3. **弹簧串联时，等效 $k$ 变小，周期变大；并联时等效 $k$ 变大，周期变小。**

4. **能量问题中，弹性势能公式 $\frac{1}{2}kx^2$ 中的 $x$ 是相对于平衡位置的形变量，而不是弹簧原长位置的形变量（竖直弹簧尤其注意）。**

5. **振幅 $A$ 是标量，表示最大位移的大小，不包含方向。**
