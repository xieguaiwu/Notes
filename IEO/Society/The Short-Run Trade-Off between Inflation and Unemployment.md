# The Short-Run Trade-Off between Inflation and Unemployment
## 1. The Phillips Curve
正如十大基本原理之一的[[Inflation -> Unemployment]]所说，通胀率和失业率间面临着交替关系（见[[Trade-offs]]），这意味着高通胀率会引起暂时的高失业率和低产量，反之则会引起暂时的低失业率和高产量。Phillips Curve所体现的正是这种关系：
![[phillips_curve.png]]
这种现象是因为总需求曲线的移动令经济**沿着短期总供给曲线变动**，如下图所示：
![[Aggregate_demand_in_short_run_with_unemployment.png]]

## 2. The Phillips Curve in the Long Run
长期中的失业率水平即natural rate of unemployment，见[[Unemployment]]的**3**，有如下公式：
$$\text{Natural Unemployment}=\text{Frictional Unemployment}+\text{Structural Unemployment}$$
**在长期中，失业并不取决于货币量或通货膨胀率**
![[phillips_curve_in_long_run.png]]

## 3. The Phillips Curve in the Short Run
弗里德曼和Phillips对于短期失业率的分析由如下公式给出：
$$\text{失业率}=\text{自然失业率}-a(\text{实际通货膨胀率}-\text{预期通货膨胀率})$$
这实际上只是[[Aggregate Demand & Aggregate Supply]]的**4**中的产量公式的改写，其中$a$也是一个衡量面对实际通货膨胀率和预期通货膨胀率差别时反应大小的常数。

如果预期通货膨胀率不变，将长期和短期的Phillips Curve放在一起则会有下图：

它所表示的正是实际通货膨胀率会对短期的Phillips Curve造成的影响
