# Synthesis & Reasoning
## 1. Def - Synthesis
Synthesis需要：
- Consideration: 考虑他人的言论或观点能够用于支持自己的观点
- Explanation: 向读者解释清楚他人的这些观点是如何**符合自己的观点**的
- Integration: 流畅地嵌入其他信息源

## 2. Def - Line of Reasoning
有以下几种line of reasoning：
- Deductive Reasoning: 从**普遍原则**推导出**具体个案**的必然结论
- Inductive Reasoning: 归纳推理，通过许多例子来推理出一个普遍命题
- Causal Reasoning: 根据事件间的有规律联系，推断**原因**与**结果**的关系
- Analogic or Comparative Reasoning: 类比/比较论证

## 3. Flaws of Reasoning
有以下几种flaw of reasoning:
- Inductive Flaws: 缺乏说服力的归纳推理，即案例远不足以说明观点
- Causal Flaws: 很多时候，时间上的先后性并不代表相关性乃至因果性
- Comparative Flaws: 将具有本质不同的两个事物进行比较论证

## 4. Methods of Development
### 4.1
为了帮助读者理解其推理链条，有以下几种methods of development：

| Methods                 | Purpose         |
| ----------------------- | --------------- |
| **Narration**           | 用叙事形式介绍一系列信息    |
| **Cause-Effect**        | 在题目下介绍事件的效应     |
| **Comparison-Contrast** | 说明主题中事物彼此间的异同   |
| **Definition**          | 使用同义词、本质定义或外延定义 |
| **Description**         | 使用视觉化的描述        |

Narration: 适合**anecdotes**以及**personal observations**
