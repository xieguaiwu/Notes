## 一、转动运动学

### 知识点

**角位移 $\theta$：** 单位 **弧度（rad）**

**角速度 $\omega$：**
- 平均：$\omega = \frac{\Delta \theta}{\Delta t}$
- 瞬时：$\omega = \frac{d\theta}{dt}$
- 单位：rad/s

**角加速度 $\alpha$：**
- 平均：$\alpha = \frac{\Delta \omega}{\Delta t}$
- 瞬时：$\alpha = \frac{d\omega}{dt}$
- 单位：rad/s²

**匀角加速转动公式**（与平动一一对应）：
$$\omega_f = \omega_i + \alpha t$$
$$\Delta \theta = \omega_i t + \frac{1}{2}\alpha t^2$$
$$\omega_f^2 = \omega_i^2 + 2\alpha \Delta \theta$$

**线量与角量的关系**（圆弧运动）：
- 弧长：$s = r\theta$
- 切向速度：$v_t = r\omega$
- 切向加速度：$a_t = r\alpha$
- 向心加速度：$a_c = \frac{v^2}{r} = \omega^2 r$

**注意：** 所有角量（$\theta, \omega, \alpha$）通常取逆时针方向为正。

## 二、力矩与转动平衡

### 知识点

**力矩定义：**
$$\tau = r F \sin\phi = r F_\perp$$

单位：N·m。

$\phi$ 为从 $r$ 转到 $F$ 的角度，$r$ 为转轴到力作用点的距离（力臂）。

**方向：** 逆时针力矩为正，顺时针为负（AP1 通常用正负表示方向）。

**转动平衡条件：** 物体静止或匀速转动时，合外力矩为零：
$$\sum \tau = 0$$

**注意：** 力矩与功的单位表面相同，但物理意义不同：焦耳是能量，牛·米是力矩。

**力臂：** 从转轴到力的作用线的垂直距离。求力矩时，可分解力，也可分解力臂。

## 三、转动惯量与转动定律

### 知识点

**转动惯量 $I$：** 描述物体转动惯性的物理量，取决于**质量分布**和**转轴位置**。

**常见形状的转动惯量（AP1 通常直接给出）：**
- 细圆环绕中心轴：$I = MR^2$
- 实心圆柱/圆盘绕中心轴：$I = \frac{1}{2}MR^2$
- 细杆绕中心：$I = \frac{1}{12}ML^2$，绕一端：$I = \frac{1}{3}ML^2$
- 实心球绕中心：$I = \frac{2}{5}MR^2$

**牛顿第二定律的转动形式：**
$$\sum \tau = I \alpha$$

这是转动版的 $F=ma$，其中 $\alpha$ 是角加速度。

**平行轴定理：**
$$I = I_{cm} + Md^2$$

其中 $d$ 为平行轴到质心轴的距离。

## 四、转动动能与功

### 知识点

**转动动能：**
$$K_{rot} = \frac{1}{2} I \omega^2$$

**力矩做功：**
$$W = \tau \Delta \theta$$

（恒力矩）或 $W = \int \tau d\theta$

**转动功率：**
$$P = \tau \omega$$

**动能定理的转动形式：**
$$W_{net} = \Delta K_{rot} = \frac{1}{2}I\omega_f^2 - \frac{1}{2}I\omega_i^2$$

对于既有平动又有转动的物体，总动能：
$$K_{total} = \frac{1}{2}mv_{cm}^2 + \frac{1}{2}I_{cm}\omega^2$$

## 五、角动量与守恒

### 知识点

**角动量：**
$$L = I \omega$$

单位：kg·m²/s。

**角动量定理：**
$$\int \tau dt = \Delta L = I\omega_f - I\omega_i$$

**角动量守恒定律：** 当合外力矩 $\sum \tau_{ext} = 0$ 时：
$$L = I\omega = \text{constant}$$

**常见守恒场景：**
- 旋转的椅子（人收臂 $I$ 减小，$\omega$ 增大）
- 子弹射入转盘
- 行星运动（对太阳的角动量守恒）

**注意：** 角动量守恒不需要系统机械能守恒，两者独立。

## 六、AP1 转动常考模型

### 模型1：有质量滑轮（Atwood 机转动版）

滑轮质量 $M$，半径 $R$，转动惯量 $I = \frac{1}{2}MR^2$。绳子不打滑，两侧张力 $T_1$ 和 $T_2$ **不等**。

**动力学方程：**
- 对 $m_1$：$m_1g - T_1 = m_1 a$
- 对 $m_2$：$T_2 - m_2g = m_2 a$
- 对滑轮：$(T_1 - T_2)R = I \alpha$，且 $a = \alpha R$

联立可求 $a$ 和张力。

### 模型2：纯滚动

**约束条件：**
$$v_{cm} = \omega R$$
$$a_{cm} = \alpha R$$

**总动能：**
$$K = \frac{1}{2}mv_{cm}^2 + \frac{1}{2}I_{cm}\omega^2$$

**斜面滚下加速度：**
$$a_{cm} = \frac{g\sin\theta}{1 + \frac{I}{mR^2}}$$

代入不同形状：
- 圆环：$a = \frac{1}{2}g\sin\theta$
- 圆盘：$a = \frac{2}{3}g\sin\theta$
- 实心球：$a = \frac{5}{7}g\sin\theta$

**注意：** 纯滚动中静摩擦力**不做功**，它只负责改变平动和转动的能量分配。

### 模型3：角动量守恒实例——转台与人/哑铃

人站在可自由旋转的平台上，手臂收拢时转动惯量减小，角速度增大；手臂展开时转动惯量增大，角速度减小。角动量 $L = I\omega$ 保持不变。

**常见计算：** 已知初始 $I_i$、$\omega_i$，求最终 $\omega_f$ 或 $I_f$：
$$I_i\omega_i = I_f\omega_f$$

### 模型4：刚体转动平衡（杠杆、跷跷板、门）

合外力矩为零：$\sum \tau = 0$。常用来求未知力或距离。注意选择转轴（通常选在未知力作用点，以简化方程）。

**典型题：** 均匀杆平衡、悬挂物体、梯子靠墙等。

### 模型5：滑轮系统与机械能守恒

若系统只有保守力（重力、弹簧力）做功，且滑轮转动无摩擦，则机械能守恒。表达式：
$$mgh_1 + \frac{1}{2}mv_1^2 + \frac{1}{2}I\omega_1^2 = mgh_2 + \frac{1}{2}mv_2^2 + \frac{1}{2}I\omega_2^2$$

并利用 $v = \omega R$ 关联线量和角量。

## ⚠️ 易错点

### AP1 转动易错点

1. **力矩计算时，力臂是转轴到力作用线的垂直距离，不是到作用点的距离。**

2. **转动惯量与质量分布和转轴位置有关，同物体不同转轴 $I$ 不同。**

3. **纯滚动中，静摩擦力方向与运动趋势相反，但不做功（接触点瞬时静止）。**

4. **角动量守恒要求合外力矩为零，与动量守恒条件（合外力为零）不同。**

5. **对于既有平动又有转动的物体，总动能是平动动能加转动动能，不可遗漏。**

## ✅ 参考答案

### 章节答案

**一、转动运动学**
1. 弧度（rad）
2. $s = r\theta$
3. $v_t = r\omega$
4. $a_t = r\alpha$

**二、力矩与平衡**
1. $\tau = rF\sin\phi$
2. 0

**三、转动惯量与转动定律**
1. 质量分布，转轴位置
2. $I = \frac{1}{2}MR^2$
3. $I = \frac{1}{3}ML^2$
4. $\sum \tau = I\alpha$

**四、转动动能**
1. $K_{rot} = \frac{1}{2}I\omega^2$
2. $W = \tau \Delta \theta$
3. $P = \tau \omega$

**五、角动量与守恒**
1. $L = I\omega$
2. 合外力矩为零

### 默写清单答案

1. $s=r\theta$，$v_t=r\omega$，$a_t=r\alpha$
2. $\tau = rF\sin\phi$
3. $I_{disk}=\frac{1}{2}MR^2$，$I_{rod,end}=\frac{1}{3}ML^2$
4. $\sum\tau = I\alpha$
5. $K_{rot}=\frac{1}{2}I\omega^2$
6. $L=I\omega$，合外力矩为零
7. $v_{cm}=\omega R$，$a_{cm}=\alpha R$
8. $a_{cm}=g\sin\theta/(1+I/(mR^2))$
