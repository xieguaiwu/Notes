# Spanish Pronoun — 西班牙语代词

## 一、人称代词 (Pronombres Personales)

### 1. 主格代词

| 人称 | 单数 | 复数 |
|------|------|------|
| 第一人称 | **yo**（我） | **nosotros/as**（我们） |
| 第二人称 | **tú**（你） | **vosotros/as**（你们） |
| 第三人称 | **él/ella/usted**（他/她/您） | **ellos/ellas/ustedes**（他们/她们/诸位） |

> [!note] 
> - **usted**（您）和 **ustedes**（诸位/你们）= 正式第二人称，但语法上用第三人称变位
> - **vosotros** 在拉美基本不用，拉美都用 **ustedes**

### 2. 宾格代词（与格/直接宾语）

| 人称 | 直接宾语 (lo/la) | 间接宾语 (le) |
|------|-----------------|-------------|
| me | me | me |
| te | te | te |
| lo / la | lo（他/它/您阳）/ la（她/它/您阴） | **le**（对他/她/您） |
| nos | nos | nos |
| os | os | os |
| los / las | los / las | **les**（对他们/她们/诸位）|

## 二、直接与间接宾语代词

### 1. 直接宾语代词（代替"谁"或"什么"）

| 代词 | 示例 |
|------|------|
| **me** | Él **me** ve.（他看见我。） |
| **te** | Yo **te** conozco.（我认识你。） |
| **lo / la** | **Lo** veo.（我看见他/它。）/**La** veo.（我看见她/它。）|
| **nos** | Ella **nos** ayuda.（她帮助我们。） |
| **os** | **Os** espero.（我等你们。） |
| **los / las** | **Los** conozco.（我认识他们。）|

### 2. 间接宾语代词（代替"对谁"）

| 代词 | 示例 |
|------|------|
| **me** | Ella **me** da un libro.（她给我一本书。） |
| **te** | Yo **te** escribo.（我给你写信。） |
| **le** | **Le** hablo.（我对他说/对她说。） |
| **nos** | Él **nos** dice la verdad.（他告诉我们真相。） |
| **os** | **Os** compro un regalo.（我给你们买礼物。） |
| **les** | **Les** pregunto.（我问他们。） |

### 3. 代词位置规则

| 规则 | 示例 |
|------|------|
| **动词前**（变位动词） | **Lo** **veo**. / **Me** **gusta**. |
| **动词后连写**（不定式/现在分词/命令式）| ver**lo**, comprán**dolo**, **Dime** la verdad. |
| **两个代词**（间接+直接） | **Me** **lo** da. / **Te** **la** compro. |

## 三、与格代词特殊用法

### 1. 动词 gustar 类结构

| 动词 | 结构 | 示例 |
|------|------|------|
| **gustar**（喜欢） | me/te/le/nos/os/les + gusta(n) | **Me gusta** el café. / **Me gustan** los libros. |
| **encantar**（非常喜欢） | 同上 | **Le encanta** viajar. |
| **interesar**（感兴趣） | 同上 | **Nos interesa** la historia. |
| **doler**（疼） | 同上 | **Me duele** la cabeza. |

> [!tip] 这类动词中，代词是"体验者"，真正的主语在动词后面。
> **Me gusta** el café → "咖啡对我而言是令人喜欢的"

### 2. 代词 le → se

当第三人称间接宾语（**le/les**）与直接宾语代词（**lo/la/los/las**）相邻时，**le/les** 变为 **se**：

| 原结构 | 变为 | 示例 |
|-------|------|------|
| le + lo | **se lo** | Le doy el libro. → **Se lo** doy. |
| le + la | **se la** | Le compro la flor. → **Se la** compro. |
| les + los | **se los** | Les vendo los coches. → **Se los** vendo. |

## 四、重读代词 (Pronombres Tónicos)

用于介词后或强调：

| 人称 | 重读形式 | 示例 |
|------|---------|------|
| yo | **mí** | Para **mí**, es importante. |
| tú | **ti** | Esto es para **ti**. |
| él/ella/usted | **él/ella/usted** | Voy con **él**. |
| nosotros | **nosotros** | Vive cerca de **nosotros**. |
| vosotros | **vosotros** | Este regalo es para **vosotros**. |
| ellos/ellas/ustedes | **ellos/ellas/ustedes** | Trabajo para **ellos**. |

> [!warning] con + mí/ti 的特殊形式：**conmigo**（和我）、**contigo**（和你）
