# Spanish Preposition — 西班牙语介词

## 一、介词概述

西班牙语介词通常较短（1-2个音节），但功能非常丰富。

---

## 二、常用介词用法

### 1. **a** — 到/向/在（时间+地点）

| 用法 | 示例 |
|------|------|
| 方向/目的地 | Voy **a** Madrid. |
| 时间（几点） | La clase empieza **a** las 9. |
| 间接宾语（人） | Le doy el libro **a** María. |
| 人称直接宾语（**a personal**） | Veo **a** Juan.（我见到胡安。） |
| 距离 | **a** dos kilómetros |
| 方式/速度 | **a** pie（步行）|

> [!warning] **a personal**：当直接宾语是**特指的人**时，必须加 a：
> - Veo **a** María.（✓） Veo María.（✗ 如果María是人）
> - Veo el libro.（物品不加 a）

### 2. **de** — 从/的/关于

| 用法 | 示例 |
|------|------|
| 所属 | el libro **de** María |
| 来源 | Soy **de** España. |
| 内容/材质 | una taza **de** café |
| 主题 | un libro **de** historia |
| 起点 | **de** lunes a viernes |
| 特征 | un hombre **de** pelo negro |

### 3. **en** — 在（地点/时间/语言/方式）

| 用法 | 示例 |
|------|------|
| 地点 | Estoy **en** casa. / Vivo **en** Madrid. |
| 时间（月份/年/季节） | **En** enero, **en** 2026 |
| 语言 | Hablar **en** español |
| 交通方式 | Viajar **en** tren / **en** avión |

### 4. **por** — 通过/因为/为了（原因/方式/交换）

| 用法 | 示例 |
|------|------|
| 原因 | Gracias **por** tu ayuda. |
| 方式/通过 | Hablo **por** teléfono. |
| 经过/穿过 | Paso **por** el parque. |
| 交换/价格 | Lo compré **por** 10 euros. |
| 每/频率 | una vez **por** semana |
| 为了（代某人） | Lo hago **por** ti. |

### 5. **para** — 为了/对于/前往（目的/目标）

| 用法 | 示例 |
|------|------|
| 目的/用途 | Este regalo es **para** ti. |
| 目的地 | Salgo **para** Madrid. |
| 时间期限 | Lo necesito **para** mañana. |
| 对…而言 | **Para** mí, es difícil. |
| 就业/职位 | Estudia **para** médico. |

---

## 三、Por vs Para 辨析

| 对比维度 | **por** | **para** |
|---------|:------:|:--------:|
| **原因vs目的** | 原因（因为） | 目的/目标（为了） |
| **时间** | 经过/通过 | 截止/期限 |
| **方向** | 经过/穿过 | 往/朝向（目的地） |
| **交换** | 价格/交换 | — |
| **代某人** | 代替/代 | 为（受益者） |

### 对比示例

| 用 por | 用 para |
|--------|---------|
| Gracias **por** la ayuda.（感谢帮助。） | Esto es **para** ti.（这是给你的。）|
| Paso **por** el banco.（我经过银行。）| Voy **para** Madrid.（我去马德里。）|
| **Por** la mañana（在早上） | **Para** mañana（到明天之前）|
| Lo compré **por** 5€.（买了5欧。）| Lo compré **para** Juan.（买给胡安。）|

## 四、常用介词搭配

| 介词 | 搭配示例 |
|------|---------|
| **con**（和/用） | conmigo, contigo, café con leche |
| **sin**（没有） | sin azúcar, sin ti |
| **entre**（在…之间） | entre amigos, entre dos y tres |
| **desde**（从…开始） | desde Madrid, desde las 8 |
| **hasta**（直到） | hasta luego, hasta mañana |
| **sobre**（在…上面/关于） | sobre la mesa, sobre la historia |
| **ante**（在…面前） | ante todo, ante la ley |
| **según**（根据） | según él, según el horario |

---

## 五、介词与动词的固定搭配

| 动词 + 介词 | 含义 | 示例 |
|------------|------|------|
| pensar **en** | 想到 | Pienso **en** ti. |
| soñar **con** | 梦见 | Sueño **con** viajar. |
| hablar **de** | 谈到 | Hablamos **de** política. |
| depender **de** | 取决于 | Depende **de** ti. |
| casarse **con** | 与…结婚 | Se casó **con** Ana. |
| acordarse **de** | 记得 | Me acuerdo **de** ti. |
| contar **con** | 依靠/有 | Cuento **con** tu ayuda. |

---

## 六、常见错误

### 1. Por / Para 混淆
❌ Gracias **para** tu ayuda.
✅ Gracias **por** tu ayuda.（感谢用 por）
❌ Esto es **por** ti.（本意：这是为了你）
✅ Esto es **para** ti.（受益用 para）

### 2. 人称 a（a personal）遗漏
❌ Veo **María**.（María 是人，需加 a）
✅ Veo **a** María.

### 3. 动词后介词搭配错误
❌ Pienso **de** ti.
✅ Pienso **en** ti.
❌ Sueño **de** viajar.
✅ Sueño **con** viajar.

### 4. En 和 dentro de 混淆（时间）
❌ **En** dos días llegaré.（模糊）
✅ **Dentro de** dos días = "两天后"（更精确）
