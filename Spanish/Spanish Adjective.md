# Spanish Adjective — 西班牙语形容词

## 一、形容词的性数配合

西班牙语形容词必须与所修饰的名词在**性**（阳性/阴性）和**数**（单数/复数）上保持一致。

### 1. 以 -o 结尾的形容词

| 阳性单数 | 阴性单数 | 阳性复数 | 阴性复数 |
|---------|---------|---------|---------|
| buen**o** | buen**a** | buen**os** | buen**as** |
| alt**o** | alt**a** | alt**os** | alt**as** |

- un hombre **alto** → una mujer **alta**
- hombres **altos** → mujeres **altas**

### 2. 以 -e 或辅音结尾的形容词

这类形容词**阴阳性同形**，只变数不变性：

| 单数 | 复数 |
|------|------|
| un hombre **inteligente** | hombres **inteligentes** |
| una mujer **inteligente** | mujeres **inteligentes** |
| un libro **azul** | libros **azules** |
| una casa **azul** | casas **azules** |

### 3. 以 -ista 结尾的形容词

阴阳性同形：

| 示例 |
|------|
| un hombre **optimista** / una mujer **optimista** |
| hombres **optimistas** / mujeres **optimistas** |

### 4. 以 -or / -án / -ín / -ón 结尾的形容词

| 阳性单数 | 阴性单数 | 规则 |
|---------|---------|------|
| trabajad**or** | trabajad**ora** | +a |
| holgaz**án** | holgaz**ana** | +a |
| pequeñ**ín** | pequeñ**ina** | +a |
| comil**ón** | comil**ona** | +a |

> [!warning] **例外**：mejor（更好）、peor（更差）、superior/inferior 等比较级形容词阴阳性同形。

## 二、形容词的位置

### 1. 一般规则：名词之后

大部分西班牙语形容词放在**名词之后**：

| 示例 |
|------|
| un libro **interesante**（一本有趣的书）|
| una casa **grande**（一所大房子）|
| un vestido **rojo**（一条红裙子）|

### 2. 名词前的形容词

| 类别 | 示例 |
|------|------|
| **限定性形容词**（数量/序数）| **primer** libro, **muchas** personas |
| **评价性形容词**（移前表主观）| **Buena** idea, **mal** momento |
| **固有属性形容词** | **blanca** nieve, **dulce** azúcar |
| **缩短形式**的形容词 | **gran** hombre, **buen** amigo |

### 3. 位置改变含义的形容词

| 形容词 | 名词前（引申义） | 名词后（本义） |
|--------|----------------|---------------|
| **gran / grande** | un **gran** hombre（伟人）| un hombre **grande**（高个子男人）|
| **pobre** | un **pobre** hombre（可怜的人）| un hombre **pobre**（贫穷的男人）|
| **nuevo** | un **nuevo** coche（新换的车）| un coche **nuevo**（崭新的车）|
| **viejo** | un **viejo** amigo（老朋友）| un amigo **viejo**（年纪大的朋友）|
| **único** | la **única** solución（唯一的办法）| una solución **única**（独特的办法）|

## 三、缩短形式 (Apócope)

某些形容词在**阳性单数名词前**去掉词尾：

| 原形 | 缩短形式 | 示例 |
|------|---------|------|
| **grande** | **gran** | un **gran** día |
| **bueno** | **buen** | un **buen** amigo |
| **malo** | **mal** | un **mal** momento |
| **primero** | **primer** | el **primer** libro |
| **tercero** | **tercer** | el **tercer** piso |
| **alguno** | **algún** | **algún** día |
| **ninguno** | **ningún** | **ningún** problema |
| **ciento** | **cien** | **cien** años |

> [!note] 这些缩短只在**阳性单数名词前**发生。阴性不变：**una buena** amiga

## 四、比较级简介

| 比较类型 | 结构 | 示例 |
|---------|------|------|
| **更高程度** | **más + adj + que** | Él es **más alto que** yo. |
| **更低程度** | **menos + adj + que** | Ella es **menos alta que** él. |
| **同等程度** | **tan + adj + como** | Soy **tan alto como** tú. |

### 不规则比较级

| 原级 | 比较级 | 最高级 |
|------|--------|-------|
| **bueno**（好的） | **mejor**（更好） | **óptimo / el mejor** |
| **malo**（坏的） | **peor**（更坏） | **pésimo / el peor** |
| **grande**（大的） | **mayor**（更大/年长） | **máximo / el mayor** |
| **pequeño**（小的） | **menor**（更小/年幼） | **mínimo / el menor** |
