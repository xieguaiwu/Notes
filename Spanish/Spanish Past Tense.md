# Spanish Past Tense — 现在完成时 (Pretérito Perfecto)

## 一、时态概述

现在完成时（Pretérito Perfecto Compuesto / Pretérito Perfecto）用于表达：
- **近期已完成的动作**：Hoy **he comido** pizza.（今天我吃了披萨。）
- **对现在有影响的过去**：**Has ganado** la lotería.（你中了彩票！）
- **未指明具体时间的经验**：**He visitado** Madrid tres veces.（我去过马德里三次。）

---

## 二、构成公式

```
haber（现在时变位）+ 过去分词
```

### 1. 助动词 haber 的现在时变位

| 人称 | haber 变位 |
|------|-----------|
| yo | **he** |
| tú | **has** |
| él/ella/usted | **ha** |
| nosotros | **hemos** |
| vosotros | **habéis** |
| ellos/ellas/ustedes | **han** |

### 2. 过去分词构成

| 动词类型 | 规则 | 示例 |
|---------|------|------|
| **-ar** | 词干 + **-ado** | habl**ar** → habl**ado** |
| **-er / -ir** | 词干 + **-ido** | com**er** → com**ido** / viv**ir** → viv**ido** |

---

## 三、不规则过去分词

| 动词 | 过去分词 | 含义 |
|------|---------|------|
| **hacer** | **hecho** | 做 |
| **decir** | **dicho** | 说 |
| **ver** | **visto** | 看 |
| **escribir** | **escrito** | 写 |
| **abrir** | **abierto** | 打开 |
| **cubrir** | **cubierto** | 覆盖 |
| **volver** | **vuelto** | 回来 |
| **romper** | **roto** | 打破 |
| **poner** | **puesto** | 放 |
| **morir** | **muerto** | 死 |
| **resolver** | **resuelto** | 解决 |
| **freír** | **frito** | 煎 |
| **satisfacer** | **satisfecho** | 满足 |

> [!tip] 复合词也遵循相同规则：
> - descubrir → **descubierto**（发现）
> - devolver → **devuelto**（归还）
> - componer → **compuesto**（组成）

---

## 四、现在完成时的否定与疑问

### 否定式

```
no + haber + 过去分词
```

- **No he** comido nada.（我什么都没吃。）
- Ella **no ha** llegado todavía.（她还没到。）

### 疑问式

```
¿(疑问词) + haber + 主语 + 过去分词 ?
```

- ¿**Has** visto esta película?（你看过这部电影吗？）
- ¿Dónde **has** puesto las llaves?（你把钥匙放哪儿了？）

## 五、简单过去时 (Pretérito Indefinido) — 简介

A2 阶段认知即可，B1 系统学习。用于表达**已完成的过去动作**（通常有时间状语）。

| 过去时 | 现在完成时 (Perfecto) | 简单过去时 (Indefinido) |
|-------|--------------------|--------------------|
| 时间感 | 与现在相关/近期 | 与现在无关/已完成 |
| 常用时间词 | hoy, esta semana, ya, todavía no | ayer, anoche, la semana pasada |
| 示例 | **He comido** hoy. | Ayer **comí** pizza. |

### 简单过去时 -ar 动词变位

| 人称 | hablar → |
|------|---------|
| yo | habl**é** |
| tú | habl**aste** |
| él/ella/ud. | habl**ó** |
| nosotros | habl**amos** |
| vosotros | habl**asteis** |
| ellos/ellas/uds. | habl**aron** |

---

## 六、时间标志词速查

| 现在完成时常用词 | 简单过去时常用词 |
|----------------|----------------|
| **hoy**（今天）| **ayer**（昨天）|
| **esta semana**（这周）| **anoche**（昨晚）|
| **este mes**（这个月）| **la semana pasada**（上周）|
| **ya**（已经）| **el año pasado**（去年）|
| **todavía no**（还没有）| **en 2025**（在2025年）|
| **nunca**（从未）| **entonces**（那时）|
| **alguna vez**（曾经）| **de repente**（突然）|

> [!note] A1-A2 阶段优先掌握**现在完成时**。简单过去时了解基本概念即可。

---

## 七、常见错误

### 1. Haber 变位混淆
❌ Yo **ha** comido.
✅ Yo **he** comido.

### 2. 不规则过去分词拼写
❌ He **hacido**.
✅ He **hecho**.

### 3. 现在完成时与简单过去时混用
❌ Ayer **he comido** pizza.（ayer 是明确过去时间词）
✅ Ayer **comí** pizza.

### 4. Haber 和 Tener 混淆
❌ **He** un libro.（haber 不作"拥有"解）
✅ **Tengo** un libro.
