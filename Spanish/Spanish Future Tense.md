# Spanish Future Tense — 将来时

## 一、口语将来时：Ir a + 不定式

这是西班牙语最常用的将来表达方式（A1 即可掌握）。

### 1. 构成公式

```
ir（现在时变位）+ a + 不定式
```

| 人称 | ir 变位 | 完整形式 |
|------|:-------:|---------|
| **yo** | voy | **voy a** hablar |
| **tú** | vas | **vas a** hablar |
| **él/ella/usted** | va | **va a** hablar |
| **nosotros** | vamos | **vamos a** hablar |
| **vosotros** | vais | **vais a** hablar |
| **ellos/ellas/ustedes** | van | **van a** hablar |

### 2. 用法

| 用途 | 示例 |
|------|------|
| **即将发生的动作** | **Voy a** salir ahora.（我马上要出去。）|
| **计划/打算** | **Vamos a** viajar a México.（我们打算去墨西哥。）|
| **预测（即将）** | **Va a** llover.（要下雨了。）|

### 3. 否定式

```
no + ir + a + 不定式
```

- **No voy a** venir.（我不来了。）
- **No va a** llover.（不会下雨。）

## 二、陈述式将来时 (Futuro Simple / Futuro de Indicativo)

用于表达**较远将来**或**推测**，A2 阶段认知即可。

### 1. 构成公式

```
不定式 + 词尾：-é, -ás, -á, -emos, -éis, -án
```

以 **hablar** 为例：

| 人称 | 变位 |
|------|------|
| yo | hablar**é** |
| tú | hablar**ás** |
| él/ella/usted | hablar**á** |
| nosotros | hablar**emos** |
| vosotros | hablar**éis** |
| ellos/ellas/ustedes | hablar**án** |

> [!tip] 所有三组动词（-ar, -er, -ir）使用**相同的词尾**，直接加在不定式后。

### 2. 常见不规则词根

| 动词 | 将来时词根 |
|------|-----------|
| **decir**（说）| **dir-** → diré |
| **hacer**（做）| **har-** → haré |
| **poder**（能够）| **podr-** → podré |
| **poner**（放）| **pondr-** → pondré |
| **querer**（想要）| **querr-** → querré |
| **saber**（知道）| **sabr-** → sabré |
| **salir**（出去）| **saldr-** → saldré |
| **tener**（有）| **tendr-** → tendré |
| **venir**（来）| **vendr-** → vendré |
| **valer**（值得）| **valdr-** → valdré |

### 3. 用法

| 用法 | 示例 |
|------|------|
| **将来动作**（较远、正式）| **Viajaré** a España el año que viene.（明年我将去西班牙。）|
| **推测/可能性**（现在）| ¿Dónde **estará** María?（玛丽亚会在哪儿呢？）|
| **承诺** | Te **llamaré** mañana.（我明天给你打电话。）|

## 三、现在时表将来

现在时 + 时间状语也可表达将来（确定计划）：

| 现在时（表将来） | 中文 |
|-----------------|------|
| **Salgo** mañana para Madrid. | 我明天出发去马德里。 |
| El tren **llega** a las 8. | 火车8点到。 |

---

## 四、三种将来表达对比

| 表达方式 | 构成 | 使用场景 | 示例 |
|---------|------|---------|------|
| **现在时** | 现在时 + 时间状语 | 确定安排 | **Salgo** mañana. |
| **Ir a + inf** | ir + a + 不定式 | 即将/打算（口语）| **Voy a** salir. |
| **将来时** | 不定式 + 词尾 | 较远/推测/正式 | **Saldré** mañana. |

> [!note] A1-A2 阶段优先掌握 **ir a + 不定式**，陈述式将来时了解即可。
