# Spanish Noun Gender — 名词的性与数

## 一、名词的性 (Género)

西班牙语名词分为**阳性 (masculino)** 和 **阴性 (femenino)**。

> [!tip] 核心规律：看词尾！
> - **-o** 结尾 → 绝大多数阳性
> - **-a** 结尾 → 绝大多数阴性

### 1. 阳性名词常见词尾

| 词尾 | 示例 |
|------|------|
| **-o** | el libr**o**, el coch**o**, el trabaj**o** |
| **-ma**（希腊词源） | el problem**a**, el sistem**a**, el tem**a** |
| **-aje** | el viaj**e**, el pais**aje**, el mensaj**e** |
| **-án / -és / -ín** | el pan, el café**s**, el jardí**n** |
| **-or**（职业） | el act**or**, el profes**or** |
| **-ama / -ema / -oma** | el progr**ama**, el po**ema**, el idi**oma** |

> [!note] -ma 结尾的阳性名词来自希腊语，在西班牙语中保留阳性：
> el problema（问题）、el tema（主题）、el sistema（系统）

### 2. 阴性名词常见词尾

| 词尾 | 示例 |
|------|------|
| **-a** | la mes**a**, la cas**a**, la sill**a** |
| **-ción / -sión** | la nación, la televisión |
| **-dad / -tad / -tud** | la universi**dad**, la liber**tad** |
| **-ez / -eza** | la vej**ez**, la bellez**a** |
| **-umbre** | la cost**umbre**, la incertid**umbre** |
| **-ie** | la ser**ie**, la superfic**ie** |

### 3. 例外记忆

| 看似阳性实为阴性 | 看似阴性实为阳性 |
|-----------------|----------------|
| **la** mano（手）-a 结尾 | **el** día（天）-a 结尾 |
| **la** foto（照片）缩写 | **el** mapa（地图）-a 结尾 |
| **la** moto（摩托车）缩写 | **el** sofá（沙发）-a 结尾 |
| **la** radio（收音机） | **el** tranvía（有轨电车）|

## 二、名词的数 (Número)

### 1. 复数构成规则

| 情况 | 规则 | 单数 → 复数 |
|------|------|-------------|
| **元音结尾** | +**s** | libro → libro**s**, casa → casa**s** |
| **辅音结尾** | +**es** | papel → papel**es**, ciudad → ciudad**es** |
| **z 结尾** | z → c + es | lápiz → láp**ices**, voz → vo**ces** |
| **重读元音结尾 (í, ú)** | +es / +s | rubí → rubí**es** / marroquí → marroquí**es** |

### 2. 特殊复数

| 单数 | 复数 | 说明 |
|------|------|------|
| **el** problema | **los** problema**s** | -ma 结尾正常+s |
| **el** lunes | **los** lunes | 不变 |
| **el** jueves | **los** jueves | 不变 |
| **la** crisis | **las** crisis | 不变 |
| el café | los café**s** | 重读 e 结尾+s |

## 三、表示人的名词的性数变化

| 阳性 | 阴性 | 规则 |
|------|------|------|
| el amig**o** | la amig**a** | -o → -a |
| el profes**or** | la profes**ora** | -or → -ora |
| el estudiante | la estudiante | 不变 |
| el dentista | la dentista | 不变 |
| el act**or** | la act**riz** | 特殊 |

## 四、记忆技巧

### 词尾规律口诀

> **-o 阳，-a 阴**，规律记在心
> **-ción, -dad, -tad** → 阴性别搞差
> **-ma, -pa, -ta** 希腊来 → 阳性要明白
> **-aje, -or** → 阳性不会错

### 核心策略

> [!note] 最佳记忆策略
> 1. 学新名词时连冠词一起记：**el** libro, **la** mesa
> 2. 先记词尾规律（-o 阳 / -a 阴），再重点关注例外
> 3. **-ma** 结尾的希腊借词（problema, tema, sistema, programa）全是阳性，高频考点
