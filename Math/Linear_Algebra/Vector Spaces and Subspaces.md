# Vector Spaces and Subspaces

## 1. 核心概念：一句话概括

线性代数研究的是"可以做加法和数乘的集合"。**向量空间**就是这样一个集合——其中的元素（向量）可以相加、可以乘以标量，且结果仍在集合内；**子空间**是向量空间内的一个子集，它本身也构成一个向量空间。为什么这很重要？因为现实世界中到处都是这类结构：物理中的力可以合成和缩放，图像可以叠加和调亮度，多项式可以相加和伸缩。向量空间是所有这些操作的统一数学框架。

## 2. 向量空间的定义

### 2.1 Why This Matters

向量空间的概念是线性代数的"舞台布景"——它定义了"在哪里工作"。就像运动员需要知道球场边界一样，线性代数需要在明确定义的空间中运作。一旦你确认某个集合是向量空间，所有线性代数的工具（矩阵、基、维数、线性变换）都可以直接使用。这就是为什么它在数学、物理、工程、计算机图形学、机器学习中无处不在。

### 2.2 直觉先行

想象你能在三维空间中画的所有箭头。你可以把两个箭头头尾相接（加法），也可以把箭头拉长或缩短（数乘）。不管你怎么操作，结果依然是三维空间中的一个箭头——你不会"跑出"三维空间。这就是向量空间的本质：**一个在加法和数乘下封闭的集合**。

同样的道理适用于多项式：两个二次多项式相加还是二次多项式，一个二次多项式乘以常数还是二次多项式。它们构成一个向量空间，只不过"箭头"换成了"多项式"。

一个集合 $V$ 称为 $\mathbb{R}$ 上的向量空间（vector space），如果定义了**加法**和**数乘**两种运算，且满足以下 **8 条公理**：

### 加法封闭性

对任意 $\mathbf{u}, \mathbf{v} \in V$，$\mathbf{u} + \mathbf{v} \in V$

> **直觉**：两个向量相加不会"逃出"空间。就像两个三維空间中的箭头相加还是三维空间中的箭头——你不会意外跑到四维去。如果运算可能把你带出空间，那么这个集合就不能叫做向量空间。

### 数乘封闭性

对任意 $\mathbf{v} \in V$ 和标量 $c \in \mathbb{R}$，$c\mathbf{v} \in V$

> **直觉**：把一个向量拉伸、压缩、甚至反向，结果仍然在空间内。如果某个方向拉伸后箭头会"穿出"空间，那么这个方向就不属于这个向量空间。

### 8 条公理

这些公理不是任意的——它们可以被分为**三个逻辑组**，每组保证一种运算"讲道理"：

**第 1 组：加法公理（1-4）——保证加法直观合理**

就像日常数字加法一样，向量的加法也应该交换顺序不变、分组方式不变、有零元素、有相反数。没有这些，加法就不可预测。

**第 2 组：数乘公理（5, 8）——保证数乘直观合理**

连续两次缩放等价于一次缩放（先缩 2 倍再缩 3 倍 = 直接缩 6 倍），乘以 1 不改变任何东西。这保证了缩放操作的一致性。

**第 3 组：分配律（6, 7）——架接两种运算**

加法和数乘不能各玩各的——分配律确保它们协调工作：把两个数乘结果相加等价于先加标量再数乘；把两个向量相加后再数乘等价于分别数乘再加。

| # | 名称 | 条件 | 为什么需要 |
|---|------|------|----------|
| 1 | 加法交换律 | $\mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}$ | 顺序无关——加法的结果不依赖于谁先谁后 |
| 2 | 加法结合律 | $(\mathbf{u} + \mathbf{v}) + \mathbf{w} = \mathbf{u} + (\mathbf{v} + \mathbf{w})$ | 分组无关——不用担心括号位置 |
| 3 | 零元存在 | 存在 $\mathbf{0} \in V$ 使得 $\mathbf{v} + \mathbf{0} = \mathbf{v}$ | 需要一个"什么都不做"的元素——加法的基础 |
| 4 | 负元存在 | 对每个 $\mathbf{v} \in V$，存在 $-\mathbf{v}$ 使得 $\mathbf{v} + (-\mathbf{v}) = \mathbf{0}$ | 每个向量都需要一个"撤销"操作 |
| 5 | 数乘结合律 | $c(d\mathbf{v}) = (cd)\mathbf{v}$ | 连续缩放等价于一次缩放 |
| 6 | 数乘分配律 1 | $(c + d)\mathbf{v} = c\mathbf{v} + d\mathbf{v}$ | 标量加法的分配——拆开缩放倍数 |
| 7 | 数乘分配律 2 | $c(\mathbf{u} + \mathbf{v}) = c\mathbf{u} + c\mathbf{v}$ | 向量加法的分配——拆开缩放对象 |
| 8 | 数乘单位元 | $1\mathbf{v} = \mathbf{v}$ | 数乘的"单位元"——乘以 1 不做任何改变 |

> [!note] 不需要死记
> 实际使用时，绝大多数自然出现的集合都满足这些公理。核心记住**加法和数乘封闭**最关键。公理存在的意义是确保运算行为良好——就像交通规则一样，你不需要每次开车都默念一遍，但它们保障了道路秩序。

### 常见向量空间举例

| 空间 | 元素 | 维数 |
|------|------|------|
| $\mathbb{R}^n$ | $n$ 维实向量 | $n$ |
| $\mathbb{R}^{m \times n}$ | $m \times n$ 实矩阵 | $mn$ |
| $\mathcal{P}_n$ | 次数 $\le n$ 的多项式 | $n+1$ |
| $\mathcal{C}[a,b]$ | $[a,b]$ 上的连续函数 | $\infty$ |

> **为什么这些例子重要**：虽然我们常常用 $\mathbb{R}^n$ 的箭头做直觉练习，但向量空间远不止于此。图像处理把图片看作向量（每个像素一个分量），量子力学把状态看作向量，金融把资产组合看作向量。**每当你发现"可以加、可以缩放"的结构，向量空间的视角就适用。**

### 2.5 向量空间的基本性质

从 8 条公理可以推导出一些基本但重要的性质。它们看起来很简单，但每次使用背后都有公理支撑。

#### 1. 零向量唯一

向量空间中有**且只有一个**零向量 $\mathbf{0}$。

> **证明**：假设有两个零向量 $\mathbf{0}$ 和 $\mathbf{0}'$。那么 $\mathbf{0}' = \mathbf{0}' + \mathbf{0} = \mathbf{0}$。所以它们其实是同一个。

#### 2. 每个向量的负元唯一

对每个 $\mathbf{v} \in V$，存在**唯一的** $-\mathbf{v}$ 使得 $\mathbf{v} + (-\mathbf{v}) = \mathbf{0}$。

> **证明**：假设 $w$ 和 $w'$ 都是 $\mathbf{v}$ 的负元。则 $w = w + \mathbf{0} = w + (\mathbf{v} + w') = (w + \mathbf{v}) + w' = \mathbf{0} + w' = w'$。

#### 3. $0 \cdot \mathbf{v} = \mathbf{0}$（标量 0 乘任何向量得零向量）

> **证明**：$0\mathbf{v} = (0+0)\mathbf{v} = 0\mathbf{v} + 0\mathbf{v}$。两边同时加上 $0\mathbf{v}$ 的负元，得 $\mathbf{0} = 0\mathbf{v}$。

#### 4. $a \cdot \mathbf{0} = \mathbf{0}$（任何标量乘零向量得零向量）

> **证明**：$a\mathbf{0} = a(\mathbf{0}+\mathbf{0}) = a\mathbf{0} + a\mathbf{0}$。两边同时加上 $a\mathbf{0}$ 的负元，得 $\mathbf{0} = a\mathbf{0}$。

#### 5. $(-1)\mathbf{v} = -\mathbf{v}$

> **证明**：$\mathbf{v} + (-1)\mathbf{v} = 1\mathbf{v} + (-1)\mathbf{v} = (1 + (-1))\mathbf{v} = 0\mathbf{v} = \mathbf{0}$。所以 $(-1)\mathbf{v}$ 就是 $\mathbf{v}$ 的负元。

#### 6. 如果 $a\mathbf{v} = \mathbf{0}$，则 $a = 0$ 或 $\mathbf{v} = \mathbf{0}$

> **直觉**：数乘结果为零的唯一可能是要么标量为零，要么向量本身为零。如果 $a \neq 0$，那么可以两边同乘 $1/a$：$\mathbf{v} = (1/a)(a\mathbf{v}) = (1/a)\mathbf{0} = \mathbf{0}$。

```
基本性质速记：
  • 零向量只有一个，负元也唯一
  • 0 · v = 0，a · 0 = 0
  • (-1)v = -v（数乘 -1 就是取负元）
  • av = 0 ⇒ a=0 或 v=0
```

> [!tip] 这些性质的价值
> 这些推论虽然看似显然，但其意义在于：**它们完全从 8 条公理推导而来，不依赖任何具体的向量空间**。无论你是在 $\mathbb{R}^n$、多项式空间还是函数空间中工作，这些性质都自动成立。这就是公理化方法的威力——从少量规则推出大量结论。

## 3. 子空间（Subspace）

### 3.0 Why Subspaces Matter

子空间让我们可以在一个更大的向量空间内**聚焦于感兴趣的部分**。想象你在三维空间中，但只关心某个过原点的平面——这个平面本身就是一个完整的二维世界，所有线性代数规则在其中依然成立。子空间就是这样的"空间中的空间"：它继承了母空间的运算，但维度更低、结构更简单。

### 3.1 定义

$H \subseteq V$ 是 $V$ 的**子空间**（subspace），当且仅当：
1. $\mathbf{0} \in H$（含零向量）
2. **加法封闭**：$\mathbf{u}, \mathbf{v} \in H \implies \mathbf{u} + \mathbf{v} \in H$
3. **数乘封闭**：$\mathbf{v} \in H, c \in \mathbb{R} \implies c\mathbf{v} \in H$

```
子空间的直觉示意图:

  ℝ³（整个三维空间）          过原点的平面 H（子空间）
  ┌──────┐                    ┌──────┐
  │┼┼┼┼┼┼│                    │//////│
  │┼┼┼┼┼┼│                    │//////│
  │┼┼┼┼┼┼│                    │//////│
  └──────┘                    └──────┘
  所有方向都可用               只在平面内操作
                              但所有线性规则仍然成立
  
  关键在于：平面 H 必须过原点（含零向量），
  否则加法或数乘的结果会逃出平面。
```

> [!tip] 检验技巧
> 要验证 $H$ 是子空间，只需验证三条：**零向量 + 加法封闭 + 数乘封闭**。如果 $H$ 是用"满足某条线性方程/条件"描述的，通常就是子空间。三条中任何一条不满足，$H$ 就不是子空间。

### 3.2 例子

**是子空间：**

- $\mathbb{R}^2$ 中过原点的直线：$\{t(1,2) \mid t \in \mathbb{R}\}$
  > 过原点所以含零向量；直线上任意两个向量相加仍在直线上；数乘只是把向量沿着直线滑动。
  
- $\mathbb{R}^3$ 中过原点的平面：$\{s(1,0,1) + t(0,1,0) \mid s,t \in \mathbb{R}\}$
  > 过原点的无限大平面：任意两个面上的向量相加还在面上，缩放也在面上。
  
- 所有对称矩阵构成 $\mathbb{R}^{n \times n}$ 的子空间
  > 对称矩阵相加仍对称，数乘也对称。
  
- [[Rank and Nullity]] 中介绍的零空间 $\text{Null}(\mathbf{A})$ 和列空间 $\text{Col}(\mathbf{A})$
  > 这是子空间最重要的来源之一：任何矩阵的零空间和列空间都是子空间。

```
子空间示例：ℝ³ 中过原点的平面 H

          z
          │
          │  ╱ H（过原点的平面）
          │ ╱
          │╱
          └────── y
         ╱
        ╱
       x
    
    特征：
    • 原点 (0,0,0) ∈ H ✓
    • 平面上任两点相加仍在平面 ✓
    • 平面上任一向量缩放仍在平面 ✓
    
    H 本身就是一个二维向量空间！
```

**不是子空间：**

- $\mathbb{R}^2$ 中**不过**原点的直线：不含零向量
  > **为什么失败**：直线 $L: y = 2x + 1$ 不含 $(0,0)$ —— 最基本的"零元"条件就不满足。而且这条直线上的两个向量相加会得到更高/更低位置的向量，不在直线上。一条不过原点的直线就像一个偏离中心的舞台——演员（向量）会掉下去。
  
- 第一象限 $\{(x,y) \mid x \ge 0, y \ge 0\}$：数乘 $-1$ 不封闭
  > **为什么失败**：虽然含零向量 $(0,0)$，加法也封闭（两个非负坐标相加仍非负），但数乘 $-1$ 会把向量翻转到第三象限。集合在数乘下"漏了"。比如 $(1,2)$ 在第一象限，但 $-1 \cdot (1,2) = (-1,-2)$ 就不在第一象限了。**一条封闭性被破坏，整个子空间资格就丧失了。**
  
- 单位圆 $\{(x,y) \mid x^2 + y^2 = 1\}$：加法和数乘都不封闭
  > **为什么失败**：单位圆上的两个向量 $(1,0) + (0,1) = (1,1)$，长度 $\sqrt{2} \ne 1$，不在圆上。数乘 2 倍也会离开圆。**单位圆是弯曲的 1 维对象，它不遵守线性规则——线性代数是关于直线和平面的，不是曲线。**

### 3.3 子空间的和与直和

#### 3.3.0 Why This Matters

子空间的和让我们可以从较小的"积木"构造更大的空间，而直和则确保这种构造是"无重叠"的——每个元素有唯一的分解方式。这就像乐高积木：你可以把小块拼成大块（和），但如果两块积木的功能重叠（非直和），那么拼法就不唯一了。直和在后续学习[[Diagonalization]]、[[Eigenvalues and Eigenvectors]]中的空间分解时至关重要。

#### 3.3.1 子空间的和（Sum of Subspaces）

设 $U_1, \dots, U_m$ 是 $V$ 的子空间。它们的**和**定义为：

$$
U_1 + \cdots + U_m = \{\mathbf{u}_1 + \cdots + \mathbf{u}_m \mid \mathbf{u}_1 \in U_1, \dots, \mathbf{u}_m \in U_m\}
$$

即所有来自各子空间的向量之和的集合。子空间的和**仍然是子空间**。

> **直觉**：和就是"把所有子空间的向量任意组合相加"。它类似于集合论中的并集，但并集通常不是子空间（见习题 9），而和是包含所有子空间的**最小子空间**。

**例子：** 在 $\mathbb{R}^3$ 中，设
$$
U = \{(x, 0, 0) \in \mathbb{R}^3 \mid x \in \mathbb{R}\}, \quad 
W = \{(0, y, 0) \in \mathbb{R}^3 \mid y \in \mathbb{R}\}
$$
则 $U + W = \{(x, y, 0) \in \mathbb{R}^3 \mid x, y \in \mathbb{R}\}$（即 $xy$-平面）。

```
子空间的和的直觉：

  ℝ³ 中子空间 U = x 轴：
  U = {(x,0,0)}
  
  ℝ³ 中子空间 W = y 轴：
  W = {(0,y,0)}
  
  U + W = xy-平面：
  {(x,y,0)}
  
  就像两个垂直的方向"张成"了一个平面。
```

#### 3.3.2 直和（Direct Sum）

如果每个向量 $\mathbf{v} \in V$ 可以**唯一地**表示为 $\mathbf{v} = \mathbf{u}_1 + \cdots + \mathbf{u}_m$（其中每个 $\mathbf{u}_j \in U_j$），则称 $V$ 是 $U_1, \dots, U_m$ 的**直和**（direct sum），记作：

$$
V = U_1 \oplus U_2 \oplus \cdots \oplus U_m
$$

> 符号 $\oplus$（圆内加号）强调直和是一种特殊的和——每个元素的表示方式唯一。

**如何判断直和？** 只需要检验 **零向量的表示唯一**：

> [!abstract] 直和判定定理
> $V = U_1 \oplus \cdots \oplus U_m$ 当且仅当：
> 1. $V = U_1 + \cdots + U_m$（能生成整个空间）
> 2. $\mathbf{0}$ 的唯一表示是 $\mathbf{0} = \mathbf{0} + \cdots + \mathbf{0}$（即若 $\mathbf{u}_1 + \cdots + \mathbf{u}_m = \mathbf{0}$，则所有 $\mathbf{u}_j = \mathbf{0}$）

**两个子空间的直和：** 对于两个子空间，有一个更简单的判别法：

> [!abstract] 两个子空间的直和判定
> $V = U \oplus W$ 当且仅当 $V = U + W$ **且** $U \cap W = \{\mathbf{0}\}$。

> **证明**：如果 $V = U \oplus W$，显然 $V = U + W$，且若 $\mathbf{v} \in U \cap W$，则 $\mathbf{0} = \mathbf{v} + (-\mathbf{v})$（$\mathbf{v} \in U, -\mathbf{v} \in W$），由零向量表示唯一得 $\mathbf{v} = \mathbf{0}$。
> 
> 反之，若 $V = U + W$ 且 $U \cap W = \{\mathbf{0}\}$，设 $\mathbf{0} = \mathbf{u} + \mathbf{w}$（$\mathbf{u} \in U, \mathbf{w} \in W$），则 $\mathbf{u} = -\mathbf{w} \in W$，所以 $\mathbf{u} \in U \cap W = \{\mathbf{0}\}$，故 $\mathbf{u} = \mathbf{w} = \mathbf{0}$。由判定定理得证。

#### 3.3.3 例子

**✅ 是直和：**

- $\mathbb{R}^3 = \{(x,y,0)\} \oplus \{(0,0,z)\}$
  > $xy$-平面与 $z$ 轴只交于原点，且它们的和是整个 $\mathbb{R}^3$。
  
- $\mathbb{F}^n = U_1 \oplus \cdots \oplus U_n$，其中 $U_j$ 是第 $j$ 个坐标轴
  > 每个向量 $(x_1,\dots,x_n)$ 可以唯一地写为各坐标轴分量的和。
  
- 多项式空间 $\mathcal{P}(\mathbb{F}) = \mathcal{P}_{\text{偶}} \oplus \mathcal{P}_{\text{奇}}$，其中偶次多项式和奇次多项式的和
  > 每个多项式可以唯一地分解为偶次部分和奇次部分之和。

**❌ 不是直和：**

考虑 $\mathbb{F}^3$ 中的三个子空间：
$$
U_1 = \{(x,y,0)\}, \quad U_2 = \{(0,0,z)\}, \quad U_3 = \{(0,y,y)\}
$$
显然 $\mathbb{F}^3 = U_1 + U_2 + U_3$，但 $\mathbf{0}$ 有两种表示方式：
$$
\mathbf{0} = (0,1,0) + (0,0,1) + (0,-1,-1) = \mathbf{0} + \mathbf{0} + \mathbf{0}
$$
所以不是直和。即使 $U_1 \cap U_2 = U_1 \cap U_3 = U_2 \cap U_3 = \{\mathbf{0}\}$（两两相交只有零向量），三者放在一起仍可能不是直和。**对于三个及以上子空间，仅两两之交为零是不够的。**

```
直和的几何直觉：

  ℝ² = x 轴 ⊕ y 轴
  
       y
       │
       │  任意向量 (x,y) 可以
       │  唯一地分解为：
       │  • x 轴分量 (x,0)
   (0,y)│  • y 轴分量 (0,y)
       │
       └────────── x
          (x,0)
  
  这就像平面上的每个点有唯一的经纬度。
  两个坐标轴只在原点相交——这就是直和的本质。
```

#### 3.3.4 直和的应用

直和的核心思想是**分解**：将一个复杂空间拆成简单子空间的直和，分别分析每个子空间，然后组合结果。这在以下场景中反复出现：

- **特征空间分解**：如果一个算子可对角化，则 $V$ 是各特征空间的直和。参见 [[Diagonalization]]
- **广义特征空间**：即使不可对角化，$V$ 也可以分解为广义特征空间的直和（Jordan 标准型）
- **正交补**：在内积空间中，$V = U \oplus U^\perp$（子空间与其正交补的直和），参见 [[Inner Product and Norm]]

> [!tip] 核心记忆
> **直和 = 子空间的和 + 零向量表示唯一**。两个子空间时，等价于**交集只有零向量**。

## 4. 生成（Span）

### 4.0 Why Span Matters

Span 是**从少量向量构造出整个子空间**的方法。你手头有几个向量，想找到"所有能用这些向量到达的地方"——这就是 span。它是连接"几个向量"和"子空间"的桥梁。

向量集合 $\{\mathbf{v}_1, \mathbf{v}_2, \dots, \mathbf{v}_k\}$ 的**生成**（span）：

$$
\text{span}\{\mathbf{v}_1, \dots, \mathbf{v}_k\} = \{c_1\mathbf{v}_1 + \cdots + c_k\mathbf{v}_k \mid c_i \in \mathbb{R}\}
$$

即所有线性组合的集合。$\text{span}\{\mathbf{v}_1, \dots, \mathbf{v}_k\}$ 总是 $\mathbb{R}^n$ 的子空间。

```
span 的直觉：两个向量张成的平面

    假设 v₁ = (1,0), v₂ = (0,1)
    
    v₂ ↑
       │        span{v₁, v₂} = 整个 ℝ² 平面
       │        你可以到达平面上的任意点：
       │        • c₁v₁ + c₂v₂ = (c₁, c₂)
       │        • 调整 c₁, c₂ —— 到处都是可达的
       │
       └──────────►
              v₁
    
    如果 v₁ = (1,0), v₂ = (2,0)（共线）:
    span{v₁, v₂} = 一条直线（x 轴）
    因为 v₂ 没有提供新方向，所有组合都落在 x 轴上
```

> [!abstract] 生成子空间
> 向量集合的**生成**是**最小**包含这些向量的子空间。它是构造子空间的基本方法。

## 5. 线性无关与线性相关

### 5.0 Why This Matters

线性无关是线性代数中"效率和独特性"的概念。一组向量中如果有冗余（某个是其他向量的组合），那么用它做基就会浪费维度。无关性确保了**最小的表示**和**唯一的坐标**——就像 GPS 定位中你只需要经度和纬度，不需要第三个冗余的方向。

### 5.1 定义

向量组 $\{\mathbf{v}_1, \dots, \mathbf{v}_k\}$ **线性无关**（linearly independent），如果：

$$
c_1\mathbf{v}_1 + c_2\mathbf{v}_2 + \cdots + c_k\mathbf{v}_k = \mathbf{0}
\implies c_1 = c_2 = \cdots = c_k = 0
$$

也就是说：任何一个向量都不能表示为其余向量的线性组合。

**线性相关**（linearly dependent）：存在不全为零的系数使得组合为零向量。

```
线性无关 vs 线性相关的几何对比

  线性无关（ℝ² 中）:               线性相关（ℝ² 中）:
  
  v₂ = (0,1) ↑                     v₂ = (2,0) →
             |                                
             |   / v₁ = (1,1)                 
             |  /                             
             | /                              
  ───────────┼────►                 ──────────►──────►
             |                               v₁ = (1,0)
             |                               v₂ = 2v₁（与 v₁ 同方向）
  
  两个向量指向不同方向               v₂ 在 v₁ 的 span 中
  谁都"够不着"对方                  v₂ 没有提供新方向
  没有冗余                        有冗余 → 线性相关
  
  关键测试：                         关键测试：
  a(1,1) + b(0,1) = (0,0)           a(1,0) + b(2,0) = (0,0)
  → a=0, b=0（唯一解）              → a=-2b，有非零解
  → 线性无关 ✓                      → 线性相关 ✗
```

### 5.2 几何直觉

| 维数 | 线性无关的几何条件 |
|------|------------------|
| $\mathbb{R}^2$ 中 2 个向量 | 不共线 |
| $\mathbb{R}^3$ 中 3 个向量 | 不共面 |
| 任意空间 | $k > n$ 个 $n$ 维向量**必然**线性相关 |

> [!warning]
> 如果向量个数 > 向量维数，则必然线性相关。$n+1$ 个 $n$ 维向量一定存在冗余。道理很简单：$n$ 维空间最多只有 $n$ 个独立的"方向"，第 $n+1$ 个向量必然可以被前 $n$ 个组合出来。

## 6. 基（Basis）

### 6.0 Why Basis Matters

基回答了一个根本问题：**在这个空间中，最少需要几个"基本方向"才能描述所有点？** 基就是一组"刚好够用"的方向——不多不少。一旦选定了基，空间中的每个向量都有一个唯一的地址（坐标）。

### 6.1 定义

$B = \{\mathbf{b}_1, \dots, \mathbf{b}_d\}$ 是向量空间 $V$ 的**基**（basis），如果：
1. $B$ **线性无关**（没有冗余）
2. $B$ **生成** $V$：$\text{span}(B) = V$（足够到达所有地方）

```
基的直觉：最经济的导航系统

  ℝ² 中选基 B = {(1,0), (0,1)}:
  
  任意向量 (x,y) = x(1,0) + y(0,1)
  → 坐标 [ (x,y) ]_B = (x, y)
  
  ──── 这就像城市的街道网格 ────
  • 经度方向 (1,0) ── 一个独立方向
  • 纬度方向 (0,1) ── 另一个独立方向
  • 每个地址 (x,y) 唯一确定
  • 不能去掉任何一个（否则到达不了所有地方）
  • 不需要第三个方向（否则就是冗余）
```

### 6.2 标准基

| 空间 | 标准基 |
|------|--------|
| $\mathbb{R}^n$ | $\{\mathbf{e}_1, \dots, \mathbf{e}_n\}$，$\mathbf{e}_i = (0,\dots,1,\dots,0)$ |
| $\mathbb{R}^{2 \times 2}$ | $\{\begin{bmatrix}1&0\\0&0\end{bmatrix}, \begin{bmatrix}0&1\\0&0\end{bmatrix}, \begin{bmatrix}0&0\\1&0\end{bmatrix}, \begin{bmatrix}0&0\\0&1\end{bmatrix}\}$ |
| $\mathcal{P}_2$ | $\{1, x, x^2\}$ |

### 6.3 基的重要性质

- 一个向量空间可以有多个不同的基（就像城市可以用不同的方式定义"基本方向"）
- 所有基的**元素个数相同**（即维数）——这是一个深刻的结论
- 基是"**最经济的生成集**"：去掉任一元素就不足以生成整个空间
- 基也是"**最大的线性无关集**"：再加入任一向量就会变得相关

## 7. 维数（Dimension）

### 7.0 Why Dimension Matters

维数是向量空间最重要的单一数值——它告诉你**需要多少个独立参数才能唯一确定空间中的一个点**。$\mathbb{R}^3$ 需要 3 个坐标，$\mathcal{P}_2$（二次多项式）需要 3 个系数，$2\times 2$ 矩阵需要 4 个条目。维数是判定两个空间是否"一样大"的标尺。

### 7.1 定义

向量空间 $V$ 的维数 $\dim(V)$ 等于其基中向量的个数。

### 7.2 基本结论

- $\dim(\mathbb{R}^n) = n$
- $\dim(\mathbb{R}^{m \times n}) = mn$
- $\dim(\mathcal{P}_n) = n + 1$
- 零空间 $\{\mathbf{0}\}$ 的维数为 0

> **为什么零空间维数是 0？** 零空间中只有一个元素 $\mathbf{0}$。它的基是空集 $\{\}$（因为任何非零向量都不能出现在基中——它们都不在零空间里）。空集的元素个数是 0，所以维数为 0。这也符合直觉：零空间没有任何自由度——你唯一能"选"的就是零本身。

### 7.3 坐标（Coordinates）

选定了基 $B = \{\mathbf{b}_1, \dots, \mathbf{b}_n\}$ 后，任意向量 $\mathbf{v} \in V$ 可以唯一地表示为：

$$
\mathbf{v} = c_1\mathbf{b}_1 + c_2\mathbf{b}_2 + \cdots + c_n\mathbf{b}_n
$$

系数向量 $[\mathbf{v}]_B = (c_1, c_2, \dots, c_n)$ 称为 $\mathbf{v}$ 在基 $B$ 下的**坐标**（coordinates）。

```
坐标的直觉：

  同一个向量在不同基下有不同的"地址"。
  
  标准基 B₁ = {(1,0), (0,1)}:
  向量 v = (3,2) → [v]_{B₁} = (3, 2)
  
  另一组基 B₂ = {(1,1), (1,-1)}:
  同一个向量 v = (3,2) → [v]_{B₂} = (2.5, 0.5)
  因为 2.5(1,1) + 0.5(1,-1) = (3,2)
  
  向量本身没变——只是描述它的"语言"变了。
```

> [!tip] 基的选择很重要
> 同一个向量在不同基下的坐标不同——这正是 [[Diagonalization]] 中 $\mathbf{P}$ 矩阵在做的事情：将特征向量基下的坐标转换到标准基。

## 8. 子空间与基的经典结论

- 如果 $\dim(V) = n$，则 $V$ 中任意 $n$ 个线性无关的向量构成一组基
- 如果 $\dim(V) = n$，则 $V$ 中任意 $n$ 个生成 $V$ 的向量构成一组基
- 子空间的维数 $\le$ 母空间的维数
- $\dim(\text{span}\{\mathbf{v}_1,\dots,\mathbf{v}_k\})$ = $\text{rank}([\mathbf{v}_1 \; \cdots \; \mathbf{v}_k])$

## 9. 概念拓扑图

```
向量空间（有加法和数乘的"舞台"）
   │
   ├── 基本性质（零向量唯一、负元唯一、0·v=0 等）
   │
   ├── 子空间（舞台内的子舞台，本身也是向量空间）
   │     ├── 零空间 Null(A)——所有解构成的子空间
   │     ├── 列空间 Col(A)——所有像构成的子空间
   │     ├── 子空间的和——从"积木"构造更大空间
   │     └── 直和——无重叠的分解，零向量表示唯一
   │
   ├── span（从几个向量"张出"子空间）
   │
   ├── 线性无关（没有冗余——基的核心条件）
   │     └── 线性相关（有冗余——存在向量能被其他组合出来）
   │
   ├── 基 = 线性无关 + 生成（最经济的描述系统）
   │     └── 坐标（选定基后每个向量的"地址"）
   │
   └── 维数 = 基中元素个数（自由度的度量）
```

这些概念贯穿线性代数始终，是理解 [[Rank and Nullity]]、[[Linear Transformations]] 和 [[Eigenvalues and Eigenvectors]] 的基础。

## 10. 在 NumPy 中的体现

```python
import numpy as np

# 检验线性无关性
v1 = np.array([1, 2, 3])
v2 = np.array([4, 5, 6])
v3 = np.array([7, 8, 9])

A = np.column_stack([v1, v2, v3])
rank = np.linalg.matrix_rank(A)
print(f"矩阵的秩 = {rank}，3个向量{'线性无关' if rank == 3 else '线性相关'}")

# 检验一组向量是否构成 R^3 的基
B = np.column_stack([np.array([1,0,0]),
                     np.array([1,1,0]),
                     np.array([1,1,1])])
det_B = np.linalg.det(B)
print(f"行列式 = {det_B:.2f}，是{'一组基' if abs(det_B) > 1e-10 else '不是基'}")

# 求 span 的维数
vectors = np.random.randn(3, 5)  # 5 个 R^3 中的向量
span_dim = np.linalg.matrix_rank(vectors)
print(f"5 个 R^3 向量的 span 的维数 = {span_dim}（必有冗余）")
```
