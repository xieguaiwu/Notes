# Ellipse
好的，我们来系统性地梳理关于椭圆方程的基本知识。这是理解和应用椭圆的核心。

## 1. 核心定义

椭圆是由一个**几何定义**推导出的曲线：

**在平面内，到两个定点（称为焦点 $F_1$ 和 $F_2$）的距离之和等于常数（大于两焦点间距离）的点的集合。**

- 设两焦点距离为 $2c$，常数和为 $2a$，则必须满足 $2a > 2c$，即 $a > c > 0$

---

## 2. 标准方程 (Standard Equation)

这是最基本、最重要的形式。我们通常将椭圆放置于一个对称的位置，以便推导出简洁的方程。

## 情况一：焦点在 x 轴上 (Major Axis Horizontal)

当椭圆中心在原点 (0, 0)，焦点位于 x 轴上时，其标准方程为：

**$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$**

其中：
- $a > b > 0$
- $c^2 = a^2 - b^2$ （非常重要的关系式）
- **焦点坐标**: $F_1(-c, 0)$, $F_2(c, 0)$
- **顶点坐标** (与坐标轴的交点): $(\pm a, 0)$, $(0, \pm b)$
- **长轴**长度为 2a$，**短轴**长度为 $2b$



## 情况二：焦点在 y 轴上 (Major Axis Vertical)

当椭圆中心在原点 (0, 0)，焦点位于 y 轴上时，其标准方程为：

**$\frac{x^2}{b^2} + \frac{y^2}{a^2} = 1$**

其中：
- $a > b > 0$
- $c^2 = a^2 - b^2$ （关系式不变）
- **焦点坐标**: $F_1(0, -c)$, $F_2(0, c)$
- **顶点坐标**: $(0, \pm a)$, $(\pm b, 0)$
- **长轴**长度为 $2a$，**短轴**长度为 $2b$



**如何快速判断？**
**看分母**：分母较大的那个变量所在的轴，就是**长轴**所在的位置，也就是**焦点**所在的轴。

---

## 3. 关键参数与性质

对于标准方程 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ 或 $\frac{x^2}{b^2} + \frac{y^2}{a^2} = 1$ (其中 $a > b$)：

| 参数      | 符号/公式             | 描述                               |
| :------ | :---------------- | :------------------------------- |
| **中心**  | $(0, 0)$          | 椭圆的对称中心。                         |
| **半长轴** | $a$               | 长轴的一半，是椭圆中心到较远顶点的距离。             |
| **半短轴** | $b$               | 短轴的一半，是椭圆中心到较近顶点的距离。             |
| **焦距**  | $c$               | 从中心到任一焦点的距离。                     |
| **关系式** | $a² = b² + c²$    | 连接 a, b, c 的核心公式。                |
| **离心率** | $e = \frac{c}{a}$ | 描述椭圆的“扁平程度”。                     |
|         |                   | 范围：$0 < e < 1$                   |
|         |                   | $e$ 越接近 $0$，越像圆；$e$ 越接近 $1$，越扁平。 |

---

## 4. 中心不在原点的椭圆方程

如果椭圆的中心在点 (h, k)，而长轴平行于坐标轴，我们只需将标准方程进行**平移**。

- **长轴平行于 x 轴**：
  $\frac{(x-h)^2}{a^2} + \frac{(y-k)^2}{b^2} = 1$
  焦点: $(h\pm c, k)$
  顶点: $(h\pm a, k)$

- **长轴平行于 y 轴**：
  $\frac{(x-h)^2}{b^2} + \frac{(y-k)^2}{a^2} = 1$
  焦点: $(h, k\pm c)$
  顶点: $(h, k\pm a)$

**记忆技巧**：对照标准方程，将 $x$ 替换为 $(x-h)$，将 $y$ 替换为 $(y-k)$

---

## 5. 一般方程 (General Equation)

椭圆方程也可以表示为二元二次方程的一般形式：

**$Ax^2 + Cy^2 + Dx + Ey + F = 0$**

其中 $A$, $C$ **同号**且 $A \neq C$
(如果 $A = C > 0$，并且满足特定条件，它表示一个圆)。

**如何从一般式化为标准式？**
使用**配方法 (Completing the Square)**。
1.  将含有 $x$ 的项和含有 $y$ 的项分别分组。
2.  对 $x$ 项和 $y$ 项分别进行配方。
3.  将常数项移到等号右边。
4.  两边同时除以这个常数，使右边等于 $1$。

**示例**：将 $4x^2 + 9y^2 - 16x + 18y - 11 = 0$ 化为标准式。
1.  分组: $(4x^2 - 16x) + (9y^2 + 18y) - 11 = 0$
2.  提系数: $4(x^2 - 4x) + 9(y^2 + 2y) - 11 = 0$
3.  配方: $4[(x-2)^2 - 4] + 9[(y+1)^2 - 1] - 11 = 0$
4.  展开: $4(x-2)^2 - 16 + 9(y+1)^2 - 9 - 11 = 0$
5.  整理: $4(x-2)^2 + 9(y+1)^2 = 36$
6.  除以 36: $\frac{(x-2)^2}{9} + \frac{(y+1)^2}{4} = 1$

现在我们得到中心在 $(2, -1)$，$a=3$, $b=2$ 的椭圆。

## 总结

| 特征       | 焦点在 x 轴                                 | 焦点在 y 轴                                 |
| :------- | :-------------------------------------- | :-------------------------------------- |
| **标准方程** | $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ | $\frac{x^2}{b^2} + \frac{y^2}{a^2} = 1$ |
| **中心**   | $(0, 0)$                                | (0, 0)                                  |
| **焦点**   | $(\pm c, 0)$                            | $(0, \pm c)$                            |
| **顶点**   | $(\pm a, 0)$                            | $(0, \pm a)$                            |
| **关系**   | $c^2 = a^2 - b^2$                       | $c^2 = a^2 - b^2$                       |
| **长轴**   | 沿 $x$ 轴，长度 $2a$                         | 沿 $y$ 轴，长度 $2a$                         |
| **离心率**  | $e = \frac{c}{a} (0<e<1)$               | $e = \frac{c}{a} (0<e<1)$               |
