# 向量运算与向量方程求导

## 1. 向量基本概念

[[Vector and Matrix]]
### 1.1 向量定义
向量是既有大小又有方向的量，通常用箭头表示或用粗体字母表示。在二维平面中，向量可以表示为：
$$\vec{v} = \langle v_1, v_2 \rangle \quad \text{或} \quad \vec{v} = v_1\mathbf{i} + v_2\mathbf{j}$$

在三维空间中，向量可以表示为：
$$\vec{v} = \langle v_1, v_2, v_3 \rangle \quad \text{或} \quad \vec{v} = v_1\mathbf{i} + v_2\mathbf{j} + v_3\mathbf{k}$$

其中，$\mathbf{i} = \langle 1, 0 \rangle$、$\mathbf{j} = \langle 0, 1 \rangle$ 是二维标准单位向量；$\mathbf{k} = \langle 0, 0, 1 \rangle$ 是三维标准单位向量。

### 1.2 向量的模（长度）
向量的模表示向量的大小，记为 $|\vec{v}|$。

**二维向量**：
$$|\vec{v}| = \sqrt{v_1^2 + v_2^2}$$

**三维向量**：
$$|\vec{v}| = \sqrt{v_1^2 + v_2^2 + v_3^2}$$

### 1.3 单位向量
模为1的向量称为单位向量。向量 $\vec{v}$ 的单位向量为：
$$\hat{v} = \frac{\vec{v}}{|\vec{v}|}$$

## 2. 向量运算

### 2.1 向量的加法与减法

**加法**：两个向量 $\vec{a} = \langle a_1, a_2 \rangle$ 和 $\vec{b} = \langle b_1, b_2 \rangle$ 的和为：
$$\vec{a} + \vec{b} = \langle a_1 + b_1, a_2 + b_2 \rangle$$

**减法**：
$$\vec{a} - \vec{b} = \langle a_1 - b_1, a_2 - b_2 \rangle$$

### 2.2 向量的数乘

向量 $\vec{v}$ 与标量 $k$ 的乘积为：
$$k\vec{v} = \langle kv_1, kv_2 \rangle$$

**性质**：
- 当 $k > 0$ 时，方向不变
- 当 $k < 0$ 时，方向相反
- 当 $k = 0$ 时，得到零向量 $\vec{0}$

### 2.3 向量的点积（数量积）

**定义**：两个向量 $\vec{a} = \langle a_1, a_2 \rangle$ 和 $\vec{b} = \langle b_1, b_2 \rangle$ 的点积为：
$$\vec{a} \cdot \vec{b} = a_1b_1 + a_2b_2$$

**几何意义**：
$$\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos\theta$$

其中 $\theta$ 是两向量的夹角。

**应用**：
- 判断垂直：$\vec{a} \perp \vec{b} \iff \vec{a} \cdot \vec{b} = 0$
- 计算夹角：$\cos\theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}||\vec{b}|}$

### 2.4 向量的叉积（向量积）

**定义**（仅限三维）：
$$\vec{a} \times \vec{b} = \langle a_2b_3 - a_3b_2, a_3b_1 - a_1b_3, a_1b_2 - a_2b_1 \rangle$$

**几何意义**：$\vec{a} \times \vec{b}$ 是一个垂直于 $\vec{a}$ 和 $\vec{b}$ 的向量，其大小等于两向量构成的平行四边形面积。

**应用**：
- 判断平行：$\vec{a} \parallel \vec{b} \iff \vec{a} \times \vec{b} = \vec{0}$

## 3. 向量函数

### 3.1 向量函数定义
向量函数是一个函数，其输出值为向量：
$$\vec{r}(t) = \langle f_1(t), f_2(t), f_3(t) \rangle \quad \text{或} \quad \vec{r}(t) = f_1(t)\mathbf{i} + f_2(t)\mathbf{j} + f_3(t)\mathbf{k}$$

其中 $t$ 是参数，通常是时间。

### 3.2 位置向量、速度向量、加速度向量

- **位置向量**：$\vec{r}(t)$ 表示质点在时刻 $t$ 的位置
- **速度向量**：$\vec{v}(t) = \vec{r}'(t)$ 表示质点在时刻 $t$ 的速度
- **加速度向量**：$\vec{a}(t) = \vec{v}'(t) = \vec{r}''(t)$ 表示质点在时刻 $t$ 的加速度

## 4. 向量函数的求导

### 4.1 向量函数的一阶导数
[[differentiation]]

向量函数 $\vec{r}(t) = \langle f_1(t), f_2(t), f_3(t) \rangle$ 的导数为：
$$\vec{r}'(t) = \langle f_1'(t), f_2'(t), f_3'(t) \rangle$$

**例子**：$\vec{r}(t) = \langle t^2, \sin t, e^t \rangle$
$$\vec{r}'(t) = \langle 2t, \cos t, e^t \rangle$$

### 4.2 向量函数的二阶导数
[[Higher Order Derivative]]

$$\vec{r}''(t) = \langle f_1''(t), f_2''(t), f_3''(t) \rangle$$

**例子**：$\vec{r}(t) = \langle t^2, \sin t, e^t \rangle$
$$\vec{r}''(t) = \langle 2, -\sin t, e^t \rangle$$

### 4.3 向量导数的物理意义

| 物理量 | 向量表达式 | 标量表达式（大小） |
|--------|-----------|------------------|
| 位置 | $\vec{r}(t)$ | $|\vec{r}(t)|$ |
| 速度 | $\vec{v}(t) = \vec{r}'(t)$ | $|\vec{v}(t)| = \sqrt{(x')^2 + (y')^2 + (z')^2}$ |
| 加速度 | $\vec{a}(t) = \vec{v}'(t) = \vec{r}''(t)$ | $|\vec{a}(t)| = \sqrt{(x'')^2 + (y'')^2 + (z'')^2}$ |

### 4.4 向量函数的导数运算法则

**数乘**：$\frac{d}{dt}[k\vec{r}(t)] = k\vec{r}'(t)$

**加法**：$\frac{d}{dt}[\vec{r}_1(t) + \vec{r}_2(t)] = \vec{r}_1'(t) + \vec{r}_2'(t)$

**点积**：$\frac{d}{dt}[\vec{r}_1(t) \cdot \vec{r}_2(t)] = \vec{r}_1'(t) \cdot \vec{r}_2(t) + \vec{r}_1(t) \cdot \vec{r}_2'(t)$

**叉积**：$\frac{d}{dt}[\vec{r}_1(t) \times \vec{r}_2(t)] = \vec{r}_1'(t) \times \vec{r}_2(t) + \vec{r}_1(t) \times \vec{r}_2'(t)$

## 5. 向量函数的积分
[[Indefinite Integration]]

### 5.1 向量函数的不定积分

$$\int \vec{r}(t) \, dt = \left\langle \int f_1(t) \, dt, \int f_2(t) \, dt, \int f_3(t) \, dt \right\rangle + \vec{C}$$

其中 $\vec{C}$ 是常向量。

### 5.2 向量函数的定积分
[[Definite Integrals]]

$$\int_a^b \vec{r}(t) \, dt = \left\langle \int_a^b f_1(t) \, dt, \int_a^b f_2(t) \, dt, \int_a^b f_3(t) \, dt \right\rangle$$

## 6. 弧长与弧长参数

### 6.1 弧长公式

向量函数 $\vec{r}(t)$ 在 $t \in [a, b]$ 上的弧长为：
$$L = \int_a^b |\vec{r}'(t)| \, dt = \int_a^b \sqrt{[f_1'(t)]^2 + [f_2'(t)]^2 + [f_3'(t)]^2} \, dt$$

### 6.2 弧长参数

弧长参数 $s$ 定义为：
$$s(t) = \int_{t_0}^t |\vec{r}'(u)| \, du$$

**单位切向量**：
$$\vec{T}(t) = \frac{\vec{r}'(t)}{|\vec{r}'(t)|}$$

## 9. 典型例题

### 9.1 求向量函数的导数

**题目**：求 $\vec{r}(t) = \langle t^2, 2t, t^3 \rangle$ 的速度和加速度。

**解**：
- 速度：$\vec{v}(t) = \vec{r}'(t) = \langle 2t, 2, 3t^2 \rangle$
- 加速度：$\vec{a}(t) = \vec{v}'(t) = \langle 2, 0, 6t \rangle$

### 9.2 计算弧长

**题目**：计算 $\vec{r}(t) = \langle \cos t, \sin t \rangle$ 在 $t \in [0, 2\pi]$ 上的弧长。

**解**：
- $\vec{r}'(t) = \langle -\sin t, \cos t \rangle$
- $|\vec{r}'(t)| = \sqrt{\sin^2 t + \cos^2 t} = 1$
- $L = \int_0^{2\pi} 1 \, dt = 2\pi$

### 9.3 物理应用

**题目**：质点的位置函数为 $\vec{r}(t) = \langle 3\cos t, 3\sin t, 4t \rangle$，求速度的大小和加速度的大小。

**解**：
- 速度：$\vec{v}(t) = \langle -3\sin t, 3\cos t, 4 \rangle$
- 速度大小：$|\vec{v}(t)| = \sqrt{9\sin^2 t + 9\cos^2 t + 16} = \sqrt{9 + 16} = 5$
- 加速度：$\vec{a}(t) = \langle -3\cos t, -3\sin t, 0 \rangle$
- 加速度大小：$|\vec{a}(t)| = \sqrt{9\cos^2 t + 9\sin^2 t} = 3$
