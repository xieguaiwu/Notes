# 极坐标求导与积分

## 一、基本定义

### 极坐标系

点 $P$ 由有序对 $(r, \theta)$ 表示：
- $r$：极径，点 $P$ 到极点（原点）的距离
- $\theta$：极角，从极轴（正 x 轴）逆时针旋转到 $OP$ 的角度

### 极坐标与直角坐标转换

$$x = r\cos\theta, \quad y = r\sin\theta$$

$$r = \sqrt{x^2 + y^2}, \quad \tan\theta = \frac{y}{x}$$

### 注意事项

- $r$ 可以为负：点 $(-r, \theta)$ 与 $(r, \theta + \pi)$ 表示同一点
- 同一点有多种表示：$(r, \theta) = (r, \theta + 2k\pi) = (-r, \theta + (2k+1)\pi)$

---

## 二、极坐标下的曲线描绘

### 描点法步骤

1. 列出关键 $\theta$ 值（通常从 $0$ 到 $2\pi$）
2. 计算对应的 $r$ 值
3. 在极坐标系中标出点并连接

### 对称性判断

| 条件 | 对称性 |
|:---:|:---:|
| $r(\theta) = r(-\theta)$ 或 $r(\theta) = r(\pi - \theta)$ | 关于极轴对称 |
| $r(\theta) = r(\pi - \theta)$ 或 $r(\theta) = r(-\theta)$ | 关于 $\theta = \frac{\pi}{2}$ 对称 |
| $r(\theta) = r(\theta + \pi)$ 或 $r(\theta) = -r(\theta)$ | 关于极点对称 |

---

## 三、极坐标曲线的切线斜率

### 斜率公式

极坐标曲线 $r = r(\theta)$ 在点 $(r, \theta)$ 处的切线斜率：

$$\frac{dy}{dx} = \frac{\frac{dr}{d\theta}\sin\theta + r\cos\theta}{\frac{dr}{d\theta}\cos\theta - r\sin\theta}$$

**推导**：由 $x = r\cos\theta$，$y = r\sin\theta$，使用链式法则：

$$\frac{dy}{dx} = \frac{dy/d\theta}{dx/d\theta}$$

其中：
$$\frac{dx}{d\theta} = \frac{dr}{d\theta}\cos\theta - r\sin\theta$$
$$\frac{dy}{d\theta} = \frac{dr}{d\theta}\sin\theta + r\cos\theta$$

### 水平切线与垂直切线

- **水平切线**：$\frac{dy}{d\theta} = 0$ 且 $\frac{dx}{d\theta} \neq 0$
- **垂直切线**：$\frac{dx}{d\theta} = 0$ 且 $\frac{dy}{d\theta} \neq 0$

---

## 四、极坐标曲线的弧长

### 弧长公式

极坐标曲线 $r = r(\theta)$ 从 $\theta = \alpha$ 到 $\theta = \beta$ 的弧长：

$$L = \int_{\alpha}^{\beta} \sqrt{r^2 + \left(\frac{dr}{d\theta}\right)^2}\, d\theta$$

**推导**：
$$ds = \sqrt{(dx)^2 + (dy)^2} = \sqrt{\left(\frac{dx}{d\theta}\right)^2 + \left(\frac{dy}{d\theta}\right)^2}\, d\theta$$

化简得：
$$\left(\frac{dx}{d\theta}\right)^2 + \left(\frac{dy}{d\theta}\right)^2 = r^2 + \left(\frac{dr}{d\theta}\right)^2$$

---

## 五、极坐标曲线围成的面积

### 面积公式

极坐标曲线 $r = r(\theta)$ 与射线 $\theta = \alpha$、$\theta = \beta$ 围成的区域面积：

$$A = \frac{1}{2}\int_{\alpha}^{\beta} [r(\theta)]^2\, d\theta$$

**理解**：面积微元 $dA = \frac{1}{2}r^2\, d\theta$ 是半径为 $r$、圆心角为 $d\theta$ 的扇形面积。

### 两曲线围成的面积

两极坐标曲线 $r_1(\theta)$ 和 $r_2(\theta)$（$r_1 \geq r_2$）围成的面积：

$$A = \frac{1}{2}\int_{\alpha}^{\beta} \left[r_1^2(\theta) - r_2^2(\theta)\right]\, d\theta$$

### 注意事项

- 确定正确的积分上下限（两曲线的交点）
- 若曲线有对称性，可只计算部分面积再乘以倍数

## 六、常用极坐标方程

### 直线

| 方程 | 形式 |
|:---:|:---:|
| 过极点，倾角为 $\alpha$ | $\theta = \alpha$ |
| 与极轴垂直，距离为 $a$ | $r\cos\theta = a$ |
| 与极轴平行，距离为 $a$ | $r\sin\theta = a$ |

### 圆

| 方程 | 圆心 | 半径 |
|:---:|:---:|:---:|
| $r = a$ | 极点 | $a$ |
| $r = 2a\cos\theta$ | $(a, 0)$ | $a$ |
| $r = 2a\sin\theta$ | $(0, a)$ | $a$ |

**面积**：$A = \pi a^2$

**周长**：$L = 2\pi a$

### 玫瑰曲线

$$r = a\sin(n\theta) \quad \text{或} \quad r = a\cos(n\theta)$$

- $n$ 为奇数：$n$ 片花瓣
- $n$ 为偶数：$2n$ 片花瓣
- 花瓣长度为 $|a|$

**面积**：

- 一片花瓣面积：$A_{\text{petal}} = \frac{\pi a^2}{4n}$
- 总面积：
  - $n$ 为奇数：$A = \frac{\pi a^2}{4}$
  - $n$ 为偶数：$A = \frac{\pi a^2}{2}$

**面积推导**（以 $r = a\cos(n\theta)$ 为例）：

一片花瓣的积分区间为 $\theta \in [-\frac{\pi}{2n}, \frac{\pi}{2n}]$：

$$A_{\text{petal}} = \frac{1}{2}\int_{-\pi/(2n)}^{\pi/(2n)} a^2\cos^2(n\theta)\,d\theta$$

令 $u = n\theta$：

$$= \frac{a^2}{2n}\int_{-\pi/2}^{\pi/2}\cos^2 u\,du = \frac{a^2}{2n} \cdot \frac{\pi}{2} = \frac{\pi a^2}{4n}$$

### 利莫尼曲线（Limaçon）

$$r = a \pm b\sin\theta \quad \text{或} \quad r = a \pm b\cos\theta$$

| 条件 | 形状 |
|:---:|:---:|
| $\frac{a}{b} > 1$ | 凸形（无内环） |
| $\frac{a}{b} = 1$ | 心形线（Cardioid） |
| $\frac{a}{b} < 1$ | 有内环 |

**面积**：

$$A = \frac{1}{2}\int_0^{2\pi} (a + b\cos\theta)^2\,d\theta = \pi a^2 + \frac{\pi b^2}{2} = \frac{\pi}{2}(2a^2 + b^2)$$

**心形线** $r = a(1 + \cos\theta)$ 的特殊结果：

- 面积：$A = \frac{3\pi a^2}{2}$（代入 $a = b$）
- 周长：$L = 8a$

### 螺线

**阿基米德螺线**：$r = a\theta$

- 等间距螺线，相邻两圈间距为 $2\pi a$

**弧长**（从 $\theta = 0$ 到 $\theta = \alpha$）：

$$L = \frac{a}{2}\left[\alpha\sqrt{1+\alpha^2} + \ln\left(\alpha + \sqrt{1+\alpha^2}\right)\right]$$

**面积**（从 $\theta = 0$ 到 $\theta = \alpha$）：

$$A = \frac{a^2\alpha^3}{6}$$

**对数螺线**：$r = ae^{b\theta}$

**弧长**（从 $\theta = \alpha$ 到 $\theta = \beta$）：

$$L = \frac{a\sqrt{1+b^2}}{b}(e^{b\beta} - e^{b\alpha})$$

### 双纽线

$$r^2 = a^2\cos(2\theta) \quad \text{或} \quad r^2 = a^2\sin(2\theta)$$

- 形状类似无穷大符号 $\infty$
- 积分区间：$\theta \in [-\frac{\pi}{4}, \frac{\pi}{4}]$（一片叶）

**面积**：

$$A = 2 \cdot \frac{1}{2}\int_{-\pi/4}^{\pi/4} a^2\cos(2\theta)\,d\theta = a^2\left[\frac{\sin(2\theta)}{2}\right]_{-\pi/4}^{\pi/4} = a^2$$

**弧长**（涉及椭圆积分，AP微积分不要求）：

$$L = 4a \cdot E\left(\frac{1}{\sqrt{2}}\right) \approx 5.244a$$

其中 $E(k)$ 为第二类完全椭圆积分。
