# series_problem_05.08

## Q1

> 级数 $\displaystyle\sum_{n=1}^{\infty} \frac{1}{n^{2p}+n}$ 对于哪些 $p$ 来说是发散的？

**不能直接套用 p-级数判别法！** 分母中有两项竞争（$n^{2p}$ 与 $n$），必须根据 $p$ 的大小判断主导项，再选择合适的比较对象。

### 当 $p \leq \frac{1}{2}$ 时（发散）

此时 $2p \leq 1$，对 $n \geq 1$ 有 $n^{2p} \leq n$，因此：

$$n^{2p} + n \leq n + n = 2n$$

取倒数（不等式反向）：

$$\frac{1}{n^{2p}+n} \geq \frac{1}{2n}$$

由于 $\sum \frac{1}{2n} = \frac{1}{2}\sum \frac{1}{n}$ 发散（调和级数），由**直接比较判别法**，原级数发散。

### 当 $p > \frac{1}{2}$ 时（收敛）

此时 $2p > 1$，$n^{2p}$ 随 $n$ 增长更快，有：

$$\frac{1}{n^{2p}+n} < \frac{1}{n^{2p}}$$

由于 $2p > 1$，$\sum \frac{1}{n^{2p}}$ 收敛（p-级数），由直接比较判别法，原级数收敛。

**答案**：当且仅当 $p \leq \dfrac{1}{2}$ 时发散。

### 易错点

1. **拿 $\sum \frac{1}{n^{2p}}$ 直接判收敛**：错误论证 — 因为 $\frac{1}{n^{2p}+n} < \frac{1}{n^{2p}}$，若 $\sum \frac{1}{n^{2p}}$ 发散（$2p \leq 1$），能否推出原级数发散？**不能！** 直接比较判别法单向有效：比发散级数小的级数，无法判定。此时需要从另一边放大分母来获得有效下界。

2. **忽略了 $p$ 导致的"主导项切换"**：当 $p \leq \frac{1}{2}$ 时 $n$ 主导分母，当 $p > \frac{1}{2}$ 时 $n^{2p}$ 主导。若不加分情况讨论，直接断言 $\frac{1}{n^{2p}+n} \sim \frac{1}{n^{2p}}$（对所有 $p$），对 $p \leq \frac{1}{2}$ 会得出错误结论。

3. **遗漏边界 $p = \frac{1}{2}$**：此时 $n^{2p} = n$，通项为 $\frac{1}{2n}$，级数发散。若在 $p > \frac{1}{2}$ 收敛的结论中忘记把等号归入发散一侧，会漏掉临界情况。

## Q2

> 幂级数 $\displaystyle\sum_{n=1}^{\infty} \frac{(x-5)^n}{2^n n^2}$ 的收敛半径为 $2$。对于哪些 $x$，可以使用**交错级数判别法**来判定该幂级数的收敛性？

**收敛区间 ≠ AST 适用范围。** 收敛区间为 $[3, 7]$，但 AST 只能用于级数正负交替的情况。

### 分析

AST 适用需要三个条件同时满足：
1. 通项正负交替
2. 绝对值最终递减
3. 绝对值极限为零

**条件 1（正负交替）**：通项 $\frac{(x-5)^n}{2^n n^2}$ 的正负取决于 $(x-5)^n$。要使 $(x-5)^n$ 随 $n$ 交替变号，必须 $x-5 < 0$，即 $x < 5$。

当 $x < 5$ 时，令 $x-5 = -a\;(a > 0)$：

$$\frac{(x-5)^n}{2^n n^2} = \frac{(-a)^n}{2^n n^2} = (-1)^n \frac{a^n}{2^n n^2} = (-1)^n b_n,\quad b_n = \frac{r^n}{n^2},\; r = \frac{a}{2} = \frac{|x-5|}{2}$$

**条件 2 & 3（递减与极限为零）**：

- 若 $r < 1\;(|x-5| < 2)$：$b_n \to 0$，且 $\frac{b_{n+1}}{b_n} = r \cdot \frac{n^2}{(n+1)^2} \to r < 1$，$b_n$ 最终递减 → AST 适用 ✓
- 若 $r = 1\;(x = 3)$：$b_n = \frac{1}{n^2}$，显然递减且趋于零 → AST 适用 ✓
- 若 $r > 1\;(x < 3)$：$b_n \to \infty$，条件 2、3 均不满足 → AST 不适用 ✗

**交集**：$\{x \mid x < 5\} \cap \{x \mid |x-5| \leq 2\} = [3, 5)$

**答案**：$x \in [3, 5)$

### 易错点

1. **把收敛区间当答案**：收敛区间是 $[3, 7]$，但 $x \in (5, 7]$ 时级数全为正项，根本不是交错级数。**AST 只能用于级数正负交替的情况**，连基本前提都不满足，其他条件无需讨论。

2. **忘记检验 $\lim b_n = 0$**：对 $x < 3$（即 $|x-5| > 2$），虽然 $x < 5$ 使得级数形式上交替，但 $b_n = r^n/n^2$ 中 $r > 1$，$b_n \to \infty$。AST 的极限条件不成立，不能套用。

3. **端点 $x=3$ 与 $x=7$ 区别对待**：
   - $x=3$：$b_n = 1/n^2$，递减且趋于零，且 $(-1)^n$ 交替 → AST 适用
   - $x=7$：全正项 $\sum 1/n^2$，不是交错级数 → AST 不适用
   两端点结论相反，不能因为 "都在收敛区间内" 就同等对待。

4. **"能用 AST" ≠ "级数收敛"**：题目问的是能否用 AST 来检验，而非级数是否收敛。对 $x<3$，级数发散，AST 也不能用来"检验"其收敛性（前提条件不满足）。
