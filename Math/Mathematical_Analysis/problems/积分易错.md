# 积分易错
## 1.
对于形如下方积分的积分
>$$\int\frac{1}{3-x}dx$$

在使用换元法时**必须注意符号**
设$u=3-x$，则$du=-dx$，那么原式变形为：$$\int\frac{1}{u}\cdot(-du)$$
$$=-\int\frac{1}{u}du=-\ln|u|+C=-\ln|3-x|+C$$

除此之外，必须记得在积分后**为$\ln$加绝对值**！

## 2.
>对于$$\int^b_a f(x)dx=\int_a^c f(x)dx+\int_c^b f(x)dx$$

**$c$不一定属于$[a,b]$**

## 3. 微积分基本定理I
>计算$$\frac{d}{dx}\int_4^{3x^2+1}\sqrt{e^t+t}\text{ dt}$$

需要将$3x^2+1$**代入后再求导**：
$$=\sqrt{e^{3x^2+1}+3x^2+1}\cdot6x$$

## 4. 上下界
>定理：$$m(b-a)\leq\int_a^bf(x)dx\leq M(b-a)$$

其中$m$是lower bound，而$M$是upper bound [[Indefinite Integration]]
用这种方式可以在应用题中**快速判断函数值范围**

## 5. Improper Integration
不仅仅是上限和下限中包括了无穷的积分是improper integration，只要**上下限的区间内包含了被积函数定义域中不存在的取值**都是improper integration

对于所有的improper integration：

### 5.1
>$$\int_{-1}^1\frac{1}{x} dx$$

应该写成：$$\lim_{a\to 0^-} \int^a_{-1}\frac{1}{x} dx + \lim_{b\to 0^+} \int^1_b\frac{1}{x} dx$$

### 5.2
>$$\int_{-\infty}^\infty f(x) dx$$

应该写成$$\int_{-\infty}^{C} f(x) dx + \int_{C}^{\infty} f(x) dx$$

## 6. 积分配方法
目标：将一般二次式 $ax^2 + bx + c$ 变形为更易积分的形式。

配方三步骤

1. 提取首项系数 $a$
$$ax^2 + bx + c = a\left(x^2 + \frac{b}{a}x\right) + c$$

2. 加减配方项
$$= a\left(x^2 + \frac{b}{a}x + \frac{b^2}{4a^2}\right) + c - \frac{b^2}{4a}$$

3. 写成完全平方
$$= a\left(x + \frac{b}{2a}\right)^2 + \left(c - \frac{b^2}{4a}\right)$$

变量代换

令 $u = x + \frac{b}{2a}$，则：
$$ax^2 + bx + c = au^2 + r, \quad r = c - \frac{b^2}{4a}$$

教学目的

通过配方将二次式标准化，便于求解形如：
   - $\displaystyle\int \frac{dx}{ax^2 + bx + c}$
   - $\displaystyle\int \sqrt{ax^2 + bx + c}\,dx$

## 7. 黎曼和与梯形近似的估计偏差
>用左黎曼和 $L$、右黎曼和 $R$、梯形和 $T$ 近似定积分 $\int_a^b f(x)dx$ 时，如何判断它们与积分真值的大小关系？

**关键：判断函数的单调性和凹凸性**

### 单调性的影响

- **递增函数**：$L < \text{积分真值} < R$
- **递减函数**：$R < \text{积分真值} < L$

### 凹凸性的影响（对梯形和）

- **凹向上**（$f''(x) > 0$）：梯形在曲线上方 → $T > \text{积分真值}$
- **凹向下**（$f''(x) < 0$）：梯形在曲线下方 → $T < \text{积分真值}$

### 综合判断示例

若 $f(x)$ 在 $[a,b]$ 上**递增且凹向下**（如 $f(x)=\sqrt{x}$）：
$$L < T < \text{积分真值} < R$$

若 $f(x)$ 在 $[a,b]$ 上**递增且凹向上**（如 $f(x)=x^2$）：
$$L < \text{积分真值} < T < R$$

### 记忆技巧

- 左黎曼和：递增时低估，递减时高估
- 右黎曼和：递增时高估，递减时低估
- 梯形和：凹向下时低估，凹向上时高估（"凹低凸高"）
