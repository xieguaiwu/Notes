## 1. 特殊极限

$$\lim_{x\to 0}\frac{\sin x}{x}=1$$
$$\lim_{x\to 0}\frac{1-\cos x}{x}=0$$
$$\lim_{x\to 0}\frac{\tan x}{x}=1$$

## 2. 指数与对数运算

### 指数运算规则
$$a^m \cdot a^n = a^{m+n}$$
$$\frac{a^m}{a^n} = a^{m-n}$$
$$(a^m)^n = a^{mn}$$
$$(ab)^n = a^n b^n$$
$$\left(\frac{a}{b}\right)^n = \frac{a^n}{b^n}$$
$$a^{-n} = \frac{1}{a^n}$$
$$a^0 = 1 \quad (a \neq 0)$$

### 对数运算规则
$$\log_a (MN) = \log_a M + \log_a N$$
$$\log_a \frac{M}{N} = \log_a M - \log_a N$$
$$\log_a M^n = n \log_a M$$
$$\log_a \sqrt[n]{M} = \frac{1}{n} \log_a M$$
$$\log_a a = 1, \quad \log_a 1 = 0$$

### 换底公式
$$\log_a b = \frac{\log_c b}{\log_c a} = \frac{\ln b}{\ln a}$$
$$\log_a b \cdot \log_b a = 1$$
$$\log_a b^n = n \log_a b$$
$$\log_{a^n} b = \frac{1}{n} \log_a b$$
$$a^{\log_b c} = c^{\log_b a}$$

### 自然对数与常用对数
自然对数：$\ln x = \log_e x$

常用对数：$\lg x = \log_{10} x$

$$\ln e = 1, \quad \ln 1 = 0$$

### 特殊恒等式
$$\ln a + \ln b = \ln(ab)$$
$$\ln a - \ln b = \ln\left(\frac{a}{b}\right)$$
$$e^{\ln x} = x, \quad \ln e^x = x$$

### 常见极限
$$\lim_{x\to 0} \frac{\ln(1+x)}{x} = 1$$
$$\lim_{x\to 0} \frac{e^x - 1}{x} = 1$$
$$\lim_{x\to 0} \frac{a^x - 1}{x} = \ln a$$
$$\lim_{n\to\infty} \left(1 + \frac{x}{n}\right)^n = e^x$$

## 3. 三角恒等式

### 基本恒等式
$$\sin^2\theta + \cos^2\theta = 1$$
$$1 + \tan^2\theta = \sec^2\theta$$
$$1 + \cot^2\theta = \csc^2\theta$$

### 二倍角公式
$$\sin(2\theta) = 2\sin\theta\cos\theta$$
$$\cos(2\theta) = \cos^2\theta - \sin^2\theta = 2\cos^2\theta - 1 = 1 - 2\sin^2\theta$$
$$\tan(2\theta) = \frac{2\tan\theta}{1-\tan^2\theta}$$

### 半角公式
$$\sin^2\left(\frac{\theta}{2}\right) = \frac{1-\cos\theta}{2}$$
$$\cos^2\left(\frac{\theta}{2}\right) = \frac{1+\cos\theta}{2}$$
$$\tan\left(\frac{\theta}{2}\right) = \frac{1-\cos\theta}{\sin\theta} = \frac{\sin\theta}{1+\cos\theta}$$

### 加减法公式
$$\sin(\alpha \pm \beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\beta$$
$$\cos(\alpha \pm \beta) = \cos\alpha\cos\beta \mp \sin\alpha\sin\beta$$
$$\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha\tan\beta}$$

### 辅助角公式
$$a\sin x + b\cos x = \sqrt{a^2+b^2}\cdot\sin(x+\theta)$$
其中 $\sin\theta = \frac{b}{\sqrt{a^2+b^2}}$，$\cos\theta = \frac{a}{\sqrt{a^2+b^2}}$

### 降幂公式
$$\sin^2\theta = \frac{1-\cos 2\theta}{2}$$
$$\cos^2\theta = \frac{1+\cos 2\theta}{2}$$

### 特殊恒等式
$$\sec^2\theta - \tan^2\theta = 1$$
$$\csc^2\theta - \cot^2\theta = 1$$

### 积化和差公式
$$\sin\alpha\cos\beta = \frac{1}{2}[\sin(\alpha+\beta) + \sin(\alpha-\beta)]$$
$$\cos\alpha\sin\beta = \frac{1}{2}[\sin(\alpha+\beta) - \sin(\alpha-\beta)]$$
$$\cos\alpha\cos\beta = \frac{1}{2}[\cos(\alpha+\beta) + \cos(\alpha-\beta)]$$
$$\sin\alpha\sin\beta = -\frac{1}{2}[\cos(\alpha+\beta) - \cos(\alpha-\beta)]$$

### 和差化积公式
$$\sin\alpha + \sin\beta = 2\sin\frac{\alpha+\beta}{2}\cos\frac{\alpha-\beta}{2}$$
$$\sin\alpha - \sin\beta = 2\cos\frac{\alpha+\beta}{2}\sin\frac{\alpha-\beta}{2}$$
$$\cos\alpha + \cos\beta = 2\cos\frac{\alpha+\beta}{2}\cos\frac{\alpha-\beta}{2}$$
$$\cos\alpha - \cos\beta = -2\sin\frac{\alpha+\beta}{2}\sin\frac{\alpha-\beta}{2}$$

### 万能公式
设 $t = \tan\frac{\theta}{2}$，则：
$$\sin\theta = \frac{2t}{1+t^2} \qquad \cos\theta = \frac{1-t^2}{1+t^2} \qquad \tan\theta = \frac{2t}{1-t^2}$$

## 4. 导数公式

### 基本导数
$$\frac{d}{dx}[\ln x] = \frac{1}{x} \qquad \frac{d}{dx}[\log_a x] = \frac{1}{x\ln a}$$
$$\frac{d}{dx}[e^x] = e^x \qquad \frac{d}{dx}[a^x] = a^x \ln a$$

### 三角函数导数
$$\frac{d}{dx}[\sin x] = \cos x \qquad \frac{d}{dx}[\cos x] = -\sin x$$
$$\frac{d}{dx}[\tan x] = \sec^2 x \qquad \frac{d}{dx}[\cot x] = -\csc^2 x$$
$$\frac{d}{dx}[\sec x] = \sec x \tan x \qquad \frac{d}{dx}[\csc x] = -\csc x \cot x$$

### 三角函数平方导数
$$\frac{d}{dx}[\sin^2 x] = \sin 2x \qquad \frac{d}{dx}[\cos^2 x] = -\sin 2x$$

### 反三角函数导数
$$\frac{d}{dx}[\arcsin x] = \frac{1}{\sqrt{1-x^2}} \qquad \frac{d}{dx}[\arccos x] = -\frac{1}{\sqrt{1-x^2}}$$
$$\frac{d}{dx}[\arctan x] = \frac{1}{1+x^2} \qquad \frac{d}{dx}[\cot^{-1} x] = -\frac{1}{1+x^2}$$
$$\frac{d}{dx}[\sec^{-1} x] = \frac{1}{|x|\sqrt{x^2-1}} \qquad \frac{d}{dx}[\csc^{-1} x] = -\frac{1}{|x|\sqrt{x^2-1}}$$

## 5. 积分公式

### 基本积分
$$\int \frac{1}{x} \, dx = \ln|x| + C$$
$$\int e^x \, dx = e^x + C \qquad \int a^x \, dx = \frac{a^x}{\ln a} + C$$

### 三角函数积分
$$\int \sin x \, dx = -\cos x + C \qquad \int \cos x \, dx = \sin x + C$$
$$\int \tan x \, dx = \ln|\sec x| + C \qquad \int \cot x \, dx = \ln|\sin x| + C$$
$$\int \sec x \, dx = \ln|\sec x + \tan x| + C$$
$$\int \csc x \, dx = \ln|\csc x - \cot x| + C$$

### 三角函数平方积分
$$\int \sin^2 x \, dx = \frac{x}{2} - \frac{\sin 2x}{4} + C$$
$$\int \cos^2 x \, dx = \frac{x}{2} + \frac{\sin 2x}{4} + C$$
$$\int \sec^2 x \, dx = \tan x + C \qquad \int \csc^2 x \, dx = -\cot x + C$$
$$\int \tan^2 x \, dx = \tan x - x + C \qquad \int \cot^2 x \, dx = -\cot x - x + C$$

### 反三角函数形式积分
$$\int \frac{1}{\sqrt{1-x^2}} \, dx = \arcsin x + C$$
$$\int \frac{1}{1+x^2} \, dx = \arctan x + C$$
$$\int \frac{1}{\sqrt{a^2-x^2}} \, dx = \arcsin\frac{x}{a} + C$$
$$\int \frac{1}{a^2+x^2} \, dx = \frac{1}{a}\arctan\frac{x}{a} + C$$
$$\int \frac{1}{\sqrt{x^2+a^2}} \, dx = \ln(x + \sqrt{x^2+a^2}) + C$$
$$\int \frac{1}{\sqrt{x^2-a^2}} \, dx = \ln|x + \sqrt{x^2-a^2}| + C$$

## 6. 积分技巧

### 分部积分法
$$\int u \, dv = uv - \int v \, du$$

**LIATE 选 u 法则：**
Logarithmic → Inverse trig → Algebraic → Trigonometric → Exponential

**表格积分法：**
对于 $\int u \cdot v \, dx$，当 $u$ 能逐次求导至零时：
- 对 $u$ 逐次求导（至零）
- 对 $v$ 逐次积分
- 交替符号：$+$、$-$、$+$、$-$...
- 对角相乘直到 $u$ 为零

### 三角换元法

| 被积函数形式 | 换元 | 恒等式 |
|-------------|------|--------|
| $\sqrt{a^2-x^2}$ | $x = a\sin\theta$ | $1-\sin^2\theta = \cos^2\theta$ |
| $\sqrt{a^2+x^2}$ | $x = a\tan\theta$ | $1+\tan^2\theta = \sec^2\theta$ |
| $\sqrt{x^2-a^2}$ | $x = a\sec\theta$ | $\sec^2\theta - 1 = \tan^2\theta$ |

### $\sin^n x \cos^m x$ 积分策略

| 情况 | 策略 |
|------|------|
| $n$ 为奇数 | 提取 $\sin x$，用 $\sin^2 x = 1-\cos^2 x$，换元 $u = \cos x$ |
| $m$ 为奇数 | 提取 $\cos x$，用 $\cos^2 x = 1-\sin^2 x$，换元 $u = \sin x$ |
| $n, m$ 均为偶数 | 用降幂公式 |

### $\tan^n x \sec^m x$ 积分策略

| 情况 | 策略 |
|------|------|
| $m$ 为偶数 | 提取 $\sec^2 x$，用 $\sec^2 x = 1 + \tan^2 x$，换元 $u = \tan x$ |
| $n$ 为奇数 | 提取 $\sec x \tan x$，用 $\tan^2 x = \sec^2 x - 1$，换元 $u = \sec x$ |

### 部分分式分解

| 分母形式 | 分解形式 |
|----------|----------|
| $(x-a)(x-b)$ | $\frac{A}{x-a} + \frac{B}{x-b}$ |
| $(x-a)^n$ | $\frac{A_1}{x-a} + \frac{A_2}{(x-a)^2} + \cdots + \frac{A_n}{(x-a)^n}$ |
| $(ax^2+bx+c)$ | $\frac{Ax+B}{ax^2+bx+c}$ |

## 7. 反常积分收敛性

### p-积分判别法
$$\int_1^{\infty} \frac{1}{x^p} \, dx: \quad p > 1 \text{ 收敛}, \quad p \leq 1 \text{ 发散}$$
$$\int_0^1 \frac{1}{x^p} \, dx: \quad p < 1 \text{ 收敛}, \quad p \geq 1 \text{ 发散}$$

### 比较判别法
若 $0 \leq f(x) \leq g(x)$：
- $\int g$ 收敛 $\Rightarrow$ $\int f$ 收敛
- $\int f$ 发散 $\Rightarrow$ $\int g$ 发散

## 8. 旋转体体积

### 圆盘法
绕 x 轴：$V = \pi \int_a^b [f(x)]^2 \, dx$

绕 y 轴：$V = \pi \int_c^d [g(y)]^2 \, dy$

### 圆环法
绕 x 轴（两曲线 $R(x) \geq r(x)$）：$V = \pi \int_a^b ([R(x)]^2 - [r(x)]^2) \, dx$

### 柱壳法
绕 y 轴：$V = 2\pi \int_a^b x \cdot f(x) \, dx$

绕 x 轴：$V = 2\pi \int_c^d y \cdot g(y) \, dy$

绕 $x = k$：$V = 2\pi \int_a^b |x - k| \cdot f(x) \, dx$

## 9. 参数方程

### 导数
$$\frac{dy}{dx} = \frac{y'(t)}{x'(t)} \qquad \frac{d^2y}{dx^2} = \frac{d}{dt}\left(\frac{dy}{dx}\right) \cdot \frac{1}{x'(t)}$$

### 弧长
$$L = \int_a^b \sqrt{[x'(t)]^2 + [y'(t)]^2} \, dt$$

### 曲线下面积
$$A = \int_a^b y(t) \cdot x'(t) \, dt$$

### 旋转体积
绕 x 轴：$V = \pi \int_a^b [y(t)]^2 \cdot x'(t) \, dt$

## 10. 极坐标

### 坐标转换
$$x = r\cos\theta, \quad y = r\sin\theta, \quad r^2 = x^2 + y^2$$

### 极坐标面积
$$A = \frac{1}{2} \int_{\alpha}^{\beta} [r(\theta)]^2 \, d\theta$$

两曲线间：$A = \frac{1}{2} \int_{\alpha}^{\beta} ([r_2]^2 - [r_1]^2) \, d\theta$

### 极坐标弧长
$$L = \int_{\alpha}^{\beta} \sqrt{r^2 + \left(\frac{dr}{d\theta}\right)^2} \, d\theta$$

### 切线斜率
$$\frac{dy}{dx} = \frac{r'\sin\theta + r\cos\theta}{r'\cos\theta - r\sin\theta}$$

### 常见极坐标曲线
| 名称 | 方程 |
|------|------|
| 圆 | $r = 2a\cos\theta$（圆心 $(a,0)$，半径 $a$） |
| 玫瑰线 | $r = a\sin(n\theta)$（$n$ 奇数 $n$ 瓣，$n$ 偶数 $2n$ 瓣） |
| 心形线 | $r = a(1 \pm \cos\theta)$ |
| 双纽线 | $r^2 = a^2\cos(2\theta)$ |

## 11. 向量值函数

### 定义
$\mathbf{r}(t) = \langle x(t), y(t) \rangle$

### 速度与加速度
位置：$\mathbf{r}(t)$ → 速度：$\mathbf{v}(t) = \mathbf{r}'(t)$ → 加速度：$\mathbf{a}(t) = \mathbf{r}''(t)$

速率：$|\mathbf{v}(t)| = \sqrt{[x'(t)]^2 + [y'(t)]^2}$

### 弧长与路程
$$L = \int_a^b |\mathbf{r}'(t)| \, dt$$

位移：$\Delta\mathbf{r} = \mathbf{r}(b) - \mathbf{r}(a)$，路程：$L = \int_a^b |\mathbf{v}(t)| \, dt$

### 单位切向量
$$\mathbf{T}(t) = \frac{\mathbf{r}'(t)}{|\mathbf{r}'(t)|}$$

## 12. 无穷级数

### 几何级数
$$\sum_{n=0}^{\infty} ar^n = \frac{a}{1-r}, \quad |r| < 1$$

### p-级数
$$\sum_{n=1}^{\infty} \frac{1}{n^p}: \quad p > 1 \text{ 收敛}, \quad p \leq 1 \text{ 发散}$$

### Maclaurin 级数
$$e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!}, \quad x \in \mathbb{R}$$
$$\sin x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n+1}}{(2n+1)!}, \quad x \in \mathbb{R}$$
$$\cos x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n}}{(2n)!}, \quad x \in \mathbb{R}$$
$$\frac{1}{1-x} = \sum_{n=0}^{\infty} x^n, \quad |x| < 1$$
$$\ln(1+x) = \sum_{n=1}^{\infty} \frac{(-1)^{n+1} x^n}{n}, \quad -1 < x \leq 1$$
$$\arctan x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n+1}}{2n+1}, \quad |x| \leq 1$$

### 收敛判别法

| 判别法 | 条件与结论 |
|--------|-----------|
| 发散判别法 | $\lim a_n \neq 0 \Rightarrow$ 发散 |
| 积分判别法 | $f(x)$ 正、连续、递减，$\sum a_n$ 与 $\int f$ 同敛散 |
| 比较判别法 | $0 \leq a_n \leq b_n$：$\sum b_n$ 收敛 $\Rightarrow$ $\sum a_n$ 收敛 |
| 极限比较 | $\lim \frac{a_n}{b_n} = c \in (0, \infty) \Rightarrow$ 同敛散 |
| 比值判别法 | $L = \lim |\frac{a_{n+1}}{a_n}|$：$L < 1$ 收敛，$L > 1$ 发散 |
| 根值判别法 | $L = \lim \sqrt[n]{|a_n|}$：$L < 1$ 收敛，$L > 1$ 发散 |
| 交错级数 | $b_{n+1} \leq b_n$ 且 $\lim b_n = 0 \Rightarrow$ 收敛 |

### 交错级数误差估计
$$|S - S_n| \leq b_{n+1}$$

### 幂级数运算
逐项求导：$f'(x) = \sum n a_n x^{n-1}$

逐项积分：$\int f(x) \, dx = \sum \frac{a_n}{n+1} x^{n+1} + C$

## 13. 临界点与拐点

| 概念 | 条件 |
|------|------|
| 临界点 | $f'(c) = 0$ 或 $f'(c)$ 不存在 |
| 局部极大值 | $f'(c)=0$ 且 $f''(c) < 0$，或 $f'$ 从正变负 |
| 局部极小值 | $f'(c)=0$ 且 $f''(c) > 0$，或 $f'$ 从负变正 |
| 拐点 | $f''(c) = 0$ 或不存在，且凹凸性改变 |
| 驻点 | $f'(c) = 0$ |
