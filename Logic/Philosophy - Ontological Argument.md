# Philosophy - Ontological Argument

> [!abstract] 概述
> **本体论论证**是从"神"的概念出发，试图证明神必然存在的先验论证。历史上最重要的版本包括 Anselm（《宣讲》）、笛卡尔（《第五沉思》）、Gödel（1970，模态高阶逻辑版本）等。Gödel 的版本因其使用高阶模态逻辑而尤其受到关注，但其公理系统被发现会导致**模态坍塌**（[[Modal Logic - Modal Collapse]]）。

## 主要版本

| 哲学家 | 核心前提 | 问题 |
|--------|----------|------|
| Anselm | 不可设想比之更大的者 | Gaunilo lost island 反驳 |
| Descartes | 存在的属性蕴含于完满者的本质 | 循环论证争议 |
| Gödel | 正性质构成超滤 | **模态坍塌** |
