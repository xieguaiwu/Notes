# Modal Logic - Possible Worlds

> [!abstract] 概述
> **可能世界**是模态逻辑和哲学语义学中的核心概念。一个可能世界是事物可能存在的一种完整方式。模态算子 $\Box$（必然）和 $\Diamond$（可能）的语义通过量化可能世界来解释。模态坍塌（[[Modal Logic - Modal Collapse]]）是一类特殊的现象，其中所有可能世界在真值上完全一致。
