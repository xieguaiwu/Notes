# French Negation — 否定式

## 一、基本否定结构

法语基本否定用 **ne...pas** 包围动词：

```
ne + 动词 + pas
```

| 肯定 | 否定 |
|------|------|
| Je **parle** français. | Je **ne parle pas** anglais. |
| Il **aime** le café. | Il **n'aime pas** le thé. |
| Nous **sommes** fatigués. | Nous **ne sommes pas** fatigués. |

> [!note] 元音省略
> ne 在以元音或哑音 h 开头的动词前缩写为 **n'**：Je **n'**aime pas. / Il **n'**habite pas ici.

---

## 二、否定词分类

### 1. 否定副词

| 否定词 | 含义 | 示例 |
|--------|------|------|
| ne...**pas** | 不/没（一般否定）| Je **ne** sais **pas**. |
| ne...**plus** | 不再 | Il **ne** fume **plus**. |
| ne...**jamais** | 从不 | Je **ne** voyage **jamais**. |
| ne...**guère** | 几乎不 | Il **ne** travaille **guère**. |

### 2. 否定代词

| 否定词 | 含义 | 示例 |
|--------|------|------|
| ne...**rien** | 什么都没有 | Je **ne** vois **rien**. |
| ne...**personne** | 没有人 | Je **ne** vois **personne**. |

### 3. 带有 aucun 的否定

| 结构 | 示例 |
|------|------|
| ne...**aucun(e)** + nom | Je **n'**ai **aucune** idée. |
| ne...**ni...ni** | Je **ne** bois **ni** café **ni** thé. |

---

## 三、否定词的位置规则

### 1. 简单时态中

```
ne + 否定词 + 动词 + 否定词第二部分
```

- Je **ne** mange **pas** de viande.
- Il **ne** vient **plus**.
- Nous **ne** buvons **jamais** d'alcool.

### 2. 复合过去时中

```
ne + 助动词 + pas/plus/jamais + 过去分词
```

- Je **n'**ai **pas** mangé.
- Il **n'**a **jamais** été à Paris.
- Nous **n'**avons **rien** vu.

### 3. 不定式前

```
ne pas + infinitif
```

- **Ne pas** fumer.（禁止吸烟。）
- Il a décidé de **ne pas** venir.

### 4. personne/rien 在复合时态中的位置

```
n' + 助动词 + rien + 过去分词
n' + 助动词 + 过去分词 + personne
```

- Je **n'**ai **rien** vu.（我什么都没看见。）
- Je **n'**ai vu **personne**.（我谁都没看见。）

---

## 四、否定句中的冠词变化

### 不定冠词和部分冠词 → de

在否定句中，直接宾语前的**不定冠词**（un, une, des）和**部分冠词**（du, de la, des）变为 **de**（或 **d'**）：

| 肯定句 | 否定句 |
|--------|--------|
| J'ai **un** livre. | Je **n'**ai pas **de** livre. |
| Je prends **du** café. | Je **ne** prends pas **de** café. |
| Il a **des** amis. | Il **n'**a pas **d'**amis. |

> [!warning] 例外
> - 动词 être 后保持不定冠词：Ce **n'**est pas **un** problème.
> - ne...jamais/plus/rien 也适用此规则

## 五、否定词速查表

| 否定结构 | 含义 | 肯定对应 |
|---------|------|---------|
| ne...pas | 不/没 | — |
| ne...plus | 不再 | encore / toujours |
| ne...jamais | 从不 | toujours / souvent |
| ne...rien | 什么都不 | quelque chose |
| ne...personne | 没有人 | quelqu'un |
| ne...aucun(e) | 没有任何一个 | quelques |
| ne...ni...ni | 既不…也不 | et / ou |
| ne...guère | 几乎不 | beaucoup / assez |

> [!tip] 口语中的否定
> 口语中常省略 **ne**，只保留 **pas**：
> - 书面/正式：Je **ne** sais **pas**.
> - 口语：Je sais **pas**.

---

## 六、常见错误

### 1. Ne 在口语中省略
> 口语中常省略 ne，但写作/考试中一定要写：
❌ Je sais **pas**.（口语可接受）
✅ Je **ne** sais **pas**.（标准书面语）

### 2. Rien / Personne 在复合过去时的位置
❌ Je **n'**ai pas vu **rien**.
✅ Je **n'**ai **rien** vu.

### 3. 否定句中冠词变化
❌ Je n'ai pas **du** pain.
✅ Je n'ai pas **de** pain.

### 4. Ne...plus 与 ne...jamais 区分
> **ne...plus** = 不再（之前有过）
> **ne...jamais** = 从未（从来没有过）
✅ Je **ne** fume **plus**. = 不再抽烟了
✅ Je **ne** fume **jamais**. = 从不抽烟
