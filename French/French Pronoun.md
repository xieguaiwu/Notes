# French Pronoun — 法语代词

## 一、代词概述

法语代词分为以下几类：

| 类型 | 法语名称 | 示例 |
|------|---------|------|
| 人称代词 | Pronoms personnels | je, tu, il, elle |
| 重读代词 | Pronoms toniques | moi, toi, lui, elle |
| 直接宾语代词 | Pronoms COD | me, te, le, la |
| 间接宾语代词 | Pronoms COI | me, te, lui |
| 副代词 y 和 en | Pronoms adverbiaux | y, en |

---

## 二、人称代词 (Pronoms Personnels Sujets)

| 人称 | 法语 | 中文 |
|------|------|------|
| 第一人称单数 | **je** | 我 |
| 第二人称单数 | **tu** | 你 |
| 第三人称单数阳性 | **il** | 他/它 |
| 第三人称单数阴性 | **elle** | 她 |
| 第三人称单数泛指 | **on** | 我们/有人 |
| 第一人称复数 | **nous** | 我们 |
| 第二人称复数 | **vous** | 你们/您 |
| 第三人称复数阳性 | **ils** | 他们 |
| 第三人称复数阴性 | **elles** | 她们 |

> [!note] on 的用法
> 口语中 **on** 常代替 nous：**On** va au cinéma.（我们去看电影。）

---

## 三、重读代词 (Pronoms Toniques)

用于介词后、强调、或无动词的句中。

| 人称 | 重读代词 |
|------|---------|
| je → **moi** | Moi, j'aime le café. |
| tu → **toi** | Et toi ? |
| il → **lui** | Il travaille avec **lui**. |
| elle → **elle** | Je pense à **elle**. |
| nous → **nous** | Chez **nous**. |
| vous → **vous** | C'est pour **vous**. |
| ils → **eux** | Ils viennent avec **eux**. |
| elles → **elles** | Je parle d'**elles**. |

### 主要用法

| 用法 | 示例 |
|------|------|
| 介词后 | avec **moi**, pour **toi**, chez **lui** |
| 强调主语 | **Moi**, je suis fatigué. |
| C'est + 重读代词 | C'est **lui**. C'est **elle**. |
| 单独回答 | Qui ? — **Moi**. |

---

## 四、直接宾语代词 (Pronoms COD)

代替"谁"或"什么"（直接宾语）。

| 人称 | COD 代词 | 位置 |
|------|---------|------|
| me → **me** (m') | Il **me** voit.（他看见我。） | 动词前 |
| te → **te** (t') | Je **te** connais.（我认识你。） | 动词前 |
| le/la → **le/la** (l') | Je **le** vois.（我看见他/它。） | 动词前 |
| nous → **nous** | Il **nous** aide.（他帮助我们。） | 动词前 |
| vous → **vous** | Je **vous** écoute.（我听您说。） | 动词前 |
| les → **les** | Je **les** vois.（我看见他们。） | 动词前 |

### 否定式

```
ne + COD + verbe + pas
```

- Je **ne le** vois **pas**.（我没看见他。）
- Il **ne nous** écoute **pas**.（他没听我们。）

---

## 五、间接宾语代词 (Pronoms COI)

代替"对谁"（间接宾语，动词后接 à）。

| 人称 | COI 代词 | 示例 |
|------|---------|------|
| me → **me** (m') | Il **me** téléphone.（他打电话给我。） |
| te → **te** (t') | Je **te** parle.（我对你说话。） |
| lui → **lui** | Je **lui** écris.（我写信给他/她。） |
| nous → **nous** | Il **nous** répond.（他回答我们。） |
| vous → **vous** | Je **vous** donne.（我给你。） |
| leur → **leur** | Je **leur** dis.（我告诉他们。） |

> [!warning] 注意
> - COI 和 COD 的 me/te/nous/vous 形式相同
> - 第三人称不同：**le/la**（COD）vs **lui**（COI）

---

## 六、副代词 y 和 en

### Y（代替 à + 地点/事物）

```
y = à / dans / sur + 地点 或 à + 事物
```

| 用法 | 示例 |
|------|------|
| 代替地点 | Je vais **à Paris**. → J'**y** vais. |
| 代替事物 | Je pense **à mon travail**. → J'**y** pense. |

### En（代替 de + 名词）

```
en = de + 名词（数量/地点/事物）
```

| 用法 | 示例 |
|------|------|
| 代替数量 | J'ai **trois livres**. → J'**en** ai trois. |
| 代替事物/人 | Je parle **de ce film**. → J'**en** parle. |
| 代替地点 | Je viens **de Paris**. → J'**en** viens. |

> [!tip] 代词顺序
> 在一个动词前有多个代词时，顺序为：
> **me/te/se/nous/vous** → **le/la/les** → **lui/leur** → **y** → **en**

## 七、代词位置速查

```
肯定句: 主语 + 代词 + 动词 (+ 否定 pas)
否定句: 主语 + ne + 代词 + 动词 + pas
复合过去时: 主语 + 代词 + 助动词 + 过去分词
```

| 句类 | 语序示例 |
|------|---------|
| 肯定 | Je **le** vois. |
| 否定 | Je **ne le** vois **pas**. |
| 复合过去时 | Je **l'**ai vu. |
| 否定+复合过去时 | Je **ne l'**ai **pas** vu. |
| 命令式肯定 | Regarde**-le** ! |
| 命令式否定 | **Ne le** regarde **pas** ! |
