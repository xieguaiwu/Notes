# French Adjective — 法语形容词

## 一、形容词的性数配合

法语形容词必须与所修饰的名词在**性**（阳性/阴性）和**数**（单数/复数）上保持一致。

### 1. 阴性构成基本规则

| 情况 | 规则 | 阳性 → 阴性 |
|------|------|-------------|
| **一般情况** | +e | grand → grand**e** |
| **-e 结尾** | 不变 | jeune → jeune（同形） |
| **-er 结尾** | -er → -ère | léger → lég**ère** |
| **-x 结尾** | -x → -se | heureux → heureu**se** |
| **-f 结尾** | -f → -ve | actif → acti**ve** |
| **-el / -eil 结尾** | +le | nature**l** → nature**lle** |
| **-en / -on 结尾** | 双写 n + e | anci**en** → anci**enne** |
| **-et 结尾** | -et → -ète | comp**let** → compl**ète** |
| **-eur 结尾** | -eur → -euse | menteur → menteu**se** |

### 2. 复数构成基本规则

| 情况 | 规则 | 单数 → 复数 |
|------|------|-------------|
| **一般情况** | +s | grand → grand**s** |
| **-s, -x 结尾** | 不变 | gros → gros |
| **-eau 结尾** | +x | beau → beau**x** |
| **-al 结尾** | -al → -aux | nation**al** → nation**aux** |

## 二、形容词的位置

### 1. 常见位置：名词之后

大部分法语形容词放在**名词之后**：

| 示例 |
|------|
| un livre **intéressant**（一本有趣的书）|
| une voiture **rouge**（一辆红色的车）|
| une histoire **vraie**（一个真实的故事）|

### 2. 放在名词前的形容词（BAGS）

以下类别形容词放在**名词之前**：

| 类别 | 形容词 |
|------|--------|
| **B**eauty（美） | beau, joli, laid |
| **A**ge（年龄） | jeune, vieux, nouveau |
| **G**oodness（好坏） | bon, mauvais, gentil |
| **S**ize（大小） | grand, petit, gros, long |

| 示例 |
|------|
| **un beau** jardin（一个漂亮的花园）|
| **un grand** homme（一个高大的男人）|
| **une bonne** idée（一个好主意）|

### 3. 位置改变含义的形容词

| 形容词 | 名词前（引申义） | 名词后（本义） |
|--------|----------------|---------------|
| **grand** | un **grand** homme（伟人）| un homme **grand**（高个子男人）|
| **pauvre** | un **pauvre** homme（可怜的人）| un homme **pauvre**（贫穷的人）|
| **cher** | mon **cher** ami（亲爱的朋友）| un livre **cher**（昂贵的书）|
| **propre** | ma **propre** maison（我自己的房子）| une maison **propre**（干净的房子）|

## 三、常见不规则形容词

### 特殊阳性单数形式

有些形容词在阳性单数名词前有特殊形式：

| 阳性单数（辅音前） | 阳性单数（元音前） | 阴性单数 |
|-----------------|-----------------|---------|
| **beau** 美丽的 | **bel** homme | bell**e** |
| **nouveau** 新的 | **nouvel** an | nouveau**lle** |
| **vieux** 老的 | **vieil** homme | viei**lle** |
| **fou** 疯狂的 | **fol** espoir | fo**lle** |

> [!tip] 特殊形式只在阳性单数元音前使用！
