# French Past Tense — 复合过去时 (Passé Composé)

## 一、时态概述

复合过去时是法语最常用的过去时态，用于表达：
- **已完成的动作**：J'**ai mangé**.（我吃过了。）
- **一系列过去事件**：Il **est entré**, il **a salué** tout le monde.
- **对现在有影响的过去事件**：Elle **est partie**.（她走了。）

---

## 二、构成公式

```
主语 + 助动词（avoir/être）+ 过去分词
```

### 1. 助动词选择

约 90% 的动词用 **avoir**，约 10% 用 **être**。

### 2. 过去分词的构成

| 动词类型 | 规则 | 不定式 → 过去分词 |
|---------|------|-----------------|
| **第一组 -er** | 词尾 → **-é** | parler → parl**é** |
| **第二组 -ir** | 词尾 → **-i** | finir → fin**i** |
| **第三组 -re** | 词尾 → **-u** | vendre → vend**u** |

### 常见不规则过去分词

| 动词 | 过去分词 |
|------|---------|
| avoir | **eu** |
| être | **été** |
| faire | **fait** |
| prendre | **pris** |
| mettre | **mis** |
| dire | **dit** |
| écrire | **écrit** |
| ouvrir | **ouvert** |
| pouvoir | **pu** |
| vouloir | **voulu** |
| devoir | **dû** |
| savoir | **su** |
| voir | **vu** |
| lire | **lu** |
| boire | **bu** |
| comprendre | **compris** |
| apprendre | **appris** |
| naître | **né** |

## 三、用 être 作助动词的动词

### Dr. Mrs. Vandertramp 动词表

以下动词和所有自反代词动词用 **être** 作助动词：

| 动词 | 含义 | 过去分词 |
|------|------|---------|
| **Devenir** | 变成 | deven**u** |
| **Revenir** | 回来 | reven**u** |
| **Monter** | 上/登上 | mont**é** |
| **Rester** | 停留 | rest**é** |
| **Sortir** | 出去 | sort**i** |
| **Venir** | 来 | ven**u** |
| **Aller** | 去 | all**é** |
| **Naître** | 出生 | n**é** |
| **Descendre** | 下/落下 | descend**u** |
| **Entrer** | 进入 | entr**é** |
| **Rentrer** | 回去/进来 | rentr**é** |
| **Tomber** | 跌倒/落下 | tomb**é** |
| **Retourner** | 返回 | retourn**é** |
| **Arriver** | 到达 | arriv**é** |
| **Mourir** | 死 | mort |
| **Partir** | 离开 | parti |

> [!warning] 过去分词配合
> 用 être 的动词，过去分词必须与主语的性数配合：
> - Il **est arrivé**. / Elle **est arrivée**.
> - Ils **sont partis**. / Elles **sont parties**.

---

## 四、复合过去时的否定式

```
主语 + ne + 助动词 + pas + 过去分词
```

- Je **n'**ai **pas** mangé.
- Il **n'**est **pas** venu.

> 代词位置：主语 + ne + 代词 + 助动词 + pas + 过去分词
> - Je **ne l'**ai **pas** vu.

---

## 五、复合过去时与未完成过去时（简介）

A2 级别只需掌握复合过去时（Passé Composé）。以下是两种过去时的主要区别：

| 复合过去时 (Passé Composé) | 未完成过去时 (Imparfait) |
|------------------------|-----------------------|
| 完成的、一次性的事件 | 持续的、习惯性的状态 |
| J'**ai mangé** à 13h. | Quand j'**étais** petit... |
| 动作的"快照" | 背景/描述 |

> [!note] A2 阶段专注于复合过去时即可，Imparfait 在 B1 系统学习。

## 六、常见错误

### 1. 助动词选择错误
❌ J'**ai** allé à Paris.（aller 用 être）
✅ Je **suis** allé à Paris.

### 2. 过去分词配合遗漏（être 动词）
❌ Elle **est** arrivé.
✅ Elle **est arrivée**.（主语阴性，过去分词 +e）
❌ Ils **sont** parti.
✅ Ils **sont partis**.（主语复数，过去分词 +s）

### 3. 不规则过去分词拼写
❌ J'ai **prendé**.
✅ J'ai **pris**.

### 4. 否定式代词位置
❌ Je **n'**ai vu **le** pas.
✅ Je **ne l'**ai **pas** vu.
