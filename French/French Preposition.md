# French Preposition — 法语介词

## 一、介词概述

介词用于连接句子成分，表示时间、地点、方式、目的等关系。

---

## 二、常用介词用法

### 1. **à** — 在/到/给

| 用法 | 示例 |
|------|------|
| 地点（去/在某个城市） | Je vais **à** Paris. |
| 时间（几点） | Le cours commence **à** 9h. |
| 所属关系 | C'est le livre **à** Paul. |
| 目的（用来做） | une machine **à** café |
| 间接宾语 | Je parle **à** Marie. |

> [!warning] à + le = **au**，à + les = **aux**

### 2. **de** — 从/的/用

| 用法 | 示例 |
|------|------|
| 所属关系 | le livre **de** Marie |
| 来源/出发 | Je viens **de** Paris. |
| 内容/材质 | une tasse **de** café |
| 起点时间 | **du** matin au soir |
| 数量 | un verre **de** vin |

> [!warning] de + le = **du**，de + les = **des**

### 3. **en** — 在（国名/状态/材料/时间）

| 用法 | 示例 |
|------|------|
| 阴性国家 | Je vais **en** France. |
| 交通方式 | voyager **en** train |
| 材料 | une robe **en** soie |
| 时间（月份/年/季节） | **en** janvier, **en** 2026 |

### 4. **dans** — 在…里面/在…之后

| 用法 | 示例 |
|------|------|
| 在空间内 | Le livre est **dans** le sac. |
| 在未来时间后 | Je reviens **dans** trois jours. |

### 5. **sur** — 在…上面

| 用法 | 示例 |
|------|------|
| 在表面 | Le livre est **sur** la table. |
| 关于 | un article **sur** l'histoire |

### 6. **chez** — 在…家/在…处

| 用法 | 示例 |
|------|------|
| 在某人家里 | Je vais **chez** Paul. |
| 在某机构/专业场所 | **chez** le médecin |
| 在某群体中 | C'est une habitude **chez** les Français. |

### 7. **avec** — 和…一起/用

| 用法 | 示例 |
|------|------|
| 伴随 | Je vais **avec** toi. |
| 工具/方式 | écrire **avec** un stylo |
| 伴随状态 | **avec** plaisir |

### 8. **pour** — 为了/前往/作为

| 用法 | 示例 |
|------|------|
| 目的/为了 | Ce cadeau est **pour** toi. |
| 目的地 | partir **pour** Paris |
| 时间长度 | **pour** un mois |
| 对…而言 | **pour** moi, c'est important |

## 三、介词与冠词的缩合

| 介词 | + le | + la | + l' | + les |
|------|------|------|------|------|
| **à** | **au** | à la | à l' | **aux** |
| **de** | **du** | de la | de l' | **des** |

| 完整形式 | 缩合后 | 示例 |
|---------|-------|------|
| à + le | **au** | Je vais **au** cinéma. |
| à + les | **aux** | Je parle **aux** étudiants. |
| de + le | **du** | le livre **du** professeur |
| de + les | **des** | les livres **des** enfants |

## 四、易混淆介词辨析

### sur vs. dans

| 介词 | 维度 | 示例 |
|------|------|------|
| **sur** | 表面/上面 | sur la table（在桌上）|
| **dans** | 内部/里面 | dans la boîte（在盒子里）|

### en vs. dans（时间）

| 介词 | 含义 | 示例 |
|------|------|------|
| **en** | 在…期间（月份/年） | **en** juillet（在七月）|
| **dans** | …之后（未来） | **dans** deux jours（两天后）|

### pour vs. depuis（时间）

| 介词 | 含义 | 示例 |
|------|------|------|
| **pour** | 持续…时间（将来） | Je pars **pour** un mois. |
| **depuis** | 自从…（持续至今） | Je travaille **depuis** 8h. |
