# French Future Tense — 将来时

## 一、最近将来时 (Futur Proche)

最近将来时是法语 A1-A2 最常用的将来表达方式，口语中使用频率极高。

### 1. 构成公式

```
aller（现在时变位）+ 不定式
```

| 人称 | aller 变位 | + 不定式 | 示例 |
|------|-----------|---------|------|
| je | **vais** | + parler | Je **vais** parler. |
| tu | **vas** | + parler | Tu **vas** parler. |
| il/elle/on | **va** | + parler | Il **va** parler. |
| nous | **allons** | + parler | Nous **allons** parler. |
| vous | **allez** | + parler | Vous **allez** parler. |
| ils/elles | **vont** | + parler | Ils **vont** parler. |

### 2. 用法

| 用途 | 示例 |
|------|------|
| **即将发生的动作** | J'**vais** sortir.（我要出去了。） |
| **计划/打算** | Nous **allons** visiter Paris.（我们打算去巴黎。） |
| **推测（即将）** | Il **va** pleuvoir.（要下雨了。） |

### 3. 否定式

```
ne + aller（变位）+ pas + 不定式
```

- Je **ne vais pas** venir.
- Il **ne va pas** pleuvoir.

## 二、简单将来时 (Futur Simple) — 简介

简单将来时用于表达**较远未来**的动作，是 A2 需要认知的时态。

### 1. 构成公式

```
不定式 + 词尾：-ai, -as, -a, -ons, -ez, -ont
```

以 **parler** 为例：

| 人称 | 变位 |
|------|------|
| je | parler**ai** |
| tu | parler**as** |
| il/elle/on | parler**a** |
| nous | parler**ons** |
| vous | parler**ez** |
| ils/elles | parler**ont** |

> [!note] -re 结尾动词去掉末尾 -e 再加词尾：prendre → je prendr**ai**

### 2. 常见不规则词根

| 动词 | 简单将来时词根 |
|------|-------------|
| être | **ser-** |
| avoir | **aur-** |
| aller | **ir-** |
| faire | **fer-** |
| pouvoir | **pourr-** |
| vouloir | **voudr-** |
| devoir | **devr-** |
| venir | **viendr-** |
| voir | **verr-** |
| envoyer | **enverr-** |

### 3. 否定式

```
ne + 动词（将来时）+ pas
```

- Je **ne** viendr**ai pas** demain.

## 三、现在时表将来

法语口语中，**现在时 + 时间状语**也可表达将来：

| 现在时（表将来） | 中文 |
|-----------------|------|
| Je **pars** demain. | 我明天出发。 |
| Le train **arrive** à 20h. | 火车20点到。 |

---

## 四、三种将来表达对比

| 表达方式 | 构成 | 使用场景 | 示例 |
|---------|------|---------|------|
| **现在时** | 现在时 + 时间状语 | 确定安排 | Je **pars** demain. |
| **最近将来时** | aller + 不定式 | 即将/打算 | Je **vais** partir. |
| **简单将来时** | 词根 + 词尾 | 较远未来/正式 | Je **partirai** un jour. |

> [!tip] A1-A2 阶段优先掌握 **最近将来时 (Futur Proche)**，简单将来时了解即可。
