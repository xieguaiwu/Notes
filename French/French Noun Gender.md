# French Noun Gender — 名词的性与数

## 一、名词的性 (Genre)

法语名词分为 **阳性 (masculin)** 和 **阴性 (féminin)** 两类。没有中性。

> [!tip] 记住口诀
> 先记词尾规律，但**最好每次学新名词时连同冠词一起记**（如 le livre, la table）。

### 1. 阳性名词常见词尾

| 词尾 | 示例 |
|------|------|
| **-age** | le fromage, le voyage |
| **-ment** | le gouvernement, le bâtiment |
| **-eau** | le bureau, le chapeau |
| **-isme** | le tourisme, le capitalisme |
| **-eur**（职业/机器） | le professeur, l'ordinateur |
| **-t** | le chat, le marché |
| **-in** | le matin, le jardin |
| **-al** | le cheval, le journal |

### 2. 阴性名词常见词尾

| 词尾 | 示例 |
|------|------|
| **-tion / -sion / -ssion** | la nation, la télévision, la passion |
| **-té** | la liberté, la santé |
| **-ure** | la nature, la voiture |
| **-ance / -ence** | la France, la différence |
| **-ie** | la vie, la théorie |
| **-ette** | la cigarette, la bicyclette |
| **-eur**（抽象名词） | la douleur（疼痛）, la chaleur（热） |
| **-son** | la maison, la chanson |

> [!warning] 注意
> -eur 结尾：表示**职业/机器**→阳性；表示**抽象概念**→阴性
> - 例外需要单独记忆：**le** problème（阳性）、**le** musée（阳性）、**la** peau（阴性）、**la** main（阴性）

### 3. 表示人的名词的阴阳性对应

| 阳性 | 阴性 | 规律 |
|------|------|------|
| un ami | une **ami_e_** | +e |
| un étudiant | une étudiant**e** | +e |
| un professeur | une professeur**e** | 口语中 +e |
| un acteur | une act**rice** | -eur → -rice |
| un chanteur | une chant**euse** | -eur → -euse |
| un vendeur | une vend**euse** | -eur → -euse |

## 二、名词的数 (Nombre)

### 1. 复数构成基本规则

| 情况 | 规则 | 单数→复数 |
|------|------|-----------|
| **一般情况** | +s | livre → livre**s** |
| **-s, -x, -z 结尾** | 不变 | pays → pays, nez → nez |
| **-eau, -au 结尾** | +x | château → château**x** |
| **-eu 结尾** | +x | jeu → jeu**x** |
| **-al 结尾** | → aux | cheval → chevau**x** |
| **-ou 结尾** | +s（大部分） | clou → clou**s** |

### 2. -al 结尾的例外

大部分 -al → -aux，但少数只加 **-s**：
- le bal → les bal**s**
- le festival → les festival**s**
- le carnaval → les carnaval**s**

### 3. 特殊复数形式

| 单数 | 复数 |
|------|------|
| l'**œil**（眼睛） | les **yeux** |
| le **cadeau**（礼物） | les **cadeaux** |
| le **travail**（工作） | les **travaux** |
| monsieur | **messieurs** |
| madame | **mesdames** |
| mademoiselle | **mesdemoiselles** |

## 三、记忆技巧

### 词尾规律口诀

> **-tion, -sion, -ssion** → 阴性，不用问
> **-age, -ment, -eau** → 阳性，要记牢
> **-té, -ure** → 阴性，常出现
> **-isme, -aire** → 阳性，别大意

### 常见例外速记

| 看起来像阳性却是阴性 | 看起来像阴性却是阳性 |
|---------------------|---------------------|
| **la** main（手）| **le** musée（博物馆）|
| **la** peau（皮肤）| **le** problème（问题）|
| **la** foi（信仰）| **le** système（系统）|

### 核心策略

> [!note] 最佳记忆策略
> 1. **背名词时连冠词一起背**：le livre（书）、la table（桌子）
> 2. 先记高频词尾规律，再重点攻克例外
> 3. 复数主要靠冠词（les/des）体现，口语中词尾 -s 通常不发音
